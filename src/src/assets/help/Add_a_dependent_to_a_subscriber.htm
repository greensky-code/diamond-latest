Add a dependent to a subscriber

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to add a dependent to a subscriber.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type MEMBR, and then press ENTER.

 

2. In the Subscriber ID box, type the subscriber ID you want to add a dependent, and then press TAB.

F5 lookup is available.

3. If the List of Family Members dialog box displays, then double-click the name of the subscriber.

 

4. On the Special menu, click Add Dependent.

A new record is started.

5. Enter data into the fields on the screen.

Including selecting the relationship of the dependent to the subscriber in the Rel Code list.

6. On the File menu, click Save.

You are prompted to confirm the ID card request and the dependent is added to the subscriber.

Frequently Asked Questions

None.
