Manual Cap Pay Acct

O

Bank Account code associated with manual (external) checks for capitation payables.

Selected from a predefined list.

Bank accounts are maintained in keyword BANKA. This field must be completed in order to enter payment information from manual checks (in MCHEK).

Manual Med Pay Acct

O

Bank Account code associated with manual (external) checks for claim payables.

Selected from a predefined list.

Bank accounts are maintained in keyword BANKA. This field must be completed in order to enter payment information from manual checks (in MCHEK).

Sub/Dep Min Check.

O

Minimum amount for which a check will be written to a subscriber or dependent for payables associated with this company.

Manually entered.

You can type up to 16 digits before the decimal and 2 digits after the decimal.

When checks are set up in keyword CKPRT (Check Print Setup), this minimum amount can be typed over by using the Minimum Override field.

Tax Rep Entity

O

Tax Reporting Entity associated with this company.

Selected from a predefined list.

Tax Reporting Entities are maintained in keyword TRENT (Tax Reporting Entity), and used to derive IRA Tax ID and payer/transmitter information.

Trade Payable

Vend Min Check

O

Minimum amount for which a check will be written to a vendor for payables associated with this company.

Manually entered.

You can type up to 16 digits before the decimal and 2 digits after the decimal.

When checks are set up in keyword CKPRT, this minimum amount can be typed over by using the Minimum Override field.

ZIP

O

ZIP/Postal Code for company.

Manually entered.

You can type up to 15 alphanumeric characters (unless limited by Setup).
