Making a Tooth Rule Inactive

Restrictions

Instructions

Use the following procedure to make a tooth rule inactive.

Restrictions

None.

Instructions

Step

Comment

1. Display keyword TRULE and access the tooth rule you want to make inactive.

 

2. In the Term Date field, type a date.

The rule is no longer evaluated as of the date you enter in this field.

3. In the Term Reason field, type a reason code.

This step is optional.

F5 lookup is available.

4. On the File menu, click Save.
