VADPP Commands on the Menu Bar

 

Command            Menu  Description

Acc Pay Vendor Display

S

Display keyword APVDS (Accounts Payable Vendor Display).

Adv Pay Account

T

Display keyword VADPY (Vendor Advance Payment Account).

Adv Pay Priority

T

Display keyword VADPP (Advance Payment Offset Priority)

Adv Pay Rules

S

Display keyword VADPR (Vendor Advance Payment Rules)

Notes

Details

N

This command is only available if the VADPP window has been associated with at least one note type in NOTEW.

Displays the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.

Provider Relationships

S

Display the providers associated with this vendor ID, in each of the following categories:

  • Providers who use this vendor ID for their default vendor.

  • Providers who use this vendor for payment of capitation (PROVC, Type=C, Cap Vend ID).

  • Providers who use this vendor for payment of fee-for-service claims (PROVC, Type=Payable).

Vendor Address

T

Display keyword VENDA (Vendor Address).

Vendor Credit

T

Display keyword VENCR (Vendor Credit).

Vendor Lookup

 

S

Opens the Vendor Lookup screen.

Vendor Master

T

Display keyword VENDR (Vendor Master).

View Credit Balance

S

Displays the Transaction ID, File Type, and Net Amount of the vendors credit balance. If the vendor no longer has a credit balance, the screen displays the Paid Net Amount.
