About ACPAY

Use keyword ACPAY (Accounts Payable) to display an A/P transaction created by one of the following methods:

  • For medical claims: a successful run of an A/P update job in keyword APUPD (Accounts Payable Update). Keyword APUPD adds one A/P transaction in keyword ACPAY for every claim detail line it finalizes.

  • For capitation encounters: a successful run of a capitation calculation job in keyword CCALC (Capitation Calculation Setup).

By displaying an A/P transaction, you can view claim status, amount, check information, G/L information, or cap/special fund information.

After displaying an A/P transaction, you can change the payment selection, due date of the check, or whether the transaction is reportable as part of a Form 1099.

To display an A/P transaction from the claims detail window, on the Special menu, click Payment Detail.
