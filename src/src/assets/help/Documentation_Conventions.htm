Documentation Conventions

The following conventions are used in Help:

Convention

Description

Entries

Actual entries you type at the terminal keyboard appear in boldface type. For example, type MEMBR

Important Notes

Important notes alert you to information that could affect the outcome of a completing a task.

Keys

The keys you press on the keyboard appear in uppercase letters. If combined with a plus sign (+), press and hold the first key while you press the remaining key(s). For example, press SHIFT+TAB.

Menu Names and Options

In tasks, menu names and options appear in boldface type. For example, on the File menu, click Open.

Notes

Notes contain special information and alert you to effects of an action.

Tips

Tips provide an alternate method for completing a task.

Warnings

Warnings alert you to possible consequences of completing a task.
