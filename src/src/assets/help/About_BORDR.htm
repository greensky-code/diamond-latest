About BORDR

Use keyword BORDR (Benefit Processing Order) to specify the order in which each type of benefit rule—deductibles, copayments, exclusions, and so forth—is evaluated during claims adjudication.

You may define multiple BORDR records with different orders and attach them to different benefit packages (BENEF). The benefit package defines the order within type that each individual benefit rule is evaluated.
