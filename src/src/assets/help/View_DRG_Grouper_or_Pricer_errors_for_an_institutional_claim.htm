Use the following procedure to view DRG Grouper or Pricer errors for an institutional claim.

Restrictions

None.

Instructions

Step

Comment

1. At the Main Menu, press F3.

 

2. In the Keyword box, type INCLM, and then press ENTER.

 

3. Display the claim you want to view DRG Grouper or Pricer errors.

 

4. On the Window menu, click Processing Messages.
