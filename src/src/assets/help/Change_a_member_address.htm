Change a member address

Use this procedure to change a member address.

Alternatively, you can change certain Address Types using keywords MEMBR (Member) and ALIAS (Alias/Responsible Party/Privacy). You can use keyword MEMBR to change an address of address type R1 - Member Residential/Primary. You can use keyword ALIAS to change an address of address type R4 - Alias Residential/Primary or R5 - Responsible Party Residential/Primary.

To change a member address

  1. Press F3, type MEMBA, and then press ENTER.

  2. In the DIAMOND ID box, type the DIAMOND ID of the member for whom you want to change a member address.

  3. In the Subscriber ID box, type the Subscriber ID of the member for whom you want to change a member address.

  4. In the person number picklist, click the person number of the member for whom you want to change a member address.

  5. In the Additional Address Information picklist, click the member address you want to change.

  6. Change the fields in the window as needed. For more information, see MEMBA Field Definitions.

  7. Press CTRL+S to save.
