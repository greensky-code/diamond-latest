PROVF Parameters & Prerequisites

The following is only a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

 

Parameter

Description

Comments

DFTCNTRCTPRV

Stores the Provider ID and Oracle sequence ID for the record designated as the Default Provider.

Provider contract records can be set up for this single provider rather than each individual provider when contract characteristics are shared. LINBS records direct DIAMOND to use the Default Provider rather than searching for individual provider contracts.

PROVENROL

Determines whether provider enrollment rules will be enforced.

To activate this parameter, run the Provider/Member Enrollment Utility (Provider_Member_Enrollment.sh).
