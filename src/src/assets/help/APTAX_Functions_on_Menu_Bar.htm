APTAX Commands on the Menu Bar

 

Command            Menu  Description

Close

F

Close the AP 1099 Reporting window.

Create Extract from TRENT

S

Create a comma-delimited text file of all the IRS tax reporting entities in keyword TRENT known as the payers.

The extract file is created in the same directory as the DIAMOND application and named <PAYERS.csv>

Create Extract from VENDR

S

Create a comma-delimited text file of all the vendor TIN records, known as payees.

The extract file is created in the same directory as the DIAMOND application and named <PAYEES.csv>

Create Detail Extract For This Job ID

S

Create a comma-delimited text file of all AP tax records created in the selected 1099 job.

The extract file is created in the same directory as the DIAMOND application and named <DETAILS.csv>

New

F

Create a new 1099 job.

New from Template

F

Create a new 1099 job using an existing template.

Save as Template

F

Save the current 1099 job as a template.

One saved as a template, the job will display in a dialog box when the New from Template command is selected from the File menu.

Run Job

F

Run the active 1099 job.
