Change the Password of a User

Use the following procedure to change the password of a user.

Important Note Â  To change user passwords, you must be a super user or have been granted access to the Update User Password command on the Special menu in keyword SUSER (Function F_MAINT_USR_PASS).

To change the password of a user

  1. Press F3, type SUSER, and then press ENTER.

  2. In the User ID box, type the user ID, and then press TAB. F5 lookup is available.

  3. On the Special menu, click Update User Password. The Add Database User dialog box displays.

  4. In the Password box, type the new password.

  5. In the Confirm Password box, re-type the new password.

  6. Click OK.
