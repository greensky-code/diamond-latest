ZIPRF Field Definitions

Field Name

Req

Description

Source

Comments

Area Code

O

Telephone area code associated with the ZIP/Postal Code.

Manually entered.

Informational only; not automatically retrieved when a ZIP/Postal Code is entered in an address.

City

O

City associated with the ZIP/Postal Code.

Manually entered.

 

County

O

County code associated with the ZIP/Postal Code.

Manually entered.

You can type up to 5 characters.

Informational only will not be automatically retrieved when ZIP/Postal Code is entered in an address.

State

O

State associated with the ZIP/Postal Code.

Manually entered.

 

ZIP Code

R

Unique ZIP/Postal Code.

Manually entered.

You can type up to 15 alphanumeric characters.

Do not type dashes; they are automatically entered for 9-digit ZIP Codes.

Zip Code Qualifier

O

Identifies whether the Zip Code is Urban or Rural

Selected from predefined list

Valid values are defined in CODES using system code type ZIPCODEQUALIFIER.
