Change a Tax Reporting Entity

Use the following procedure to change a tax reporting entity.

To change a tax reporting entity

  1. Press F3, type TRENT, and then press ENTER.

  2. In the picklist, click the tax reporting entity for which you want to change.

  3. Make the changes you want. For more information about the fields in the window, see TRENT Field Definitions.

  4. On the File menu, click Save.
