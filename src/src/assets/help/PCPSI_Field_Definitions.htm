Age From

(Pediatric Indicator section box)

C

Youngest member age that qualifies as a child for PCP Auto Assignment of a provider with this primary Pediatric specialty

Manually entered

Required if you enter a Specialty in the Pediatric Indicator section box box. Only whole digits can be entered.

The same Age From/Age Thru range is used for all Specialties you enter in this section box.

Age From

(OB/GYN Indicator section box)

C

Youngest member age that qualifies for PCP Auto Assignment of a provider with this primary OB/GYN specialty

Manually entered

Required if you enter a Specialty in the OB/GYN Indicator section box. Only whole digits can be entered.

The same Age From/Age Thru range is used for all Specialties you enter in this section box.

Exclude Age From

C

Youngest member age to exclude for the entered Specialty code for PCP Auto Assignment of a provider with this primary specialty

Manually entered

Required if you enter a Specialty in the Specialty Restrictions by Age section box. Only whole digits can be entered.

The same Exclude Age From/Exclude Age Thru range is used for all Specialties you enter in this section box.

Exclude Age Thru

C

Oldest member age to exclude for the entered Specialty code for PCP Auto Assignment of a provider with this primary specialty

Manually entered

The same Exclude Age From/Exclude Age Thru range is used for all Specialties you enter in this section box.

Exclude Gender

C

Gender to be excluded from PCP Auto Assignment of a provider with the entered primary specialty

Selected from predefined list

Required if you enter a Specialty in the Specialty Restrictions by Gender section box.

Valid values are:

F - Female

M - Male

U - Unknown

Gender

(OB/GYN Indicator section box)

C

Gender for which you want to define a member age range for PCP Auto Assignment of a provider with this primary OB/GYN specialty

Selected from predefined list

Required if you enter a Specialty in the OB/GYN Indicator section box. Select Female for all Specialties you enter in this section box.
