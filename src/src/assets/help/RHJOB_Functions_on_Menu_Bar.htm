MRHJOB Commands on the Menu Bar

 

Command             Menu   Description

New From Template

F

Use to enter a new RHJOB job using an existing job template. Choose a template from any previously entered jobs that were saved as templates.

Save As Template

F

Use to save an RHJOB job you entered as a template that can be used to enter additional jobs.

Run Job

F

Use to run the current job.

Critical Error Report

Details

S

View or print a report listing information for claims in the RHJOB batch run that encountered critical errors during processing and were not updated.

Detail Report

Details

S

View or print a report showing information for all reason codes that were released (marked as historical) in the RHJOB batch run, as well as active holds.

Claims Processed Statistics

Details

S

Displays the number of claims selected for processing by the RHJOB job along with the number of claims for which a hold was released and retested.
