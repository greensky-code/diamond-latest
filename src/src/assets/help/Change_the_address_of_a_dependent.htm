Change the address of a dependent

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to change the address for a dependent. This procedure does NOT change the address of the subscriber.

If you have previously selected to pay a claim to this dependent in keyword INCLM (Institutional Claims), PSCLM (Professional Services Claims), or DNCLM (Dental Services Claims), then this procedure also changes its vendor address in keyword VENDA (Vendor Address).

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type MEMBR, and then press ENTER

 

2. In the Subscriber ID box, type the subscriber ID, and then press TAB.

F5 lookup is available.

3. In the List of Family Members dialog box, double-click the name of the dependent.

Usually, dependents are indicated with a Person No. of 2 or higher.

4. Make your changes in the address fields.

 

5. On the File menu, click Save.

 

Frequently Asked Questions

None.
