CFDST Detail Audit Report Menu Command

Field Name

Description

Source

Comments

Adjust Net Payable

The adjusted Net Amount payable to the provider after running Capitation Fund Distribution

Calculated by CFDST logic

 

Allocation Amt

Amount to be allocated from the cap/special fund for this CFDST job

Entered in Allocation field in CFDST

Total amount available for distribution for claims for all providers with contracts associated with the CF Model ID specified in CFDST

Allocation Method

The field on claim detail lines used as the basis to allocate the Cap/Special Fund amount

Specified in CFMOD

May be Allowed Amount or Net Amount

Allocation Rate

Rate calculated to apply to the Allowed Amount or Net Amount on CFDST selected claim detail lines

Calculated by CFDST

See Capitation/Special Fund Distribution, How DIAMOND Thinks for more information on how the Allocation Rate is calculated and used

Allowed Amount

Amount allowed for the claim detail line

Calculated by pricing logic in INCLM/PSCLM

 

Billed Amount

Amount billed by the provider for the claim detail line

Entered in INCLM/PSCLM

 

Cap Fund Withhold Amt

The difference between the original claim detail line Net Amount and the adjusted claim detail line Net Amount after running Cap/Special Fund Distribution

Calculated by CFDST logic

 

Cap Month

Month for which the CFDST job was run

Entered in CFDST

 

Cap Run From Date

Earliest date of service allowable on claim detail lines selected for this job

Entered in CFDST

If this date was not entered in CFDST, no date is displayed. By default, DIAMOND used the first day of the CFDST Cap Fund Month as the From Date.

Claim Number

Unique identifying number of the claim

Entered in INCLM/PSCLM

 

Claim Type

Type of claim

Specified in CFMOD

 

Comments

Free form comments

Entered in CFDST

 

Copay Deductible

The sum of the Total Copay, Total Coinsurance and Total Deductible amounts on the claim detail line, calculated by adjudication logic in INCLM/PSCLM

Calculated by CFDST logic

 

Job ID

Unique ID for this job

Entered in CFDST

 

Job Run Date

Date this CFDST job was run

System generated

 

Line No.

Claim detail line number

Entered in INCLM/PSCLM

 

Max Allocation

Maximum allocation percentage applied to limit the amount allocated to a provider

Specified in CFMOD

 

Min Allocation

Minimum percentage of the original claim detail line Net Amount that the provider must be paid

Specified in CFMOD

 

Mod

Modifier on the claim detail line

Entered in INCLM/PSCLM

 

Model ID

ID of the model used in this CFDST run

Specified in CFDST

 

Net Amount

The original Net Amount prior to running Capitation Fund Distribution

Calculated by adjudication logic in INCLM/PSCLM

 

Not Covered Amount

The claim detail line Not Covered Amount

Calculated by adjudication logic in INCLM/PSCLM

 

Orig Net Amt

Total of Net Amounts on CFDST-selected claim detail lines prior to running the CFDST job and calculating a new Net Amount

Calculated by CFDST logic

 

Primary DOS

Primary date of service of the claim

Entered in INCLM/PSCLM

 

Procedure Code

Procedure code on the claim detail line

Entered in INCLM/PSCLM

 

Status

Status of the CFDST job.

System generated

Valid values are:

Successful Run

Posted

Sub Line

Claim detail sub line number

Entered in INCLM/PSCLM

 

Total Units

The total unit value units resulting from use of the Unit Value pricing calculation method

Calculated by pricing logic in INCLM/PSCLM

Total units will only be displayed when the claim detail line was priced using the Unit Value pricing calculation method

Withhold Amount

The claim detail line withhold amount

Calculated by adjudication logic in INCLM/PSCLM
