INRUL Field Definitions

Field Name

Req

Description

Source

Comments

Access Program

O

If you want to select an access program, then click to select this check box.

Selected from check box

Only those providers with the Access Program Eligible check box selected in keyword PROVE (Provider Enrollment) are eligible for the access program on this incentive rule.

Amount

C

Type the amount to be multiplied by the provider’s enrollment count used in capitation or to be included as a single amount in the capitation payment.

Manually entered

This field is required if the Access Program check box is selected and the Pct box is blank.

Amt

C

Type the dollar amount to be used in one of the following:

  • Multiplied by the provider’s enrollment member count used in capitation

- or -

  • Included as a single amount in the capitation payment.

Manually entered

This field is required if you have selected the Access Program check box and the Pct box is cleared.

Apply Amount to

C

Select the method to apply the amount.

Selected from predefined list

 

You can select one of the following values:

  • M – Apply per member

Multiple the amount by the number of enrolled members included in the provider’s capitation calculation.

  • P – Apply once

Include the amount once in the capitation payment.

If the Amount box displays $0.00, then this field is optional.

  • P – Apply once

Description

O

Type the description of the rule.

Manually entered

 

Effective Date

R

Type the effective date of the rule.

Manually entered

The rule is effective on the first unposted capitation month after the month you typed. Capitation months are posted in keyword CCALC (Capitation Calculation Setup).

If you type a past date and the capitation month for that date is retroactively calculated, then the rule will not be applied to the retroactive calculation.

ID

R

Type the ID of the incentive rule.

Manually entered

F5 lookup is available.

You can type up to 15 alphanumeric characters.

IPA

N

Type the provider’s IPA.

 

F5 lookup is available.

You cannot change this field after the incentive rule is saved.

Line of Bus

O

Type the provider’s line of business.

Manually entered

F5 lookup is available.

You cannot change this field after the incentive rule is saved.

LOB accepts new members

O

If you want to include providers in the same line of business as this incentive rule that are accepting new members, then click to select this check box.

Selected from check box

If you select this check box, when capitation is run, the system will evaluate if the provider has at least on provider enrollment record in keyword PROVE or at least one provider contract record in keyword PROVC in the applicable line of business with the Accepting New Patients check box is selected.

This field is available if you have selected the Access Program check box.

LOB enroll within limits

O

If you want to include providers with provider contract records in keyword PROVC (Provider Contract) that are associated with a line of business record in keyword LINBS (Line of Business) that specify an enrollment limit, then click to select this check box.

Selected from check box

If you select this check box, when capitation is run, the system will evaluate whether the provider has a provider contract record in keyword PROVC for the applicable line of business that does not have an enrollment limit entered.

This field is available if you have selected the Access Program check box.

Minimum number

O

Type the minimum number of members that must be assigned to the provider in order to qualify for the access program.

Manually entered

You can type a number from 1 to 99999.

This field is available if you have selected the Minimum members check box.

Minimum members

O

If you want the provider to have a minimum number of members to quality for the access program, then click to select this check box.

Selected from check box

This check box is available if the Access Program check box is selected.

Panel

O

Type the provider’s panel.

Manually entered

F5 lookup is available.

You cannot change this field after the incentive rule is saved.

Percent

O

Type the percent to multiply by the total capitation amount to determine the amount to be included or deducted from the capitation payment.

Manually entered

This field is required if the Amount box is set to $0.00.

Pct

C

Type the percent to be multiplied by the total capitation amount to be included or deducted from the capitation payment.

Manually entered

This field is required if the Access Program check box is selected and the Amount box is set to $0.00.

Quality Program

O

Select the quality program you want to use.

Selected from predefined list

Valid values are maintained in keyword CODES (System Codes) with a code type of QUALPRGM.

Term Date

O

Type the termination date of the rule, if necessary.

Manually entered

You can type a date equal to or greater than the date in the Effective Date box.

If you type a retroactive termination date, then the following rules apply:

  • Records already calculated in a capitation month after the termination date will not be recalculated.

  • If the capitation month after the termination date is retroactively calculated, then the incentive rule will be included in the next month’s calculation.

  • If the incentive rule is included in a capitation period that has not already been capitated, then the rule will be included in that period when capitation is run.

Term Reason

C

Select the reason code to use when the rule terminates.

Selected from predefined list

Only reason codes with the Reason Code Type list set to TM – Terminate in keyword REASN (Reason) display.

This field is required if you type a date in the Term Date box.

Total enroll within limits

O

If you want to include providers that have not met the line of business individual enrollment limit entered in the Max Enroll Lmt box in keyword PROVF (Provider Master), then click to select this check box.

By selecting this check box, if the provider’s enrollment limit has been met or exceeded, then the provider is not eligible for the access program.

Selected from check box.

If you select this check box, when capitation is run, the system will evaluate whether the provider’s record in keyword PROVF has a value in the Max Enroll Lmt box.

This field available if the Access Program check box is selected.
