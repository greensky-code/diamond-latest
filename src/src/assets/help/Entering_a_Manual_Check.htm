Enter a Manual Check

Use the following procedure to enter a manual check produced outside of the DIAMOND program to pay claim detail lines or capitation payables. When you post a manual check job, the relevant claim or capitation, accounts payable, and check register records are updated. In addition, EOB and RA data are available so that EOBs and RAs can be printed in keyword EOBRA (EOB/RA Printing).

Restrictions

If the manual check was used to pay claims, those claim payables must be posted and selected for payment in keyword APUPD (Accounts Payable Update).

If the manual check was used to pay capitation, those capitation payables must be posted in keyword CCALC (Capitation Calculation Setup) and selected for payment in keyword APUPD.

To enter a manual check

  1. Press F3, type MCHEK, and then press ENTER.

  2. In the Payable Type list, click Medical (claims) or Capitated.

  3. On the File menu, click New.

  4. In the Company Code list, click the company code ID.

  5. In the Start Check No box, type the number of the manual check.

  6. In the Vendor ID box, type the vendor who was paid with the manual check. F5 lookup is available.

  7. In the Vendor Address list, click the vendor’s address.

  8. In the Claim Type list, click the type of claims you paid with the manual check.

  9. If desired, fill in the remaining fields on the screen.

  10. On the File menu, click Run Job. The Manual Check Update window displays. Depending on your selection in step 2, the claim detail lines or capitation payables that meet the selection criteria display in the picklist. A unique sequential ‘pass’ number for each set of vendor selection criteria is assigned.

  11. In the picklist, click the claims you paid with the manual check. Selected payables can be deselected by clicking on them a second time. You can use the Special menu commands View AP, View Source, and View Vendor to see the ACPAY, INCLM/PSCLM/DNCLM, and VENDR records, respectively, for each selected detail line.

  12. On the File menu, click Run Job.

  13. Once all manual checks have been assigned to their respective payables, display the main window and run a report of the Job ID, using the Special menu command Print Job Log.

  14. After reviewing the job log, under Action, click Post, and then on the File menu, click Run Job. All manual checks and associated payables for all pass IDs are posted for that Job ID.

Tip

Only detail lines that have the Proc Status list set to F – Final (posted) in the claim detail window are shown. Held and denied detail lines are not available for payment.
