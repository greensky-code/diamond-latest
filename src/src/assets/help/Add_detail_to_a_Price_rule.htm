Add Detail to a Price Rule

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to add detail to a price rule.

Restrictions

You previously must have added a price rule in keyword PRULE (Price Rule).

Instructions

Step

Comment

1. Press F3, type PRULE, and then press ENTER.

 

2. In the Price Rule box, type the Price Rule ID for which you want to add detail.

F5 lookup is available.

3. On the Topic menu, click Price Rule Detail.

 

4. On the File menu, click New.

 

5. Fill in the fields on the window.

For more information, see Online Help.

6. On the File menu, click Save.

 

To add Multiple Modifier information

 

1. In the Price Method list, click the price method you want to use.

 

2. In the Allowed Reason box, click the allowed reason code for which you want to apply to the claim line item if Price Method 2, 3, or 4 is used.

F5 lookup is available.

3. If you selected if 4 – Modifier 1 + Modifier 2% from the Price Method list, then in the Allowed Factor Ovr box, type the percentage to apply to the first modifier allowed amount.

You can type a decimal number. For example, for 1%, type .01; for 50%, type .5; for 100%, type 1; for 150%, type 1.5

4. On the File menu, click Save.
