Re-select Claims or Capitation Transactions for Payment

Use the following procedure to create an Accounts Payable (A/P) selection job for A/P processing. By creating an A/P selection job, you can re- select claims or capitation transactions previously de-selected for payment.

To re-select claims or capitation transactions for payment

  1. Press F3, type APSEL, and then press ENTER.

  2. On the File menu, click New.

  3. In the Payable Type list, do one of the following:

  4. If you want to re-select medical claims for payment, then click M – Medical.

  5. If you want to re-select capitation transactions for payment, then click C – Capitation.

  1. Fill in the remaining fields in the window. For more information about the fields in the window, see APSEL Field Definitions.

  2. In the Action Type list, click Reselect for Pay.

  3. On the File menu, click Run Job. The A/P Selection job runs. When the job ID in the picklist list displays Successful Run in the Status column, the job has completed.

Tips

  • To refresh the display of the Status column in the picklist, on the Window menu, click Refresh Job Status.

  • If the Status column displays "Error," then you can view the error messages by clicking Processing Messages from the Window menu.
