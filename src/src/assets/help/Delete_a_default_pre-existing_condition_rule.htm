Delete a default pre-existing condition rule

You cannot delete a keyword PXRUL (Pre-Existing Condition Rule) default pre-existing condition rule once the record is saved. However, you can use keyword PXRUL to limit an effective period of a default pre-existing condition rule to a single day.

To limit the effective period of a default pre-existing condition rule to a single day

  1. Type PXRUL and press ENTER.

  2. In the PEC Rule ID field, type the ID of the rule you want to change.

  3. In the effective period picklist, click the effective period you want to limit.

  4. In the Term Date field, type a date equal to the Effective Date.

  5. Press CTRL+S to save.
