AutoAudit

O

Cross-reference code used by AutoAudit claims review software.

Manually entered.

Used only if CAUDTOOL Parameter 1 is set to 3.

AutoAudit software accepts plan codes of up to 4 positions while DIAMOND plan codes can be up to 10 positions. This field translates the DIAMOND plan code to a plan code acceptable to AutoAudit. If no cross reference is entered here the interface truncates the DIAMOND plan code to its first 4 positions.
