Add a Price Rule

Price rules define the methods used to calculate allowed amounts for professional service and institutional claims. Use the following procedure to add a price rule and price rule details.

To add a price rule

  1. Press F3, type PRULE, and then press ENTER.

  2. On the File menu, click New.

  3. In the Price Rule box, type the Price Rule ID.

  4. In the Description box, type the description of the price rule.

  5. If you are using an external DRG Pricer, then click to select the Use External DRG Pricer check box.

  6. In the Procedure Price Search Option list, click the search option you want to use. If you select W – Use claim//wildcard modifier, then in the Wildcard Modifier list, click a wildcard modifier code.

  7. On the File menu, click Save.

  8. If desired, add detail to the Price rule.
