About SFUNC

Use keyword SFUNC (Function Access) to grant a user or user template access to a command on the Topic, Special, or Window menu in a keyword window.

For example, in keyword MEMBR, the Terminate/Unterminate command to terminate a member is accessed on the Special menu. To limit a user’s ability to use the Terminate/Unterminate command, use keyword SFUNC. That way, a user can update member addresses and the other demographic information in keyword MEMBR, but not terminate members.

Because a security template association in keyword SUSER overrides individual user entries in keyword SFUNC, you may only limit command access for that user’s template. In other words, individual user entries in keyword SFUNC will be overridden by that user’s template’s entries in keyword SFUNC.

To see a complete list of the commands available in the Function list in keyword SFUNC, see About SFNDS.
