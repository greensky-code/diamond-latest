GROUP Commands on the Menu Bar

 

Command             Menu   Description

Audit Display

Details

W

Displays audit trail records for the selected record.

D/C Information

Details

S

This command is available only if the Use EFT check box in keyword GRUPB (Group Billing) is selected for the group. This debit/credit account information is stored in DIAMOND’s pmb_eft_interface table, but not used during the premium billing process.

Group Lookup

S

Lookup screen of valid groups in DIAMOND

Group User Fields

S

Provides user defined text and date fields. The labels for these fields are controlled by UDTXT. Includes the following fields:

User Defined 3-22. Can contain alphanumeric text up to 20 characters in length.

User Date 3-22. Valid date associated with User Defined field.

Comments. Can contain alphanumeric text up to 80 characters in length.

Notes

Details

N

This command is only available if the GROUP window has been associated with at least one note type in NOTEW.

Takes you to the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.
