Maximum Allocation

R

Maximum allocation percentage applied to limit the fund amount allocated to a provider

Manually entered.

Valid values are from 0.00 to 99.99.

Example: 0.8 =80%, 1.25 = 125%. Must be greater than or equal to the Minimum Allocation percentage.

If the Allocation Rate calculated by the CFDST process is greater than this maximum allocation percentage, DIAMOND uses the maximum allocation percentage specified instead. (Note that if this occurs and the maximum allocation percentage is less than 100%, the full fund Allocation amount may not be distributed.)

Minimum Allocation

R

Minimum percentage of the original claim detail line Net Amount that the provider must be paid

Manually entered.

Valid values are from 0.00 to 1.00.

Example: 0.4 = 40%. Cannot be negative.

If the Allocation Rate calculated by the CFDST process is less than this Minimum Allocation percentage, DIAMOND uses the Minimum Allocation percentage specified instead. (Note that if this occurs, the fund Allocation amount for the specified fund may be exceeded.)

Model ID

R

Manually entered.

Sub Model ID

Manually entered.

Sub Models used for Reporting Purposes only. You cannot allocate funds to Sub Models.
