Add a Super User

Use the following procedure to add a super user to the DIAMOND program. Super users have unlimited security privileges in the DIAMOND program.

To add a super user

  1. Press F3, type SUSER, and then press ENTER.

  2. In the User ID box, type an ID for the user, and then press TAB.

  3. Fill in the fields in the User Information section box.

  4. Click to select the Super User check box.

  5. In the User Type list, click Operator.

  6. In the Language list, click the default language.

  7. On the File menu, click Save.

  8. On the Special menu, click Add Database User.

  9. In the Password box, type the user password.

  10. In the Confirm Password box, re-type the user password, and then click OK. The user is connected to the Oracle roles DIAMONDVIEW and DIAMONDUPDATE.
