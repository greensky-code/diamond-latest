Add a Tax Reporting Entity

Use the following procedure to add a tax reporting entity.

To add a tax reporting entity

  1. Press F3, type TRENT, and then press ENTER.

  2. On the File menu, click New.

  3. In the IRS Tax ID box, type the Internal Revenue Service Tax ID, and then press TAB. Do not type any dashes.

  4. Fill in the remaining fields in the window. For more information about the fields in the window, see TRENT Field Definitions.

  5. On the File menu, click Save.
