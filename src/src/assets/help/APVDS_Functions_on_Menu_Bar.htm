APVDS Commands on the Menu Bar

 

Command             Menu   Description

AP Transaction Information

T

View the A/P transaction.

Claim Audit Details

Details

S

For the claim and claim line selected in the keyword, displays summary information for:

  • Claim detail lines that were impacted by the CodeReview evaluation.

  • Claim detail lines that caused CodeReview to change the claim.

Claim Detail Information

T

View the claim detail record.

Notes

Details

N

This command is only available if the APVDS window has been associated with at least one note type in NOTEW.

Displays the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.

Patient Liabilities

S

View claim details regarding benefit application and the total patient liability for a claim detail line.

Vendor Information

T

Display data from the vendor and vendor address records for the vendor originally entered as a filter for viewing A/P transactions.

View Additional G/L Accounts

Details

S

Display the incentive program and administrative fee G/L codes associated with the A/P transaction.

View Administrative Fee Info

S

Display keyword ADDSP (Administrative Fee Display). Use to view the administrative fee associated with the A/P transaction.

View AP Additional Info

S

Displays additional information associated with the currently displayed A/P transaction record:

Check Number or EFT Trans Number

Offset indicator

Check EFT Amount
