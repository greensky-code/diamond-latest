MCHEK Commands on the Menu Bar

 

Command             Menu   Description

Manual Checks Job Setup

T

Display the main window.

This command is available from Manual Check Update window.

Print Job Log

S

Print a log showing composition of manual checks within a Job ID.

This command is available from Manual Checks window.

Run Job

F

Run the action selected for the active Job ID.

This command is available from Manual Checks window.

Select

S

Assign the selected claim or capitation payable to a manual check.

This command is available from Manual Check Update window.

View AP

S

View the accounts payable record for the selected claim detail line payable.

This command is available from Manual Check Update window.

View Source

S

View the source claim for the selected claim detail line payable.

This command is available from Manual Check Update window.

View Vendor

S

View the vendor for the selected claim detail or capitation payable.

This command is available from Manual Check Update window.
