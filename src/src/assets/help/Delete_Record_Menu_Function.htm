Delete Record Menu Command

The Heads Down Claims executable deletes an entire claim record or a claim detail line record from the claims text file depending on the active window (Header or Detail).

Active Window = Header

Step

1. Press CTRL+D. The executable deletes the entire claim from the claim text file.

Active Window = Detail

Step

1. Press CTRL+D. The executable deletes the claim detail line from the claim record.
