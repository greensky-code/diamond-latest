Change the Auto letter distribution method for a group

Use the following procedure to change the Auto Letter distribution method for a group's contact

To change the auto letter distribution method for a group

Press F3, type GROUP, and then press ENTER.

Display the group you want to change. F5 lookup is available

In the AL Dist. Meth list for the contact you want to change, click a distribution method.

If in the AL Dist. Meth. list you selected Email, then in the Email box, type an e-mail address. For example, type bob@bigcompany.com

If in theAL Dist. Meth. list you selected Fax, then in the Fax Numberbox, type the fax telephone number.

On the File menu, click Save.
