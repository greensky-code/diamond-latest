DRG/APC Information Menu Command, contract level

Field Name

Req

Description

Source

Comments

Case 2 % of Allowed

O

Used to calculate a payment for a second case within a single inpatient stay.

Manually entered

Used for case rates when more than one applies to a single claim.

This field is NOT related to DRG pricing.

Case 3 % of Allowed

O

Used to calculate a payment for a third case within a single inpatient stay.

Manually entered

Used for case rates when more than one applies to a single claim.

This field is NOT related to DRG pricing.

DRG Base Multiplier

O

A dollar amount which, when multiplied by a DRG weight, yields the base (before outlier calculations if any) reimbursement amount.

Manually entered

Used for DIAMOND internal DRG pricing.

Not used with external DRG pricers.

DRG Outlier % of Billed

O

Applied to calculate an outlier payment as a percent of the billed amount when the threshold, calculated by multiplying the Outlier Multiplier by the DRG weight, exceeds the total billed amount.

Manually entered

Used for internal DIAMOND DRG pricing.

Not used with external DRG pricers.

DRG Outlier Multiplier

O

Value multiplied by the DRG weight to calculate the threshold over which an outlier payment is required.

Manually entered

Used for internal DIAMOND DRG pricing.

Not used with external DRG pricers.

DRG Pricer ID

O

Identifies location of (path to) the external pricer.

Selected from predefined list

Pricer IDs are maintained in keyword DRGGP.

Not used for internal DIAMOND DRG pricing.

Facility Number (External DRG Setup)

O

Facility number used by a third-party DRG pricer.

Manually entered

You can type up to 16 alphanumeric characters.

See external pricer product documentation for a complete definition.

Facility Number (External APC Setup)

O

Facility number used by a third-party APC pricer.

Manually entered

You can type up to 16 alphanumeric characters.

See external pricer product documentation for a complete definition.

Paysource (External APC Setup)

O

Paysource code used by a third-party APC pricer.

Manually entered

The entry here must match the Pricer paysource in the HSS data files.

You can type up to 13 alphanumeric characters.

Paysource (External DRG Setup)

O

Paysource code used by an external DRG pricer.

Manually entered

The entry here must match the Pricer paysource in the HSS data files.

You can type up to 13 alphanumeric characters.

See external pricer product documentation for a complete definition of this code.

Pricer Version

C

Version of the Pricer identified above.

Selected from predefined list

Required if Pricer ID is entered.

Revision Level

C

Revision level of the Pricer Version identified above.

Selected from predefined list

Required if Pricer Version is entered. Maintained in keyword DRGPR.
