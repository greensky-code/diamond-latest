Comments

Action Code

R

Action taken when a COB Hold/Deny rule applies to a claim detail line

Selected from predefined list

Valid values:

Hold

Deny

Claim Type

R

Type of claims

Selected from a predefined list

Valid values:

Dental

Institutional

Professional

Description

O

Meaningful description of the COB Hold/Deny Rule

Manually entered

 

Determinant Column

R

Column to use as a COB Hold/Deny rule determinant

Selected from predefined list

Valid values:

Any column in the selected Determinant table.

Determinant Table

R

DIAMOND table from which to select individual claim hold release determinants

Selected from predefined list

Valid values:

DENTAL_CLAIM_HEADER (if Claim Type = Dental)

INST_CLAIM_HEADER (if Claim Type = Institutional)

MEMBER_MASTER

MEMBER_OTHER_COVERAGE

PREMIUM_MASTER

PROF_CLAIM_HEADER (if Claim Type = Professional)

From Value

O

Determinant value or beginning of a determinant value range

Manually entered

Important: If you are entering a date(s) in the From/Through determinant fields, you must enter the date in MM/DD/YYYY format (including slashes). For example, 12/25/1999, 02/01/2000.

Operator

O

Mathematical operator for comparison of two values.

Selected from predefined list

Valid values:

Equal

Greater Than

Greater Than (or) Equal

Less Than

Less Than (or) Equal

NOT

Order No

R

COB Hold/Deny rule order number. Establishes the order in which the COB Hold/Deny rules are evaluated.

Manually entered

 

Reason Code

R

The Reason Code assigned when DIAMOND determines this Hold/Deny rule applies to a claim.

Selected from predefined list

Hold reason codes and descriptions are entered in Reason (REASN) with reason code type HD. Deny reason codes and descriptions are entered in Reason (REASN) with reason code type DN.

Sec Determinant Column

R

Specific secondary determinant to which to compare the selected Determinant column

Selected from predefined list

Valid values:

Any column in the selected Determinant table.

Sec Determinant Table

R

DIAMOND table from which to select secondary COB Hold/Deny determinants.

Selected from predefined list

Valid values:

INST_CLAIM_HEADER (if Claim Type = Institutional)

MEMBER_MASTER

MEMBER_OTHER_COVERAGE

PREMIUM_MASTER

PROF_CLAIM_HEADER (if Claim Type = Professional)

Sequence

R

Evaluation sequence number of the COB Hold/Deny rule

Manually entered

 

Thru Value

O

Ending value of a determinant value range

Manually entered

Important: If you are entering a date(s) in the From/Through determinant fields, you must enter the date in MM/DD/YYYY format (including slashes). For example, 12/25/1999, 02/01/2000.
