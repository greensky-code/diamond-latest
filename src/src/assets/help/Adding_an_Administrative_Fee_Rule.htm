Add an Administrative Fee Rule

Use the following procedure to add an administrative fee rule.

To add an administrative fee rule

  1. Press F3, type ADRUL, and then press ENTER.

  2. On the File menu, click new.

  3. In the ID box, type the ID of the rule.

  4. In the Description box, type the rule description.

  5. In the Eff Date box, type the effective date of the rule. If you type a past date, then previously entered claims are not affected.

  6. If you want the rule to eventually termination, then in the Term Date box, type the date the rule terminates, and in the Term Reason list, click the reason code to use when the rule terminates.

  7. Do one of the following:

  8. If you want a flat-amount administrative fee, then in the Amount box, type the amount.

- or -

  • If you want the administrative fee to be a percentage of the claim line’s billed amount, allowed amount, or net amount, then in the Percent box, type the percent. Then, in the Apply Pct to list, click the claim value.

  1. In the Reason Code box, click the reason code to use when the rule is applied.

To add claim selection criteria

  1. In the Claim Type list, click the type of claims for which this rule will evaluate.

  2. If you want to require claims to be clean, then click to select the Require Clean Claim check box.

  3. If you want to require claims to have the Claim Status list set to C – Capitation in the claims detail window, then click to select the Require Cap Line check box.

  4. On the File menu, click Save.

To add determinants

  1. Click to select the focus box that displays the Determinant Table, and Determinant Column boxes.

  2. On the File menu, click New.

  3. In the Determinant Table box, press F5. The Tables and Columns dialog box displays.

  4. In the Table Name box, click a table.

  5. In the Column Name box, click a field.

  6. Click OK.

  7. In the Operator list, click an operand.

  8. In the From Value box, type a beginning value.

  9. If in the Operator list, if you selected Equal or Not Equal, then in the Thru Value box, type an ending value.

  10. On the File menu, click Save.
