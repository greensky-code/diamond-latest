About DRGPR

Use keyword DRGPR (DRG Pricer Revision Maintenance) to establish different pricer revision levels to handle multiple weights as they change over time for a single DRG code. Keyword DRGPR is necessary only if weight information is stored in provider contract records; DRGPR is not used by external DRG pricers.
