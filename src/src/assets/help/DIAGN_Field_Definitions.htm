DIAGN Field Definitions

Field Name

Req

Description

Source

Comments

Case Mgmt

O

Informational indicator of whether ICD code requires case management.

Selected by check box.

Valid values are:

Selected: ICD code requires case management

Cleared: ICD code does not require case management

This field is informational only.

Default Birth Wgt

O

Not currently used.

 

 

Diagnosis Class 1

O

Category of ICD code.

Manually entered.

The meaning is implementation specific; can be used for medical definitions.

Diagnosis Class 2

O

Category of ICD code.

Manually entered.

The meaning is implementation specific; can be used for medical definitions.

Diagnosis Class 3

O

Category of ICD code.

Manually entered.

The meaning is implementation specific; can be used for medical definitions.

Diagnosis Code

R

ICD diagnosis or surgical procedure code.

Manually entered.

You can type up to 30 alphanumeric characters. Unless the insertion point is in this box, then only 7 characters will display.

Edit Flag

O

Medicare Code Editor flag.

Selected from predefined list

Valid values include:

C – Non-covered procedure

M – Manifestation code

N – Non-specific diagnosis

Q – Questionable admit

U – Unacceptable admit

Can be used as a MEDEF determinant, like all Diagnosis Master fields.

Effective Date

O

First date this ICD code is valid.

Manually entered.

In claims and authorizations certain dates, listed below, must fall within the Effective and Term date range.

DNCLM, Svc Date on the header

PSCLM, Svc Date on the header

INCLM, Adm Date on the header.

AUTHS, one of the following dates, searched in the order listed here:

  • Actual Admit Date (Dates window)

  • Req Date (Member/Group/Plan window)

  • Intake Date Time (Intake window)

From Age

O

Youngest patient age to which this ICD code pertains.

Manually entered.

Leave blank if ICD code is not age specific.

In PSCLM, INCLM, DNCLM, and AUTHS an error message displayed if the member’s age does not fall within the From Age and To Age range.

Full Description

O

Complete text description of the ICD code.

Manually entered.

 

Gender

O

Gender to which this ICD code pertains.

Selected from a predefined list.

Leave blank if ICD code is not gender specific.

In PSCLM, INCLM, DNCLM, and AUTHS an error message is displayed if the member’s gender does not match the entry here.

Valid values are:

Male

Female

ID Qualifier

R

Code type that is represented.

Selected from a predefined list.

Hard coded valid values are:

DX: ICD-9-CM Diagnosis Codes

ID: ICD 9 Procedure Codes

ZZ: Mutually Defined Codes

User defined values are maintained in CODES.

MDC Code

O

Major diagnostic category, used to link to DRG codes.

Manually entered.

 

OR Procedure

O

If checked indicates ICD code is a surgical procedure.

Selected by check box.

In PSCLM, INCLM, DNCLM, and AUTHS, an error message is displayed if a code marked as an OR Procedure is entered in a diagnosis field rather than a surgical code field.

Procedure Class

O

Classification which is used by DRG groupers.

Manually entered.

Valid values are:

Class 1 Surgery

Class 2 Significant procedure

Class 3 Significant procedure

Class 4 Other

Short Description

O

Short text description of the ICD code.

Manually entered.

This description is displayed when the Diagnosis Code is entered elsewhere in DIAMOND.

Term Date

O

Last date this ICD code is valid.

Manually entered.

In claims and authorizations certain dates, listed below, must fall within the Effective and Term date range.

DNCLM, Svc Date on the header

PSCLM, Svc Date on the header

INCLM, Adm Date on the header.

AUTHS, one of the following dates, searched in the order listed here:

  • Actual Admit Date (Dates window)

  • Req Date (Member/Group/Plan window)

  • Intake Date Time (Intake window)

To Age

O

Oldest patient age to which this ICD code pertains.

Manually entered.

Used only if From Age is entered.

Leave blank if ICD code is not age specific.

In PSCLM, INCLM, DNCLM, and AUTHS an error message displayed if the member’s age does not fall within the From Age and To Age range.

Trauma Flag

O

Informational indicator of whether the ICD code is related to a trauma case.

Selected by check box.

Valid values are:

Selected: ICD code is related to a trauma case.

Cleared: ICD code is not related to a trauma case.

This field is informational only.
