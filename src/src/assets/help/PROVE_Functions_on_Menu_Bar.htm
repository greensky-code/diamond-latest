PROVE Commands on the Menu Bar

 

Command             Menu   Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Override Enrollment Edits

S

Use to override any or all of the following provider enrollment rule criteria entered on the PROVE record:

  • Accepts New Patients

  • Patient Gender

  • Patient Age

  • Enrollment Limit

  • Access Program Eligible

When one of these criteria is selected, during member PCP assignment, a warning displays when selected criteria "exceptions" occur but you can override the exception and proceed with the member/provider enrollment process.

When one of these criteria is cleared (the default) for any of the above criteria, during PCP assignment, an error message displays when selected criteria "exceptions" occur and you cannot assign the member to the provider.

These overrides do not apply to the PCP Auto Assignment batch process.

You do not have to terminate an existing or create a new PROVE record when you change soft/hard edit criteria using this command.

The status of these criteria are displayed in the Override Edit display in the Provider Enrollment Rule window.

Current Member Enrollment

S

Use to change the Current Enrollment (base) number displayed in the PROVE – Current Enrollment field. DIAMOND adds and subtracts from this number during member/provider enrollment and when certain changes are made to member/provider records.

Enter the new current enrollment (base) number in the dialog box.

Term/Unterminate Provider Enrollment

S

Use to terminate/unterminate the selected PROVE record. You must enter both a termination date and a termination reason in the dialog box.

Modify Enrollment Criteria

S

Use to change the following existing enrollment rule criteria:

  • Accepts New Patients

  • Accepts Existing Patients

  • Gender

  • Age (From/Through)

  • Enrollment Limit
