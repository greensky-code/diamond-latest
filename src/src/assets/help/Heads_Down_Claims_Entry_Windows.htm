Heads Down Claims Entry Windows

You enter claims using the Heads Down Claims Header, Detail, and Lens Prescription windows.

The Header window is a data entry/display area for the following claim header related data:

  • Patient Information

  • Referring Physician Information

  • Diagnosis Information

  • Claim Information

  • Provider Information

  • Vendor Information

The Detail window is a data entry/display area for claim detail line information such as dates of service, place of service, billed amount, etc.

The Lens Prescription window is a data entry/display area for optical lens prescription data.

Unlike DIAMOND’s Professional Services Claims (PSCLM) windows, all entry fields on the Heads Down Claims Entry windows are optional.
