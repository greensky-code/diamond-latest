Enabling Automatic Subscriber ID Generation

Restrictions

Instructions

Frequently Asked Questions

You can have DIAMOND automatically generate a Subscriber ID when you add a new subscriber. You can specify that this occurs when you are adding a subscriber online, using the ANSI 834 Benefits and Member Maintenance transaction, or both.

For more information, see Automatic Subscriber ID Generation, How DIAMOND Thinks.

Restrictions

In order to enable Automatic Subscriber ID Generation, your DBA or other technician must have specified the beginning sequence number for automatic generation when prompted during DIAMOND installation.

Instructions

Step

Comment

1. If desired, enable online Member (MEMBR) automatic Subscriber ID generation by setting system parameter MEMBSUBAUTGEN Param1 = Y.

For more information, see MEMBSUBAUTGEN.

2. If desired, enable API automatic Subscriber ID generation by setting system parameter MEMBSUBAUTGEN Param2 = Y.

For information on API automatic Subscriber ID generation, see the ANSI 834 Benefits and Member Maintenance transaction API Guide.

Frequently Asked Questions

  • Will Automatic Subscriber ID Generation affect existing members and their Subscriber IDs?

No.

  • Does Automatic Subscriber ID Generation affect Change Subscriber ID or DIAMOND ID functionality?

No, neither is impacted. However, if DIAMOND assigns the DIAMOND ID as the Subscriber ID and that is not the desired Subscriber ID, you can delete it and automatic Subscriber ID Generation functionality still applies.
