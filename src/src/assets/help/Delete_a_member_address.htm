Delete a member address

Use this procedure to delete a member address.

To delete a member address

  1. Press F3, type MEMBA, and then press ENTER.

  2. In the DIAMOND ID box, type the DIAMOND ID of the member for whom you want to delete a member address.

  3. In the Subscriber ID box, type the Subscriber ID of the member for whom you want to delete a member address.

  4. In the person number picklist, click the person number of the member for whom you want to delete a member address.

  5. In the Additional Address Information picklist, click the member address you want to delete.

  6. On the File menu, click Delete.

  7. Press CTRL+S to save.
