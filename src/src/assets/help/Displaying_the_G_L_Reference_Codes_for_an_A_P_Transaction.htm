Display the G/L Reference Codes for an A/P Transaction

Use the following procedure to display the G/L reference codes associated with an A/P transaction.

To display the G/L reference codes for an A/P transaction

  1. Press F3, type ACPAY, and then press ENTER.

  2. In the AP Transaction ID box, type the A/P transaction for which you want to display the G/L reference codes. F5 lookup is available.

  3. On the Special menu, click Additional G/L Info.
