From

 

O

Used only with   Region Type Z - Geo Zip.

The first zip code in a range of zip codes.

Manually entered

Valid values are:

A 1 - 5 character alphanumeric value

The length of the value you can enter in this field is determined by the GEOZIPMATCH parameter. For example, if GEOZIPMATCH is set to 3, you can enter 3 characters in this field. If you attempt to enter a longer code, you will receive an error message.

Region Code

G - Geographic

P - Price

S - Service

O

Used only with Region Type Z - Geo Zip.

The last zip code in a range of zip codes.

Manually entered

Valid values are:

A 1 - 5 character alphanumeric value

The length of the value you can enter in this field is determined by the GEOZIPMATCH parameter. For example, if GEOZIPMATCH is set to 3, you can enter 3 characters in this field. If you attempt to enter a longer code, you will receive an error message.

If you want a range to consist of a single zip code, enter the zip code in the From field and leave this field blank.
