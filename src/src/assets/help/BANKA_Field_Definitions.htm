ABA Routing No.

O

Valid ABA routing number for this bank account, from which EFT funds are withdrawn.

Manually entered

You can type up to 15 alphanumeric characters.

  • Savings

Capitation Check Template

O

Name of a check print template used for capitation payments.

Manually entered

Print templates are created using Intelligent Query (IQ). If you don’t use IQ print templates, leave this field blank.

In addition to any implementation specific templates, the following generic IQ template is available:

CHECKS

Capitation RA Template

Manually entered

In addition to any implementation specific templates, the following generic IQ template is available:

RA: For printing separate remittance advices.

Manually entered

 

G/L Cash Account

O

General ledger cash account number, if applicable.

Manually entered

If you do not set up a cash account by entering a number in this field, Diamond assigns credit transaction records a default GL account number of –9D in the general_ledger_interface table.

Medical Check Template

O

Name of a check print template used for medical and capitated (encounter) claims.

Manually entered

Print templates are created using Intelligent Query (IQ). If you don’t use IQ print templates, leave this field blank.

In addition to any implementation specific templates, the following generic IQ templates are available:

  • CHECKS: For printing separate checks on pre-numbered or unnumbered check stock.

  • COMBINED CHECK/RA: For printing combined checks/RAs on pre-numbered or unnumbered check stock. The check is printed on the first page, and subsequent checks, if additional stubs are required to handle RA detail lines, are imprinted with XXXs to render them non-negotiable.
