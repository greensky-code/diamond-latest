About PRULD

Use keyword PRULD (Price Rule Detail) to add or change pricing differences within a Price rule in keyword PRULE (Price Rule). These differences are based on either a claim detail line Modifier code or lack of modifier, or a specific calculation method. You may also use keyword PRULD to define Reason code, default, and override values for each calculation method that is used within the associated Price rule record in keyword PRULE.

DIAMOND selects Price rule details automatically. You can add up to 5 levels and sequence numbers within each calculation method.

Modifier codes with the Informational Only check box selected in keyword MODIF are not available for Price rule detail.
