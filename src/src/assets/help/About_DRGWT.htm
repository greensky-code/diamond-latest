About DRGWT

Use keyword DRGWT (DRG Weights Maintenance) to establish weights for each DRG code. Weights can vary for each code by DRG pricer, version and revision level. This keyword is required when internal DIAMOND DRG pricing is used. It is not used if an external pricer is used.
