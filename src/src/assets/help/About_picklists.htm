About picklists

A picklist is a section of a keyword window that displays a set of records. Column headings are indicated in bold type.

You can do the following in most picklists:

  • Click a column header to sort the records in ascending order. In the above example, you can click Event ID and sort the records by event ID, ascending from A.

  • Press CTRL and click a column header to sort the records in descending order. In the above example, you can press CTRL and then click Event ID to sort the records by event ID, descending from Z.

  • Click a record to display complete information in the window.
