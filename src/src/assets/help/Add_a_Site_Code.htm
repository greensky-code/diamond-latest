Add a Site Code

Use the following procedure to add a Site Code for a line of business, IPA, and Panel combination.

To add a Site Code

  1. Press F3, type SITEM, and then press ENTER.

  2. In the Site Code box, type a site code.

  3. In the Description box, type the description.

  4. In the LineBus box, type a line of business. F5 lookup is available.

  5. In the IPA box, type an IPA. F5 lookup is available.

  6. In the Panel box, type a panel. F5 lookup is available.

  7. Press CTRL+S to save.
