Add History for a Tooth

Use the following procedure to add history for a tooth.

To add history for a tooth

  1. Press F3, type THIST, and then press ENTER.

  2. In the Diamond ID or Member ID boxes, type the DIAMOND or Member ID. F5 lookup is available.

  3. On the File menu, click New.

  4. In the Date of Service box, type the date the service occurred. Type the date in the MM/DD/YYYY format. For example, 12/31/2002.

  5. Enter data in the appropriate fields on the window. For more information, see THIST Field Definitions.

  6. On the File menu, click Save.
