Setting Up Site Validation

To set up Site Validation:

1. Enable Site Validation

2. Add a Site Code

3. Add a site address

4. Specify Site Validation options for a line of business

5. Add a provider address linked to a site address

6. Specify a Site and linked provider address on a provider enrollment rule

To make changes to setup:

7. Change a site address

8. Terminate a site address

9. Change a provider address linked to a site address

1. Enable Site Validation

Enable Site Validation using system parameter MEMSITEVAL.

For more information, see Enable Site Validation.

2. Specify Site Validation options for a line of business

Specify Site Validation options defining when and how validation of site assignments is applied. Options include whether Site is required, whether certification is required, and whether a member/provider enrollment age range match is required for site validation an certification.

For more information, see Specify Site Validation options for a line of business.

3. Add a Site Code

Add a Site Code to use in Site Validation using keyword SITEM (Site Code Master).

For more information, see Add a Site Code.

4. Add a site address

Add a site address to use in Site Validation using keyword SITEA (Site).

Address).

For more information, see Add a site address.

5. Add a provider address linked to a site address

Add a provider address linked to a site address using keyword PROVA (Provider Address).

For more information, see Add a provider address linked to a site address.

6. Specify a Site and linked provider address on a provider enrollment rule

Specify a Site on a provider enrollment rule using keyword PROVE (Provider Enrollment Rule). Site assignment can be required by the line of business.

For more information, see Specify a Site and linked provider address on a provider enrollment rule.

7. Change a site address

Change a site address using keyword SITEA. Changes ripple to all linked provider address records.

For more information, see Change a site address.

8. Terminate a site address

Terminate a site address using keyword SITEA. The termination date then displays on all linked provider address records.

For more information, see Terminate a site address.

9. Change a provider address linked to a site address

Change a provider address linked to a site address using keyword SITEA. In keyword SITEA, you change the site address and your changes ripple to all linked provider addresses.

For more information, see Change a provider address linked to a site address.
