About FACTY

Use FACTY to set up categories of conversion factors, called Factor Types, for which region- and schedule-specific dollar amount conversion factors are established in CONVF. Conversion factors are multiplied by unit values to calculate allowed amounts when unit value pricing applies. A maximum of thirty factor types may be set up. Examples of factor types are:

  • Anesthesia

  • Medical/Surgery

  • Pathology

  • Surgery

  • Radiology
