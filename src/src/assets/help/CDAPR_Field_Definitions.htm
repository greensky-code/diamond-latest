Price on Auth Proc?

O

Indicates whether an amount greater than $0.00 (zero) is required on the authorization PRCDR record to satisfy this CDAPR rule match condition.

Selected by check box

Valid values:

Selected - Authorization procedure detail record must have an Auth Amount greater than $0.00 (zero) to satisfy this CDAPR rule match condition

Clear - Authorization procedure detail record can have an Auth Amount equal to zero, greater than zero, or less than zero and still satisfy this CDAPR rule match condition.

OP – Open (approved)

DN – Denied

HD – Held

CL – Closed

Submitted vs Auth Proc Code

O

Relationship of the submitted procedure code to the authorized procedure code required for this rule to apply.

Selected from predefined list

Valid values:

E - Exact Match – The submitted claim detail line procedure code must be an exact match to the authorized procedure code for this rule to apply

I - In Range – The submitted claim detail line procedure code must be within the corresponding PRCRA ranges specified for this rule to apply.

O - Out of Range – The submitted claim detail line procedure code must be outside the corresponding PRCRA range and the authorized procedure code must be inside the corresponding PRCRA range for this rule to apply.
