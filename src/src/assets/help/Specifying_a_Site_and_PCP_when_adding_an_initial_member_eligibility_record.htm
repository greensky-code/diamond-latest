Specifying a PCP and Site when adding an initial member eligibility record

You can specify a PCP and Site when adding the initial member eligibility record for a new member. The Site field is only available when parameter MEMSITEVAL Parameter 1 = Y and the keyword LINBS Site check box is selected.

To specify a PCP and Site on an initial member eligibility record

  1. Press F3, type MEMBR, and then press ENTER.

  2. Press CTRL+N to create a new member record.

  3. Fill in the fields preceding the PCP ID field as appropriate. For more information, see MEMBR Field Definitions.

  4. In the PCP ID field, press F5 for a lookup on PCP ID. The lookup returns the Provider Address Selection window listing all PCP IDs that have a provider enrollment rule matching the effective period, line of business, Panel, and IPA of the eligibility period, and includes Site in the listing. Select the desired PCP ID/Site combination. DIAMOND logic then populates the PCP ID and Site fields.

  5. Fill in the remaining fields.

  6. Press CTRL+S to save.
