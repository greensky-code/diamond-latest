APUPD Commands on the Menu Bar

 

Command             Menu   Description

Close

F

Close the Accounts Payable Update window.

If there is an open or unsaved A/P update job, you will be prompted to run it.

New

F

Create a new A/P update job.

New from Template

F

Create a new A/P update job using an existing template.

Open

F

Close the active A/P update job, re-opens the job picklist, and allows you to select another batch job.

Run Job

F

Run the active A/P update job.

Save as Template

F

Save the current A/P update job as a template.

Once saved as a template, the batch job will display in a dialog box when the New from Template command is selected from the File menu.
