PCPAA Field Definitions

Field Name

Req

Description

Source

Comments

Days

(Member Reinstated)

C

Maximum number of days prior to the member reinstate effective date that the prior eligibility period can terminate in order for its PCP to be evaluated for priority assignment

Manually entered

Available only if Event = 1 - Validate/Replace Submitted PCP.

If blank, DIAMOND searches for the most recent member eligibility record for the member and evaluates the corresponding PCP for priority assignment.

If the most recent member eligibility record located using the DIAMOND ID has an open termination date, DIAMOND treats this as 1 elapsed day.

Days

(DIAMOND ID)

C

Maximum number of days prior to the current member eligibility effective date that another eligibility period for the same DIAMOND ID can terminate in order for its PCP to be evaluated for priority assignment

Manually entered

Available only if Event = 1 - Validate/Replace Submitted PCP.

If blank, DIAMOND searches for the most recent eligibility record for the member’s DIAMOND ID and evaluates the corresponding PCP for priority assignment.

If the most recent Member record located using the DID has an open termination date, DIAMOND treats this as 1 elapsed day.

DIAMOND ID

(Override Enrollment Threshold Capacity Levels section)

O

Allows provider's enrollment limit capacity to be exceeded for PCP Auto Assignment if the provider is the member’s PCP for another member record (with the same DIAMOND ID)

Selected with check box

Provider’s enrollment limit can only be exceeded up to the provider’s applicable PROVE Enrollment Limit/LINBS Ind PCP/LINBS MaxEnrolLmt/PROVF Max. Enroll Lmt.

DIAMOND ID

(Evaluate Possible Selections section)

O

If selected, PCP Auto Assignment evaluates the PCP from the most recent eligibility record the current member’s DIAMOND ID for priority assignment

Selected with check box

Available only if Event = 1 - Validate/Replace Submitted PCP. Used if submitted PCP does not pass validation.

Order of evaluation among selections chosen in Evaluate Possible Selections depends on the corresponding Order entered.

Must satisfy the corresponding Days criterion.

Event

R

Type of the event that may be evaluated when making a PCP Auto Assignment.

Selected from a predefined list

Valid values are:

1 - Validate/Replace Submitted PCP

2 - Replace Default/No PCP

F5 lookup is available.

Family Affiliation

(Evaluate Possible Selections section)

O

If selected, PCP Auto Assignment evaluates PCPs from family members’ most recent eligibility records for priority assignment

Selected with check box

Available only if Event = 1 - Validate/Replace Submitted PCP. Used if submitted PCP does not pass validation.

Order of evaluation among selections chosen in Evaluate Possible Selections depends on the corresponding Order entered.

DIAMOND searches for other family member records using the search criteria defined on the member’s LINBS record.

Proc Sequence

R

Sequence for evaluation of this rule within the Rule Type

Manually entered

Valid values are 001 - 999.

Each processing sequence is an AND condition except for rules of Rule Type Preference Rule.
