About VENTX

Use keyword VENTX (Vendor TIN) to store the master vendor Tax Identification Number (TIN) information for 1099 reporting.

The name and address that appears in this window is the name and address that will appear on the 1099 form. There can be only one 1099 name and address per TIN. However, there may be several vendor master records associated with one TIN.

A list of vendor master records associated with a TIN is available by selecting the List Vendors command on the Special menu.
