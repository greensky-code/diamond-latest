ARADJ Commands on the Menu Bar

 

Command             Menu   Description

Aradj Post

S

Posts adjustments to the general ledger. Once posted, adjustments cannot be changed, and if in error, must be corrected with additional adjustments. Adjustments with a "zero" or empty Transactions Amount value will fail to post. After posting, DIAMOND will print an A/R Adjustment Posting Report by Customer.

Aradj Report By Customer

S

Prints a report by customer ID for all customers of all posted adjustments that have not yet been reflected on a premium bill.
