About PCPAA

Use keyword PCPAA (PCP Auto Assign) to enter key elements of the PCP Auto Assignment batch process. Key elements are entered separately for each included line of business and PCP Auto Assignment Event Type. PCP Auto Assignment includes two Event Types:

  • Event 1 - Validate/Replace Submitted PCP

  • Event 2 - Replace a Default PCP or No PCP Selection

Key elements include:

  • Build a rules package for PCP Auto Assignment, including specification of the processing sequence

  • Select special conditions when you can override enrollment threshold capacity levels (section Override Enrollment Threshold Capacity Levels)

  • Select priority assignment evaluations to perform in order when a submitted PCP does not pass validation (section Evaluate Possible Selections)
