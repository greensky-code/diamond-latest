Reissue a Posted Check

Use the following procedure to re-issue a posted check. This procedure assumes a check(s) will be reissued for the same amount using keyword CKPRT (Check Print Setup) or MCHEK (Manual Check).

Important Note  Payment on this check must already be stopped and the check's status must be "Stop Confirmed" or "Voided."

To reissue a posted check

  1. Press F3, type CHECK, and then press ENTER.

  2. Select the check or EFT transaction.

  3. On the Special menu, click Stop/Void.

  4. In the Action list, click Reissue.

  5. In the Reason list, click a reason.

  6. Click OK. The check's status changes to "In Process."

  7. On the File menu, click Save.

  8. In keyword CVSUP, run the clear/void/stop job to verify its accuracy, and then post it.
    When you post a clear/void/stop job, the following occurs:

  9. In keyword CHECK, the Check Status field is set to Reissued.

  10. Removes all claim detail line payment data and accounts payable transactions associated with the check(s).

  11. Writes the check(s) status, reason and date that existed prior to running CVSUP to PSCLM/INCLM/DNCLM (Claims) - Notes.
