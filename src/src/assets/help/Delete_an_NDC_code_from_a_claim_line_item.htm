Delete an NDC code from a claim line item

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to delete an NDC code from an unfinalized claim line item.

Restrictions

This feature is only available in keywords DNCLM (Dental Services Claims), INCLM (Institutional Claims), and PSCLM (Professional Claims).

Instructions

Step

Comment

1. On the Claims Detail screen, display the detail line item you want to delete the NDC code.

 

2. On the Special menu, click NDC Codes.

The Attach NDC Codes dialog box displays.

3. Click the NDC Code you want to delete.

 

4. Click Delete.

The code is deleted from the line item.

Frequently Asked Questions

None.
