CLTRN Field Definitions

Field Name

Req

Description

Source

Comments

Action

R

The action to be taken on the run.

Selected from a predefined list

For new Job IDs, selection must be Run.

For existing Job IDs, if Run, Reprint Request must be selected.

Post and Cancel available only after Job ID has Run.

Comments

O

Text describing job.

Manually entered.

 

Date

D

System date.

System displayed.

 

File Type

R

The kind of claims to be transmitted.

Selected from a predefined list

Valid values are

All

Dental

Institutional

Professional services

From Batch Number

O

The first in a range to be included in the run.

Manually entered.

 

From Claim Number

O

The first in a range of claims to be included in the run.

Manually entered.

 

From Vendor ID

O

The first in a range of vendors to be included in the run.

Selected from a predefined list

 

Include Holds

O

Indicator of whether held claims are to be included.

Selected by checkbox

 

Job ID

R

A unique identifier of a batch job.

Manually entered or system generated.

Once a job record is saved the job ID cannot be changed.

Reprint Request

O

If Job ID has run and must be rerun, indication that this is a valid choice.

Selected by checkbox

 

Request Type

R

Tells DIAMOND when to execute the job.

Selected from a predefined list

 

Request User

D

The current signed-on user.

System displayed.

 

Status

D

Status of current job.

System displayed.

 

Through Batch Number

O

The last in a range to be included in the run.

Manually entered.

 

Through Claim Number

O

The last in a range of claims to be included in the run.

Manually entered.

 

Through Vendor ID

O

The last in a range of vendors to be included in the run.

Selected from a predefined list
