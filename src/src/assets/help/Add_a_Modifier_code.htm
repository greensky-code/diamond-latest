Add a Modifier Code

  1. Press F3, type MODIF, and then press ENTER.

  2. On the File menu, click New.

  3. In the Modifier box, type a modifier ID.

  4. In the Description box, type the description of the modifier code.

  5. If this modifier code is for anesthesia services, then click to select the Anesthesia check box.

  6. If this modifier code is informational, only, then click to select the Informational Only check box. This may be convenient if you do not want to affect reimbursements to a provider.

  7. On the File menu, click Save.
