BLHST Commands on the Menu Bar

 

Command             Menu   Description

View Adjustments

Details

S

Shows individual adjustments that equal the adjustment total shown on the selected invoice + adjustments not yet reflected on a bill.

View Invoice Details

Details

S

Shows member-level billing information for the selected invoice.

Group Summary

Details

S

Shows combined group summary records for the selected invoice when the Customer and Group Type are Combined. A group summary record is shown for each billing period processed in the selected date range if the Combined Billing Date field is Yes. If the Common Billing Date field is No, a single record is shown for all periods within the selected date range.
