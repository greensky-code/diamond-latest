Hold Reasons Menu Command

Field Name

Req

Description

Source

Comments

Date/Time

D

Date and time the hold was placed.

Displayed.

 

Description

D

Description of the Reason Code.

Displayed.

 

History

R

Indicator of whether the hold is still in effect.

System generated.

‘H’ means the hold is no longer in effect (=historical); ‘blank’ means the hold is still in effect.

Hold Reason Code

R

The reason code assigned to the claim detail line.

Selected from a predefined list.

F5 lookup is available.

Hold Rsn Seq

D

Sequence of this hold record in the Hold Reason table.

Displayed.

 

Line Number

D

Number of the highlighted claim detail line.

Displayed.

 

Rsn Code

D

Code explaining the nature of the hold.

Displayed.

 

Source ID

D

Indicator of the record or reason that led to the hold.

System generated.

As an example, LIMIT means the hold was assigned during adjudication by a benefit rule of a type LIMIT.

Sub-Line

D

Shows the sub-line number, if one exists.

Displayed.

 

Unclean Indicator

D

If the Unclean Indicator of the reason code record is checked, the Unclean Indicator box will display as checked on this window.

Displayed.

 

User ID

D

Identifies the computer operator who entered the hold reason code.

Displayed.
