TRULE Field Definitions

Field Name

Req

Description

Source

Comments

Arch

O

Displays the word Some, All, or None depending on information you enter in the Arch command on the Special menu.

Displayed.

Valid values:

  • Some

  • All

  • None (default)

The value displayed in this field depends on the information entered in the Arch dialog box that is accessible from the Special menu on TRULE or accessible by pressing Ctrl+A.

Description

O

A description of the tooth rule.

Manually entered.

A 30-character, free-form text field.

Eff Date

R

Date on which this tooth rule becomes effective.

Manually entered.

 

From Age

O

For this rule to match, the member’s age must be equal to or greater than the age you enter here.

Manually entered.

Valid values:

  • A number from 0 – 999.

  • Blank – will match any age.

Oral Cavity

O

Displays the word Some, All, or None depending on information you enter in the Oral Cavity command on the Special menu.

Displayed.

Valid values:

  • Some

  • All

  • None (default)

The value displayed in this field depends on the information entered in the Oral Cavity dialog box that is accessible from the Special menu on TRULE or accessible by pressing Ctrl+O.

Procedure Code

C

Procedure code for which this rule is effective.

Selected from a predefined list.

You must supply either a procedure code or a tooth number, or both.

Quadrant

O

Displays the word Some, All, or None depending on information you enter in the Quadrant command on the Special menu.

Displayed.

Valid values:

  • Some

  • All

  • None (default)

The value displayed in this field depends on the information entered in the Quadrant dialog box that is accessible from the Special menu on TRULE or accessible by pressing Ctrl+Q.

Surfaces

O

Displays the word Some, All, or None depending on information you enter in the Surfaces command on the Special menu.

Displayed.

Valid values:

  • Some

  • All

  • None (default)

The value displayed in this field depends on the information entered in the Surface dialog box that is accessible from the Special menu on TRULE or accessible by pressing Ctrl+S.

Term Date

O

Date on which this tooth rule is no longer effective.

Manually entered.

This date must be greater than or equal to the effective date.

Term Reason

O

The reason code that describes why this tooth rule was terminated.

Selected from a predefined list.

This field is only available if you entered a date into the Term Date field.

Only reason codes of type TM can be entered.

To Age

O

For this rule to match, the member’s age must be less than the age you enter here.

Manually entered.

Valid values:

  • A number from 0 – 999.

  • Blank – will match any age.

 

Tooth Number

C

Tooth number for which this rule is effective.

Selected from a predefined list.

You must supply either a tooth number or a procedure code, or both.

Tooth Rule ID

R

The unique tooth rule ID number.

Manually entered.

A 15-character, alphanumeric identifier.

F5 lookup is available.

User Date 1 and 2

O

Use of these fields is determined by your organization.

Manually entered.

The literals for these fields are defined in keyword UDTXT (User Defined Fields).

User Defined 1 and 2

O

Use of these fields is determined by your organization.

Manually entered.

The literals for these fields are defined in keyword UDTXT (User Defined Fields).
