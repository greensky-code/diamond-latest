About PREMM

Use keyword PREMM (A/R Matrix Premium Rate Entry) to enter premium rates for groups that use either the subscriber or Whole Family premium billing tiering method (GRUPB>Tiering Mthd).

This keyword, using a matrix from MATRX which defines the member and subscriber attributes to which different rates apply, is used to enter and maintain the actual dollar amount of the rates which the premium billing program will use when billing each group.
