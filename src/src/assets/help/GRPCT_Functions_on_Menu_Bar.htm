GRPCT Commands on the Menu Bar Command  Menu  Description

D/C Information

S

Used to store account information for groups using Electronic Funds Transfer (EFT).

For detailed field information about D/C Information, see GROUP reference documentation.

Edit Billed Through Date

S

A way to manually adjust the group’s premium Billed Through date, to be used only if the Group Type on the GROUP record is 1, Group. If this command is used, the GRPCT record must be saved, CTRL+S, in order for the new Billed Through date to apply.

Edit Paid Through Date

S

A way to manually adjust the group’s premium Paid Through date, to be used only if the Group Type on the GROUP record is 1, Group. If this command is used, the GRPCT record must be saved, CTRL+S, in order for the new Paid Through Date to apply.
