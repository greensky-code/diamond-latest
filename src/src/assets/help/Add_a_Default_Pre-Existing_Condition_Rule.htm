Add a default pre-existing condition rule

Use the following procedure to add a default pre-existing condition rule. You add these rules using keyword PXRUL (Pre-Existing Condition Rule).

Default pre-existing condition rules can be applied at the line of business and group level. You can associate a pre-existing condition rule with specific groups in a line of business using keyword LBPEC (Line of Business Pre-Existing Conditions).

LBPEC pre-existing condition rules are considered "default" rules for the group and line of business in that DIAMOND applies LBPEC pre-existing condition rules only when no member-specific pre-existing condition rules apply.

To add a default pre-existing condition rule

  1. Press F3, type PXRUL, and then press ENTER.

  2. Fill in the fields on the window. For more information, see PXRUL Field Definitions.

  3. Press CTRL+S to save.
