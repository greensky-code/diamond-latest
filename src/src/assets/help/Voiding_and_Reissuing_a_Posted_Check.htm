Void and Reissue a Posted Check

Use the following procedure to void and re-issue a posted check. This procedure assumes a check will be reissued for the same amount using keyword CKPRT (Check Print) or MCHEK (Manual Check).

Important Note  In keyword CHECK (Check Register), the check status must be "Posted."

To void and reissue a posted check

  1. Press F3, type CHECK, and then press ENTER.

  2. Select the check or EFT transaction.

  3. On the Special menu, click Stop/Void.

  4. On the Action list, click Void//Reissue.

  5. In the Reason list, click a reason.

  6. Click OK. The check's status changes to "In Process."

  7. On the File menu, click Save.

  8. In keyword CVSUP, run the clear/void/stop job to verify its accuracy, and then post it.
    When you post a clear/void/stop job, the following occurs:

  9. In keyword CHECK, the Check Status field is set to Reissued.

  10. In the claim detail window for each affected claim detail line, the Proc Stat list is set to F – Final (posted).

  11. In keyword ACPAY, the original claim’s A/P transaction record has the Processing Status box set to R, the Select For Payment list set to Selected, and the Check Number and Check/EFT Dt fields are cleared.

  12. The check status, reason, and date that existed prior to running CVSUP is written to CHECK - Notes.
