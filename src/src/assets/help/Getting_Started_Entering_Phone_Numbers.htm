Entering a Telephone Number

The DIAMOND program uses pre-loaded U.S. Standard Attributes and Edits that control:

  • How you enter numbers in a phone number field on keyword windows.

  • How telephone numbers are displayed on keyword windows.

Enter a telephone number as digits, without punctuation; i.e., no parentheses or hyphens. DIAMOND automatically adds parentheses and hyphens in the phone number field after you type the number. If you type 212555212, then (212) 555-1212 displays. If you type 5551212, then (    ) 555-1212 displays.

If you want to enter telephone numbers with formats other than the U.S. Standard, then you can override these internal pre-loaded attributes and create customized telephone number attributes. For more information, see UDATT, User Defined Attributes.
