Case 2 % of Allowed

O

Used to calculate a payment for a second case within a single inpatient stay.

Manually entered

Used for case rates when more than one applies to a single claim.

This field is NOT related to DRG pricing.

O

Manually entered

Used for case rates when more than one applies to a single claim.

This field is NOT related to DRG pricing.

DRG Base Multiplier

O

A dollar amount which, when multiplied by a DRG weight, yields the base (before outlier calculations if any) reimbursement amount.

Manually entered

Used for DIAMOND internal DRG pricing.

Not used with external DRG pricers.

DRG Outlier % of Billed

O

Applied to calculate an outlier payment as a percent of the billed amount when the threshold, calculated by multiplying the Outlier Multiplier by the DRG weight, exceeds the total billed amount.

Manually entered

Used for internal DIAMOND DRG pricing.

Not used with external DRG pricers.

DRG Outlier Multiplier

O

Value multiplied by the DRG weight to calculate the threshold over which an outlier payment is required.

Manually entered

Used for internal DIAMOND DRG pricing.

Not used with external DRG pricers.

DRG Pricer ID

Selected from predefined list

Pricer IDs are maintained in DRGGP.

Not used for internal DIAMOND DRG pricing.

Pricer Facility Number

O

Facility number used by an external DRG pricer.

Manually entered

You can type up to 16 alphanumeric characters.

See external pricer product documentation for a complete definition.

Pricer Pay Source

O

External pricer pay source code used by an external DRG pricer.

Manually entered

You can type up to 13 alphanumeric characters.

The entry here must match the Pricer Pay Source in the HSS data files.

See external pricer product documentation for a complete definition of this code.
