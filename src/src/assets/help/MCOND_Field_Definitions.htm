DIAMOND ID

 

Unique identifying number.

If you use this feature, this is a single identifier for every instance of a person in the Member Master records.

Manually entered

A DIAMOND ID enables dual coverage.

If this person is covered by more than one insurance, the other instances of the person appear in a popup window. You can select the appropriate one from the list.

For more information, see Dual Coverage, How DIAMOND Thinks.

Term Reason

O

Code explaining the reason for termination of this member condition record.

Manually entered

Required if there is a date entered in the Term Date field.

F5 lookup is available.

Termination reason codes are maintained in keyword REASN with reason code type TM.

From Value

R

Diagnosis code or class value, or the beginning of a range of diagnosis codes or class values for which this member condition record applies.

Selected from predefined list

F5 lookup is available.

Diagnosis codes and class values are maintained in keyword DIAGN.

Surgical procedure diagnosis codes are not valid diagnosis code values for this field.

Thru Value

O

End of a range of diagnosis codes or class values for which this member condition record applies.

Selected from predefined list

F5 lookup is available.

Diagnosis codes and class values are maintained in keyword DIAGN.

Surgical procedure diagnosis codes are not valid diagnosis code values for this field.

Condition Type

R

The type of member condition.

Selected from predefined list

Valid values are maintained in keyword CODES (System Codes Maintenance), with System Code Type MEMCONDTYPE.

Predefined valid values:

  • Pre-existing

  • Privacy

  • Rated-up

  • Ridered-out

Claim Action

O

Action taken when a claim satisfies the criteria of this member condition record.

Selected from predefined list

Valid values:

  • D - Deny

  • H - Hold

  • P - Pay

 

Reason

O

Reason code reflecting the reason the Claim Action is being applied.

Selected from predefined list

Valid values are maintained in keyword REASN with reason code type HD.
