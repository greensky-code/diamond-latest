VADPY Parameters and Prerequisites

The following is only a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

Parameter

Description

Comments

VENDSECAPP

Enables the user to define whether or not special security requirements are required

to approve or activate an advance payment account.

Yes - indicates that security is required and the VADPY logic will keep the account

in an inactive status until this approval takes place.

 

No - indicates that security is not required and the advance payment record can be

active without the approval of a second user.
