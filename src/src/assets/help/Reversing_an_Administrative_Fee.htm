Reversing an Administrative Fee

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to reverse an administrative fee.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type ADDSP, and then press ENTER.

 

2. Select the administrative fee for which you want to reverse.

 

3. On the Special menu, click Reverse Administrative Fees.

 

4. In the Administrative Fee Display Selection dialog box, type information in the fields, and then click OK.

 

5. On the File menu, click Save.

 

Frequently Asked Questions

None.
