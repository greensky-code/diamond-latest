Disch Date

R

Date of discharge for this claim.

Manually entered

Used as part of the members eligibility check and may be used when DIAMOND calculates Benefit Renewal Periods (see Benefit Rules and Packages user documentation, section Benefit Renewal Periods for more information).

You can change this date if the claim has not been finalized; however, when you change this date after the claim has been priced and adjudicated:

  • The changed date must be later than or equal to the INCLM-Detail-Thru date on any of the associated claim detail lines. If not, an error message is displayed and you will not be able to save the record.

  • DIAMOND displays a message warning that associated benefit renewal periods may be affected by the date change and instructs you to reprice and readjudicate the claim. See Benefit Rules and Packages user documentation, section Benefit Renewal Periods for more information.

DRG

O

Code of the diagnosis related grouping for this claim.

System generated or manually entered

You can type up to 5 alphanumeric characters.

If a 5-character code is submitted, the first character is assumed to be the Severity Indicator. The Severity Indicator may be 0 - Baseline, 1 - non-major CC, or 2 - major CC. the DIAMOND system does not validate the severity indicator code, nor does it use the code for any processing functionality.

When a 5-character DRG code is submitted either online or through the DDS API Institutional transaction set, the DIAMOND system strips the first position and tests the resulting 4 characters. If the system finds a matching DRG code, it continues processing; if not, it provides a critical error message indicating that the DRG code is not valid.

If external DRG grouping software is used, then this field is system generated.

The primary diagnosis code must go in Dx 1. Entered codes are validated against the members gender/age and a warning is issued if the code is flagged for case management. DIAMOND displays an error message if codes do not pass validation.

Diagnosis codes are maintained in keyword DIAGN (Diagnosis Code).
