About SFNDS

Use keyword SFNDS (Function Description) to display the list of commands on the menu bar you can grant or deny user access in keyword SFUNC (Function Access). These commands display in the Function list in keyword SFUNC.

The only circumstance in which you might update these entries is if you have created software external to DIAMOND and without Perot Systems Healthcare DIAMOND Solutions assistance, you want those commands to appear on DIAMOND menus and work with DIAMOND security. At some point you would add those externally created commands to keyword SFNDS and then make the appropriate selections in keyword SFUNC.
