Entering a Country

The DIAMOND program uses internal pre-loaded U.S. Standard Attributes and Edits that hide the Country field from display on applicable keyword windows.

You can override these internal pre-loaded U.S. Standard Attributes and Edits and display the Country field on applicable windows. You can then enter a country as part of the address information. For more information, see UDATT, User Defined Attributes and CNTRY, Country.
