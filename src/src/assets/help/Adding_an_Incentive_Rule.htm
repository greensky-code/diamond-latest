Add an Incentive Rule

Use the following procedure to add an incentive rule. To add an incentive rules, you must do the following:

Add an incentive rule

Select an access program

Select a quality program

To add an incentive rule

  1. Press F3, type INRUL, and then press ENTER.

  2. On the File menu, click New.

  3. In the Description box, type the description of the rule.

  4. In the Line of Bus box, type the provider’s line of business. F5 lookup is available.

  5. In the IPA box, type the provider’s IPA. F5 lookup is available.

  6. In the Panel box, type provider’s panel. F5 lookup is available.

  7. In the Effective Date box, type the effective date of the rule. The rule is effective on the first unposted capitation month after the month you typed. Capitation months are posted in keyword CCALC (Capitation Calculation Setup). If you type a past date and the capitation month for that date is retroactively calculated, then the rule will not be applied to the retroactive calculation.

  8. If you want the rule to eventually termination, then in the Term Date box, type the date the rule terminates, and in the Term Reason list, click the reason code to use when the rule terminates.

To select an access program

  1. Click to select the Access Program check box.

  2. If you want to enter a percent to be multiplied by the total capitation amount to determine the amount to be included or deducted from the capitation payment, then in the Pct box, type a percent and skip to step 5.

  3. In the Amt box, type the amount to multiplied by the provider’s enrollment member count in used in capitation or to be included as a single amount in the capitation payment.

  4. In the Apply Amt to list, do one of the following:

  5. If you want to multiply the amount in the Amt box by the number of enrolled members included in the provider’s capitation calculation, then click M – Apply per member.

  6. If you want to include the amount in the Amt box to the capitation payment only once, then click P – Apply once.

  1. If you want providers to have a minimum number of members to quality for the access program, then click to select the Minimum Members check box. In the Minimum number box, type the number of members the provider must have to qualify for the access program.

  2. If you want to include providers that have not met the line of business individual enrollment limit entered in the Max Enroll Lmt box in keyword PROVF (Provider Master), then click to select the Total enroll within limits check box.

  3. If you want to include providers in the same line of business as this incentive rule that are accepting new members, then click to select the LOB enroll with limits check box.

  4. If you want to include providers with provider contract records in keyword PROVC (Provider Contract) that are associated with a line of business record in keyword LINBS (Line of Business) that specify an enrollment limit, then click to select the LOB accepts new members check box.

  5. On the File menu, click Save.

To select a quality program

  1. Click in the Quality Program Information focus box.

  2. On the File menu, click New.

  3. In the Quality Program list, click the quality program.

  4. If you want to enter a percent to be multiplied by the total capitation amount to determine the amount to be included or deducted from the capitation payment, then in the Percent box, type a percent and skip to step 7.

  5. In the Amount box, type the amount to multiplied by the provider’s enrollment member count in used in capitation or to be included as a single amount in the capitation payment.

  6. In the Apply Amount to list, do one of the following:

  7. If you want to multiply the amount in the Amt box by the number of enrolled members included in the provider’s capitation calculation, then click M – Apply per member.

  8. If you want to include the amount in the Amt box to the capitation payment only once, then click P – Apply once.

  1. On the File menu, click Save.

  2. Repeat steps 1 through 7 for each quality program you want to select.
