About FINAL

Use keyword FINAL (Final Non-Ap Setup) to run a finalize job and finalize non-payable claim detail lines.

Non-payable claim detail lines are characterized by the Claim Status list set to one of the following values in the claim detail window:

  • C – Capitated

  • S – Staff

  • I – Informational

When you run a finalize job, the Proc Status list is changed to F – Final (posted) in the claim detail window for each finalized claim detail line. In addition, the Post Date box is set to the date the finalize job ran. The finalized detail lines are then available for inclusion in capitation deduction and capitation stoploss calculations.

About claims with informational detail lines

A finalize job skips the informational detail lines on a claim if one of the following is true:

  • The claim does not have a system-generated detail line

- or -

  • The claim has a system-generated detail line that is unposted (the Proc Status list is set to U - Unposted). Generally, you can run an A/P update job in keyword APUPD (Accounts Payable Update) to finalize system-generated claim lines.

- or -

  • The claim has a system-generated detail line that is held (the Proc Status list is set to H - Held)

By skipping a detail line, the Proc Status list remains set to U - Unposted (open).
