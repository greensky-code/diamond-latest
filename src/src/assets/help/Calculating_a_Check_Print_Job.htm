Calculate a Check Print Job

Use the following procedure to calculate a check print job. By calculating a check print job, you are calculating check and electronic funds transfer (EFT) transactions.

To calculate a check print job

  1. Press F3, type CKPRT, and then press ENTER.

  2. On the File menu, click New.

  3. In the Payable Type list, do one of the following:

  4. If you want to print checks for capitation transactions, then click C – Capitation.

  5. If you want to print checks for medical claims, then click M – Medical.

  1. In the Item Selection Criteria section box, fill in the fields on the screen. For more information, see CKPRT Field Definitions.

  2. In the Next Action section box, click Calc.

  3. On the File menu, click Run Job.
