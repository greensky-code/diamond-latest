About BENEF

Use keyword BENEF (Benefit Package) to associate a set of benefit rules with a package that can be linked to a plan code and employer group. Each subscriber and member has an associated employer group, plan code, and benefit package.

To create working benefit packages, both BENEF and Benefit Processing Order, BORDR, must be set up.
