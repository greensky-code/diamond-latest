Add an NDC code

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to add an NDC (National Drug Code) code.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type NDCCD, and then press ENTER.

 

2. On the File menu, click New.

 

3. In the NDC Code box, type an 11-digit code ID, and then press TAB.

DIAMOND automatically inserts a dash after the 5th and 9th characters.

4. In the Description box, type the code description (optional).

 

5. In the Drug Class box, type the class of the drug (optional).

 

6. In the Avg Wholesale Price box, type the average wholesale price of the drug (optional).

Type up to 16 digits to the left of the decimal and 2 digits to the right of the decimal.

7. In the Effective Date box, type the date this drug code is effective (optional).

The valid format is MM/DD/YYYY

For example, 11/25/2001

If you do not set up an NDC code with an effective date, then the code is considered effective on the claim line item’s date of service.

8. In the Term Date box, type the date this drug code becomes ineffective (optional).

The valid format is MM/DD/YYYY

For example, 11/26/2001

9. On the File menu, click Save.

 

Frequently Asked Questions

None.
