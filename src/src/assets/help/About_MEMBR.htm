About MEMBR

Use keyword MEMBR (Member Master) to add new members, both subscribers and dependents, or to change existing member data.

The data you enter in keyword MEMBR updates information in two tables: the Member table and the Member Eligibility History table. The Member table contains basic demographic information: name, date of birth, gender, address, phone number, etc. The Member Eligibility History table contains information about the member’s benefit coverage(s) during a specific period(s) of time.

When you first add a member, the eligibility portion of the MEMBR window is available for data entry. Once you save the new member record, the eligibility data is protected from data entry and can be modified only by using commands on the Special menu.
