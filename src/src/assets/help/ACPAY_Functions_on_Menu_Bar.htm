ACPAY Commands on the Menu Bar

 

Command             Menu   Description

Additional G/L Info

S

Display the incentive program and administrative fee G/L codes associated with the currently displayed A/P transaction record.

Admin Fee Info

S

Display keyword ADDSP (Administrative Fee Display). Use to view the administrative fee associated with the currently displayed A/P transaction record.

Notes

Details

N

This command is only available if the ACPAY window has been associated with at least one note type in NOTEW.

Displays the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.

View Additional G/L Accounts

Details

S

Displays penalty General Ledger account numbers.

View AP Additional Info

S

Displays additional information associated with the currently displayed A/P transaction record:

Check Number or EFT Trans Number

Offset indicator

Check EFT Amount
