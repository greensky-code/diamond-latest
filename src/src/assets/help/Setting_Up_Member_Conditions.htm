Setting Up Member Conditions

To set up member conditions functionality:

1. Add a member condition

2. Add a member privacy address

3. Add a member privacy contact

To make changes to set up:

4. Change a member condition

5. Delete a member condition

6. Change a member privacy address

7. Delete a member privacy address

8. Change a member privacy contact

9. Delete a member privacy contact

1. Add a member condition

Add a member condition using keyword MCOND (Member Conditions).

Learn how to add a member condition.

2. Add a member privacy address

Add a member privacy address using keyword ALIAS (Alias/Responsible Party/Privacy)

Learn how to add a member privacy address.

3. Add a member privacy contact

Add a member privacy contact using keyword ALIAS.

Learn how to add a member privacy contact.

4. Change a member condition

Change a member condition using keyword MCOND.

Learn how to change a member condition.

5. Delete a member condition

You cannot delete a keyword MCOND member condition once the record is saved. However, you can use keyword MCOND to limit the effective period of the member condition record to a single day. To do so, in MCOND section box Specifications enter a Term Date equal to the Effective Date.

Learn how to delete a member condition.

6. Change a member privacy address

Change a member privacy address using keyword ALIAS.

Learn how to change a member privacy address.

7. Delete a member privacy address

Delete a member privacy address using keyword ALIAS.

Learn how to delete a member privacy address.

8. Change a member privacy contact

Change a member privacy contact using keyword ALIAS.

Learn how to change a member privacy contact.

9. Delete a member privacy contact

You cannot delete a member privacy contact. You can however use keyword ALIAS to blank out privacy contact fields Email Id and Fax Number.

Learn how to delete a member privacy contact.
