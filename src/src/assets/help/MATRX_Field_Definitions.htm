Field Name

Req

Description

Source

Comments

Age From

O

If this determinant is a range of member ages, this is the earliest member age to which the premium rate will apply.

Manually entered

Default is 0.

Age From

O

If this determinant is based on the member’s age, this is the lowest age to which the premium will apply.

Manually entered

 

Age To

O

If this determinant is a range of member ages, this is the oldest member age to which the premium rate will apply.

Manually entered

Default is 0.

Entry in this field must be greater that entry in the Age From field.

Age To

O

If this determinant is based on the member’s age, this is the oldest age to which the premium will apply.

Manually entered

Text description of this determinant.

Manually entered

This description will appear in PREMM, where actual rate dollar amounts are entered.

Description

R

Text description of this determinant.

Manually entered

This description will appear in PREMM, where actual rate dollar amounts are entered.

Family Size From

O

If this determinant is based on the subscriber’s family size, this is the smallest family size to which the premium will apply.

Manually entered

 

Family Size To

O

If this determinant is based on the subscriber’s family size, this is the largest family size to which the premium will apply.

Manually entered

 

Gender

O

If this determinant is based on member gender, this is the gender to which the premium rate will apply.

Selected from predefined list

Hard coded values are:

Female

Male

Blank: (Not applicable)

Gender

If this determinant is based on member gender, this is the gender to which the premium rate will apply.

Manually selected

Hard coded values are:

Female

Male

Blank: (Not applicable)

Matrix Description

O

Text description of this matrix determinant.

Manually entered

This description will appear in PREMM, where actual rate dollar amounts are entered.

Matrix Determinant

R

Identifier of the matrix.

Manually entered or selected from a predefined list

 

Medicare Status

O

If this determinant is based on the member’s Medicare Status, this is the value of Medicare Status to which the premium will apply.

Manually entered

All 11 digits of the Medicare Status must be entered.

Rate Type

O

An optional code to indicate a specific rate type to be used during premium billing rate calculation.

Selected from predefined list after user defined values are set up in keyword CODES.

Rate Type codes are maintained in keyword CODES. Examples of user defined rate types might be:

Rate0

Rate1

Rate2

Rate3

See the following online documentation for more information:

Premium Billing>How DIAMOND Thinks>Member-by-Member (Whole Family) Family Billing

Premium Billing>Rate Matrix Examples>Member-by-Member Family Rates by Member Age/Rate Type

Salary From

O

If this determinant is based on the member’s salary, this is the beginning salary of a range.

Manually entered

You can type up to 16 digits before the decimal and 2 digits after the decimal.

Salary To

O

If this determinant is based on the member’s salary, this is the highest salary of a range.

Manually entered

You can type up to 16 digits before the decimal and 2 digits after the decimal.

Spouse Flag

O

If this box is selected, the premium rate will apply only if the subscriber has a spouse or domestic partner.

Selected from check box

 

X-Matrix

D

Sequential column number.

System generated

Deleting column determinants: Before a MATRX record is saved you may remove a column determinant by placing the cursor in one of the column fields and selecting File, Delete.

Y-Matrix

D

Sequential row number.

System generated

Deleting row determinants: Before a MATRX record is saved you may remove a row determinant by placing the cursor in one of the row fields and selecting File, Delete.
