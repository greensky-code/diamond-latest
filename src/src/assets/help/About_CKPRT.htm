About CKPRT

Use keyword CKPRT (Check Print Setup) to run a check print job that calculates check and EFT payments, prints checks, creates EFT transaction workfiles, and provides source information for EOBs, RAs, and EOPs.

Checks are printed for vendors with an A/P balance equal to zero or greater dollars ($0.00+).

You can also use keyword CKPRT to cancel a check print job, when necessary.
