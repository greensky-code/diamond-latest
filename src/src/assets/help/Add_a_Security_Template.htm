Add a Security Template

Use the following procedure to add a security template. Later, you can grant privileges to the template and then add users assigned to the template. Users assigned a template exclusively use the template's privileges.

To add a security template

  1. Press F3, type SUSER, and then press ENTER.

  2. In the User ID box, type an ID for the template, and then press TAB.

  3. Click to select the Template check box.

  4. In the Description box, type the description of the template.

  5. If desired, fill in the fields in the User Information section box.

  6. If you want this template to have unlimited security privileges, then click to select the Super User check box. The Field Level Security ID box and Row Level Security box become unavailable for entry.

  7. In the User Type list, click Operator.

  8. In the Field Level Security box, type the field level security ID.

  9. In the Language list, click the default language.

  10. If you want this template to have access to claims with a security code equal to or less than a particular value, then in the Row Level Security box, type the security code. You can type a number from 0 to 9.
    For example, if you type 5, then this template only has view/change privileges on claims that have the Security Code box set to 5, 4, 3, 2, 1, or 0. The Security Code box displays in keywords CLDSP, DNCLM, DSPBN, INCLM, and PSCLM.

  11. On the File menu, click Save.

  12. If desired, do the following (except super user templates):

  13. If you want this template to only access claim records with a particular group, then on the Special menu, click Group Affiliation.

  14. If we want this template to only access claim records with a particular panel, then on the Special menu, click Panel Affiliation.

  15. If you want this template to only access claim records with a particular provider, then on the Special menu, click Provider Affiliation.

  16. If you want this template to only access claim records with a particular vendor, then on the Special menu, click Vendor Affiliation.

  17. If you want this template to only access claim records with a particular IPA, then on the Special Menu, click IPA Affiliation.

  18. If you want this template to only access claim records with a particular line of business, then on the Special menu, click LOB Affiliation.
