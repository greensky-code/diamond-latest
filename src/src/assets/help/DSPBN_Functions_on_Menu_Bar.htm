DSPBN Commands on the Menu Bar

 

Command             Menu   Description

Benefit Value Filters

S

View Benefit Value Filters that are associated with the Benefit Rule. Benefit Value Filters consist of Par, Auth, and Rider codes.

Claim Audit Details

Details

S

For the claim and claim line selected in the keyword, displays summary information for:

  • Claim detail lines that were impacted by the CodeReview evaluation

  • Claim detail lines that caused CodeReview to change the claim.

Patient Liabilities

S

Displays the patient liability window, which shows underlying copay, coinsurance, limit, deductible, and exclusion benefit rules and amounts for detail lines to which more than one benefit rule of the same benefit type was applied. When more than one rule was applied, the benefit reason code field on the claim detail is filled with asterisks.

Select Claim

S

An alternative means of displaying claim header and detail data for the claim highlighted in the Benefit Detail focus box.

Important - You cannot view the processing messages associated with a claim from DSPBN - Select Claim. Use PSCLM, DNCLM, or INCLM to view processing messages.

Select Member

S

An alternative means of displaying Select Member data entry window.
