BLHST View Invoice Details Menu Command

Field Name

Req

Description

Source

Comments

# Adult Dep

D

Number of dependents whose age is equal to or greater than the Adult Dependent Age associated with the group.

 

 

# Dep

D

Number of dependents in the family.

 

 

Amount

D

Premium amount for this member and this plan or rider.

 

DIAMOND may truncate the display of the currency amount.

Family Size

D

Number of members in the family.

 

 

First Name

D

Member’s first name.

 

 

From

D

DIAMOND displays the From date associated with the premium amount for the line.

 

 

Invoice #

D

Invoice number.

 

 

Last Name

D

Member’s last name.

 

Displays up to 35 alphanumeric characters.

Person #

D

Person number with whom the invoice details are associated.

 

 

Plan/Rider

D

Benefit Plan or Rider code.

 

 

Through

D

DIAMOND displays the Through date associated with the premium amount for the line.

 

 

Type

D

Indicator of whether the premium information is related to a Benefit Plan, Rider, or Admin Fee.

 

Values include:

P = Plan

R = Rider

F = Admin Fee (From GRUPB>Admin Fee)

Admin fees billed to the group, for the group are listed next to the first member in the group with a Type = F.
