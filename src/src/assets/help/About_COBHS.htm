About COBHS

Use keyword COBHS (Member COB History) to document alternate or supplemental coverage for a member or family unit. If COB information is entered for a member who has primary coverage with another carrier, each time a claim is entered for that member DIAMOND can take various actions based on this other coverage information. For example, you can set up various conditions to have DIAMOND:

  • Display a COB warning message

  • Display a COB message, automatically hold the claim and assign a selected hold reason

  • Display a COB message, automatically deny the claim and assign a selected deny reason

  • Take other actions - See Coordination of Benefits Overview User Documentation for more information.
