About PCPSI

Use PCPSI (PCPAA Support Info Details) to define Specialty age range and gender restrictions for use in PCP Auto Assignment rules. You can specify:

  • Age range restrictions for assignment of PCPs with primary Pediatric specialties

  • Age range and gender restrictions for assignment of PCPs with primary OB/GYN specialties

  • Gender exclusions for assignment of PCPs with any specified primary specialty

  • Age range exclusions for assignment of PCPs with any specified primary specialty

You only use PCPSI indicator and restriction data in PCP Auto Assignment rules. DIAMOND does not otherwise use PCPSI information.
