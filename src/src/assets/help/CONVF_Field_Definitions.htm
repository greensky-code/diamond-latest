CONVF Field Definitions

Field Name

Req

Description

Source

Comments

Conversion Factors

R

Dollar amounts corresponding to each factor type.

Manually entered.

Factor Types are set up in FACTY.

Description

O

Text description of this conversion factor schedule.

Manually entered.

 

Effective Date

R

First date on which the conversion factors entered in this record are applicable.

Manually entered.

 

Price Region

O

Geographic region. If entered, indicates that pricing varies by region.

Manually entered.

Regions are maintained in REGIN.

Price Schedule

R

Set of pricing information.

Selected from a predefined list.

Price Schedules are maintained in PSCHD.

Scale Type

R

Origin of the unit values used, such as Medicare, McGraw-Hill, RBRVS.

Selected from a predefined list.

Scale values are maintained in SCALE.
