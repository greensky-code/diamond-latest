DIAMOND ID

O

Unique Identifying Number

There is one identifier for every instance of a person in the Member Master records

Manually entered

Use of a DIAMOND ID allows tracking of more than one instance of coverage for a person in DIAMOND.

If this person has more than one insurance coverage, the other instances of the person appear in a popup and the appropriate one can be selected from the list.

For more information, see Dual Coverage, How DIAMOND Thinks and Dual Coverage - Overview.

Distr.Method

C

Displays the preferred method of auto letter distribution.

 

Selected from a predefined list

Select the alias, responsible party, or privacy contact’s preferred method of receiving auto letter correspondence: printed, faxed, or sent in email.

This field is required if you select a contact in the Contact Source list.

For more information about auto letters, see Auto Letter, How DIAMOND Thinks.

Email ID

 

The email address of the alias or responsible party.

Manually entered

If in the Distr.Method list you selected E-mail, then this e-mail address is used to send auto-letter correspondence. If E-mail is not selected in the Distr.Method list, then this field is for informational purposes.

For more information about auto letters, see Auto Letter, How DIAMOND Thinks.

Fax Number

 

The fax telephone number for the alias or responsible party.

Manually entered

If in the Distr.Method list you selected Fax, then this telephone number is used to fax auto-letter correspondence. If Fax is not selected in the Distr.Method list, then this field is for informational purposes.

For more information about auto letters, see Auto Letter, How DIAMOND Thinks.
