Adding Geo Zip Regions

Use the following procedure to create Geo Zip Regions and their associated zip code ranges. Geo Zip Regions are used for Geo Zip Calculation Method pricing.

To Create a Geo Zip Region:

  1. Access the REGIN (Region) Screen.

  2. In the Region Type field, type Z for Geo Zip.

  3. In the Region Code field, type a unique identifying code.

  4. In the Description field, type a description of the Geo Zip Region.

  5. To access the From and Through fields of the Geo Zip Code Ranges section box, click on the Geo Zip Code Ranges section box, then press CNTL + N or from the File menu, select New.

  6. In the From and Through fields, type beginning and ending codes for each zip code range you want to define for this Geo Zip Region. If you want to enter a single zip code instead of a range, enter the zip code in the From field and leave the Through field blank.  

Note

The length of the value you can enter in this field is determined by the GEOZIPMATCH parameter. For example, if GEOZIPMATCH is set to 3, you can enter 3 characters in this field. If you attempt to enter a longer code, you will receive an error message.

  1. To save the record, press CNTL + S or, from the File menu, select Save
