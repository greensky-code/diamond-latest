Adding Users

What do you want to do?

Add a User

Use this procedure to add a user that is generally independent from other users or groups. This procedure does not assign the user to a security template. You must grant or deny the user security privileges independently.

Add a User Assigned to a Template

Use this procedure to add a user and assign it to a security template. That way, security is based exclusively on the privileges granted the template. The privileges granted the template always supersede privileges independently granted the user.

Add a Super User

Use this procedure to add a user that has unlimited access to the DIAMOND program.

Add a Security Template

Use this procedure to add a security template. Generally, templates represent groups of users in your organization. You can then grant and deny the template security privileges. Later, you can add users assigned to the template and the users will be assigned the privileges granted the template.
