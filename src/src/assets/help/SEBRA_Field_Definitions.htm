SEBRA Field Definitions

Field Name

Req

Description

Source

Comments

Claim Type

R

Type of claim for which this rule will apply.

Selected from predefined list

Hard coded values are:

  • D Dental Claims

  • I Institutional Claims

  • P Professional Claims

Description

O

Brief description for suppressing the EOB or RA.

Manually entered

 

Determinant Column

R

Name of the column from the claim header or detail table to be used as a determinate for suppression.

Selected from predefined list

List is accessed through F5 lookup

Determinant Table

R

Name of the table from the claim header or detail table to be used as a determinate for suppression.

Selected from predefined list

List is accessed through F5 lookup

From Value

R

First value to be included in the sequence.

Manually entered

Important: If you are entering a date(s) in the From/Through determinant fields, you must enter the date in MM/DD/YYYY format (including slashes). For example, 12/25/1999, 02/01/2000.

Operator

R

Logic definer for the values entered in the from and thru range.

Selected from predefined list

Hard coded values are:

  • Equal

  • Not Equal

Order No

R

Location in the rules order where this rule should fall.

Manually entered

Values are from 0 to 9999.

Srch Seq

R

Sequence of determinants within the single rule order. All determinants within the sequence must match to assign this rule.

Manually entered

Values are from 0 to 9999.

Suppress EOB/RA

R

Indicates which of the two reports will be suppressed by this rule.

Selected from predefined list

Hard coded values:

  • 1. EOB

  • 2. RA

Thru Value

O

Last value to be included in the sequence.

Manually entered

This value must be greater than the value entered in the From Value field.

Important: If you are entering a date(s) in the From/Through determinant fields, you must enter the date in MM/DD/YYYY format (including slashes). For example, 12/25/1999, 02/01/2000.
