BLHST View Group Summary Menu Command

Field Name

Req

Description

Source

Comments

Adjustments

D

Adjustments posted for the group, or zero.

Displayed

DIAMOND posts payments and adjustments received as of the GPBIL run date against the first billing period included in a billing run. Therefore only the first or only the combined group summary record for an invoice will show payments or adjustments greater than zero.

If the highlighted combined group summary record highlighted in the Select Group Summary Section is either the first combined group summary record, or is the only such record, the total payments posted for the group are shown.

Otherwise, this field will display zero (0).

Bill Seq No.

D

Sequence number from combined group summary record.

Displayed

 

Bill Through Request Date

D

The Bill Thru Req Date entered in GPBIL.

Displayed

Displayed for the premium billing run which includes the From/Through date range entered.

Bill Type

D

Whether billing was a regular or supplemental bill.

Displayed

 

Common Billing Date

D

Whether or not group has a common billing date as designated on GRUPB.

Displayed

If Yes, the top focus box displays one or many combined group summary records. A combined group summary record will be displayed for each period processed.

If No, the top focus box displays a single combined group summary record that includes all periods in the premium billing run. The From date displayed will be the earliest From date of any included period and the Through Date displayed will be the latest Through Date of any included period. The combined group summary record From and Through dates will not necessarily match any individual family billing period.

Ending Balance

D

Ending balance for the billing period.

Displayed

System calculated as follows:

Prior Balance – Payments +/- Adjustmnts +New Charges = Ending Balance

From

D

From date for the combined group summary record.

Displayed

 

Group ID

D

Unique identifier of the group.

Displayed

 

New Charges

D

Total new charges for the group in the selected billing period.

Displayed.

If the group associated with this record has a common billing date, this amount will equal the total new charges for the single common billing period. If this group does not have a common billing date, this amount is the total new charges for all the different billing periods included in the selected date range.

Payments

D

Total payments posted for the group, or zero.

Displayed

DIAMOND posts payments and adjustments received as of the GPBIL run date against the first billing period included in a billing run. Therefore only the first or only combined group summary record for an invoice will show payments or adjustments greater than zero.

If the highlighted combined group summary record highlighted in the Select Group Summary Section is either the first combined group summary record, or is the only such record, the total payments posted for the group are shown.

Otherwise, this field will display zero (0).

Prior Balance

D

If any, the prior balance for the selected group.

Displayed

If there is a common billing date, this is the ending balance from the period immediately preceding the billing period of the highlighted record. If no common billing date, this is the ending balance as of the last Billed Through date for the group.

Through

D

Through date for the combined group summary record.

Displayed
