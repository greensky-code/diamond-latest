NOTES Field Definitions

Field Name  Req Description Source Comments

All Types

O

List all note types.

Selected from predefined list

 

Date (Note List)

D

Date of this note record.

Displayed

 

Date (Note Detail)

R

Date of this note record.

Manually entered

This field defaults to the date you are creating the note. You can overwrite this date with a later date. If the date you enter is later than the current date, the system will still provide notes notifications for this note (if your system is set up to use notes notification).

Expire Date

O

Date this note is no longer active.

Manually entered

Note notification actions for this note will no longer occur once the expiration date is reached.

Follow Up Code (Note Detail)

C

The type of follow up action required.

Manually entered

 

Follow Up Date (Note Detail)

O

The date by which a follow up action is required to be completed.

Manually entered

 

Follow Up Date (Sort By)

O

Sort the list of notes by Follow Up Date

 

 

Followup Date (Note List)

D

Follow up date for this note record.

Displayed

 

Group

O

Display all notes that are linked to the source record by Group.

 

When you select Consolidate by Group, DIAMOND checks for Window and Note Type Links that exist for the source window (the window from which you accessed the Note List screen). If Window and Note Type Links do exist, and the context for any of the links is Group, DIAMOND then looks for notes that are related to the source record by Group. If DIAMOND finds related notes, these notes are added to the default list of notes displayed.

D

Originator of this note.

Displayed

D

This column is only populated when the note type is CHK.

Priority (Note Detail)

R

Priority for this note record.

System generated

This field determines the note notification action users will see when accessing records related to this note, if your system is set up to use notes notification.

This field defaults to the priority from the Note Type record.

This field can only be changed by the note author at the time of note entry, prior to saving the note.
