Audit Display Menu Command Field Definitions

Field Name

Req

Description

Source

Comments

Action

D

The action taken on the audited table.

Displayed

One of the following values may display:

  • Insert - a record was added to the table.

  • Update - a field on the record was updated.

Audit Tms

D

The date the audited table was inserted or updated by the user.

Displayed

 

Audited Table

D

Displays the name of the table audited.

Displayed

 

Column Name

D

Displays the column in the audited table that has been updated. Columns represent fields on the table.

Displayed

The AUTH_SCREEN_MOD column displays if a user selected a command on the Topic menu, then made a change on the focus box that subsequently displays. The Current Value column will display a 1 each time the same user selected a command on the Topic menu and made a change on the subsequent focus box that displays. For example, if the same user selected three different commands on the Topic menu, and a field was changed on each of the three focus boxes, then the Current Value column will display 111.

Current Value

D

Displays the new field value. The old field value displays in the Previous Value column.

Displayed

 

Previous Value

D

Displays the old field value before it was changed to the new value displayed in the Current Value column.

Displayed

 

Process

D

Displays the keyword.

Displayed

 

User

D

Display the name of the user who inserted or updated the audited table.
