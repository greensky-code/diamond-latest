PRULD Field Definitions

Field Name

Req

Description

Source

Comments

Add to Base Units

C

Number of units automatically added to the units retrieved from UNITV, most commonly used in conjunction with modifiers such as anesthesia.

Manually entered.

Value entered must be between 0 and 99999999.

If Calculation Method is either "Lesser of Unit Value or Pct" or "Unit Value," this field is required. Not used otherwise.

Allowed Factor Ovr (Price Rule Information)

O

Percentage that is applied to the calculated allowed amount to yield the final allowed amount.

Manually entered.

 

Allowed Factor Ovr (Multiple Modifier Pricing Information)

C

Type the percentage to apply to the first modifier allowed amount.

Manually entered.

This box is available if available if 4 – Modifier 1 + Modifier 2% is selected from the Price Method list.

You can type a decimal number. For example, type .01 for 1%, .5 for 50%, 1 for 100%, and 1.5 for 150%.

Allowed Factor Prompt

O

Indicator of whether the claims operator will be is prompted for a percentage that will be applied to the calculated allowed amount to yield the final allowed amount.

Selected by check box.

Valid values are:

Selected: Operator is prompted for a percentage

Cleared: Operator is not prompted for a percentage

Allowed Reason (Price Rule Information)

O

Allowed Reason code that will be entered on the detail line when this rule is used.

Selected from a predefined list.

Allowed Reason codes are maintained in keyword REASN (Reason) with the Reason Code Type list set to AL – Allowed.

Allowed Reason (Multiple Modifier Pricing Information)

O

Type the Allowed Reason code you want to apply to the claim line item if Price Method 2, 3, or 4 is used.

Select from a predefined list.

F5 lookup is available.

Allowed Reason codes are maintained in keyword REASN (Reason) with the Reason Code Type list set to AL – Allowed.

Calculation Method

R

Calculation method for this detail rule

Selected from a predefined list.

Valid values are:

Fee Schedule

Flat Rate

Lesser (of) Fee Schedule or Pct of Billed

Lesser of Multiple Fee Schedules

Lesser (of) Unit Value or Pct of Billed

None (manually price)

Percent of Billed

Geo Zip

Unit Value

Carve Out

O

Indicator of whether detail lines priced using this method are eligible for carve out logic, an aspect of whole claims pricing in keyword INCLM.

Selected by check box.

Used for whole claims pricing (PRULW) only.

Valid values are:

Selected: Detail lines are eligible

Cleared: Detail lines are not eligible

Hold Reason

O

Hold reason that will be entered on the detail line when this rule is used, placing applicable detail lines on hold.

Selected from a predefined list.

Hold reasons are maintained in keyword REASN with the Reason Code Type list set to HD – Hold.

Include/Exclude

R

Indicates whether claim detail lines priced using this method should be included or excluded from accumulations applied to whole claims pricing Base Price Rule, Outlier Rules, or Cap Rules.

Selected from a predefined list.

Used for whole claims pricing (PRULW) only.

Information Only

O

Indicator of whether detail lines priced using this method are marked "I" (informational), commonly used in whole claims pricing.

Selected by check box.

Used for whole claims pricing (PRULW) only.

Valid values are:

Selected: Detail will be marked with an "I" for Informational

Cleared: Detail lines will not be marked

Rule Level

R

Order in which this detail rule will be evaluated for applicability.

Manually entered.

Five levels are available.

Message to Operator

O

Message entered here is displayed to claim operator when a detail line is priced using this method.

Manually entered.

 

Modifier Code

O

Modifier to which this detail rule applies; if blank, the rule applies to claim detail lines with no modifier.

Manually entered.

Modifiers are maintained in keyword MODIF (Modifier).

Mult By Pct Allowed

O

Indicator of whether the calculated allowed amount will be priced by either the PROVC or GRUPD Pct Allowed, yielding the final allowed amount.

Selected by check box.

Keyword PROVC or GRUPD is searched first for the allowed percentage, depending on the entry in Primary Price Rule on the applicable LINBS record.

Valid values are:

Selected: Calculated allowed amount will be priced by either the PROVC or GRUPD Pct Allowed

Cleared: This method will not be used

Price Method

O

Click the pricing method you want to use.

Select from a predefined list.

Select from the following list:

  • 1 – Standard Modifier 1

  • 2 – Pay Modifier 2

  • 3 – Modifier 1 + Modifier 2

  • 4 – Modifier 1 + Modifier 2%

Price Rule

R

Type the Price rule ID you want to add or display.

Manually entered.

F5 lookup is available.

Search Sequence

R

Sequence within level that this detail rule will be evaluated for applicability.

Manually entered.

Multiple sequence numbers identify different pricing according to the procedure code. Procedure codes themselves are entered on the associated PRULS record.

Unit Value Type

C

Indicator of which unit value should be used when Calculation Method is either Lesser of Unit Value or Pct of Billed, or Unit Value.

Selected from a predefined list.

Valid values are

Total Units

Professional Units

Technical Units

Anesthesia Units

If Calculation Method is either "Lesser of Unit Value or Pct" or "Unit Value," this field is required. Not used otherwise.

User Defined 1

O

User-defined field available for customization.

Manually entered.

For more information, see Customize a user-defined field.

User Defined 2

O

User-defined field available for customization.

Manually entered.

For more information, see Customize a user-defined field.
