Add a member address

You can add a member address of any Address Type using keyword MEMBA (Member Address).

Alternatively, you can add certain Address Types using keywords MEMBR (Member) and ALIAS (Alias/Responsible Party/Privacy). You can use keyword MEMBR to add an address of address type R1 - Member Residential/Primary. You can use keyword ALIAS to add an address of address type R4 - Alias Residential/Primary or R5 - Responsible Party Residential/Primary.

To add a member address

  1. Press F3, type MEMBA, and then press ENTER.

  2. Fill in the fields on the window. For more information, see MEMBA Field Definitions.

  3. Press CTRL+S to save.

Tip

If you use keyword MEMBR to add a member address, this address is by default Address Type R1- Member Residential/Primary. This address displays in keyword MEMBA.

If you use keyword ALIAS to add a member address in section box Alias Information, the address is by default Address Type R4 - Alias Residential/Primary. If you enter a member address in ALIAS section box Responsibility Party Information, the address is by default Address Type R5 - Responsible Party Residential/Primary. These addresses also display in keyword MEMBA.
