PROVE Parameters & Prerequisites

The following is only a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

 

Parameter

Description

Comments

MEMSITEVAL

Used to enable member Site Validation functionality.

If MEMSITEVAL Parameter 1 = N, keyword PROVE fields Site and Cert are unavailable.

PROVENROL

This parameter is used in conjunction with running the Provider/Member Enrollment Utility scripts to initiate provider enrollment functionality.

See Provider/Member Enrollment and PARAM user documentation for additional information.
