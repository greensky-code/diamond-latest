Copy a Benefit Package

Use the following procedure to copy benefit rules from one benefit package to another. Many have a multitude of benefit rules and from year to year or product to product packages may only slightly differ. In some cases, the same benefits must be attached to benefit packages with different names.

To copy a benefit package

  1. Press F3, type BENEF, and then press ENTER.

  2. Select a benefit package or add new benefit package to be the receiver of the rules to be copied.

  3. On the Special menu, click Copy Benefit Package.

  4. Under Benefit Package ID, click the benefit rule you want to copy. In the bottom focus box, the selected price rule determinant code data displays.

  5. Under Rule ID, click the benefit rules you want to copy.

  6. To copy all benefit rules, click Select All.

  7. To select consecutive benefit rules, press SHIFT and click the first rule and then the last rule. All benefit rules between and including the selected first and last rules are selected.

  8. To select non-consecutive benefit rules, click a rule, then press CTRL and click each additional rule you want to copy. To unselect a rule, press CTRL and click the selected rule again.

  1. Click OK to copy the benefit rules to the receiving record; click OK again to complete the copy operation or Cancel to cancel the operation.

  2. Press CTRL+S to save the record.
