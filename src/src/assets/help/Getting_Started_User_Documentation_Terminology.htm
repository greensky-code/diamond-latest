User Documentation Terminology

Some of the terms used in our documentation include the following:

 

Term

Definition

Drop-down list box

or

Drop-down data window

A selection list available in certain window fields (when the insertion point is in a field). An arrow is displayed (right-hand side of the field) when the insertion point moves into a drop-down data window field. Click the arrow or press ALT+UP ARROW or ALT+DOWN ARROW to display the items in the list.

The items in a drop-down list box are programmed into DIAMOND (hard-coded) and cannot be changed.

The items in a drop-down data window are table records that you or another user has previously entered in another keyword or file utility.

Field

A field is a data entry or display area on the keyword window. The term "column" is sometimes used when discussing relational data elements; however, we have chosen to use "field" in our user documentation.

File

A computer file that consists of a relational database table. In our documentation, "file" is used interchangeably with "table."

Focus box

A focus box encloses a set of related options or fields. When the insertion point is in a field, the entire focus box appears active by a shadowing effect. Many focus boxes also contain multiple section boxes.

Literal

The description of a field that appears on a keyword window.

Picklist

A picklist is a focus box that displays a set of records you can select. Column headings are indicated in bold type. Click a column heading to sort the records. Click a record to display complete information in the window.

Record

A record is a unit or units of data within a file (table) that constitutes a complete set. For example, in the Provider table, all data related to one provider is considered one record. The term "row" is sometimes used when discussing relational databases; however, we have chosen to use "record" in our user documentation.

Screen

Sometimes used as a synonym for "window."

Section box

A section box encloses a set of related options or fields. It further describes sections in a focus box.

Window

A window is used to display and enter data associated with a specific keyword.

Table

See "File."
