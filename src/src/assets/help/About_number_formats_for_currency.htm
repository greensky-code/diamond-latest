About number formats for currency

Most currency fields allow you to type up to 16 digits before the decimal and 2 after the decimal. For example, you can type 1234567890123456.12

The dollar sign ($) and commas are automatically entered for you.

Display of currency amounts

If a text box or picklist has limited space and cannot display the entire currency amount, then the display of the amount is truncated using one of the following methods in the order in which they appear:

  • Remove the dollar sign ($). For example, $1,234,567,890,123,456.78 would be replaced by 1,234,567,890,123,456.78

  • Remove the cents (.00). For example, $1,234,567,890,123,456.78 would be replaced by 1,234,567,890,123,456

  • Replace thousands with the Thousands symbol (k). For example, $1,234,567,890,123,456.78 would be replaced by 1,234,567,890,123k

  • Replace millions with the Millions symbol (M). For example, $1,234,567,890,123,456.78 would be replaced by 1,234,567,890M

  • Replace billions with the Billions symbol (G). For example, $1,234,567,890,123,456.78 would be replaced by 1,234,567G. DIAMOND uses the French and American method of numeration where a billion is a thousand million (1,000,000,000).

To display the entire full-length currency amount, click in the text box that displays the truncated currency amount. Use the LEFT ARROW and RIGHT ARROW keys to move the insertion point left and right in the text box. You cannot display entire full-length currency amounts in display-only fields.
