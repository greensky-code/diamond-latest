Add a User

Use the following procedure to add a user to the DIAMOND program. You can also add a user assigned to a template or add a super user.

To add a user

  1. Press F3, type SUSER, and then press ENTER.

  2. In the User ID box, type an ID for the user, and then press TAB.

  3. Fill in the fields in the User Information section box.

  4. In the User Type list, click Operator.

  5. In the Field Level Security box, type the field level security ID.

  6. In the Language list, click the default language.

  7. If you want this user to have access to claims with a security code equal to or less than a particular value, then in the Row Level Security box, type the security code. You can type a number from 0 to 9.
    For example, if you type 5, then this user only has view/change privileges on claims that have the Security Code box set to 5, 4, 3, 2, 1, or 0. The Security Code box displays in keywords CLDSP, DNCLM, DSPBN, INCLM, and PSCLM.

  8. On the File menu, click Save.

  9. On the Special menu, click Add Database User.

  10. In the Password box, type the user password.

  11. In the Confirm Password box, re-type the user password, and then click OK. The user is connected to the Oracle roles DIAMONDVIEW and DIAMONDUPDATE.

  12. If desired, do the following:

  13. If you want this user to only access claim records with a particular group, then on the Special menu, click Group Affiliation.

  14. If we want this user to only access claim records with a particular panel, then on the Special menu, click Panel Affiliation.

  15. If you want this user to only access claim records with a particular provider, then on the Special menu, click Provider Affiliation.

  16. If you want this user to only access claim records with a particular vendor, then on the Special menu, click Vendor Affiliation.

  17. If you want this user to only access claim records with a particular IPA, then on the Special Menu, click IPA Affiliation.

  18. If you want this user to only access claim records with a particular line of business, then on the Special menu, click LOB Affiliation.
