Run an A/P Update Job for Administrative Fees

Use the following procedure to run an A/P update job for administrative fees, only. If you want to run an A/P update job for both administrative fees and regular medical claim payables, then see Run an A/P Update Job.

Note  The dates in the Entered Date, Thru Entered Date, From Due Date, and Thru Due Date boxes are ignored for administrative fees.

To run an A/P update job for administrative fees

  1. Press F3, type APUPD, and then press ENTER.

  2. On the File menu, click New.

  3. In the Payable type list, click M – Medical.

  4. In the Company Code box, type the company code for which you want to run this update (optional). F5 lookup is available. Type the date in MM/YYYY format. For example, 11/2002

  5. In the Post Month box, type the G/L posting month.

  6. In the Admin Fee list, click O – Only Admin Fee.

  7. In the Up To Amount box, type the maximum payable amount to include in this update (optional).

  8. In the Vendor ID box, type the vendor code for which you want to run this update (optional). F5 lookup is available.

  9. On the File menu, click Run Job. The A/P Update job runs. When the job ID in the picklist displays Successful Run in the Status column, the job has completed.
