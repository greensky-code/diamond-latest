CFDST Commands on the Menu Bar

 

Command             Menu   Description

New From Template

F

Use to enter a new CFDST job using an existing job template. Choose a template from any previously entered jobs that were saved as templates.

Save As Template

F

Use to save a CFDST job you entered as a template that can be used to enter additional jobs.

Run Job

F

Use to run the current job.

Summary Audit Report

Details

S

View or print a report showing the totals for all claim lines for a CFD model as well as grand totals for each provider with claims selected for that fund and month.

If you select this command when no claim lines were selected for a Cap Fund run, DIAMOND displays an error message.

The report displays a summary of the results of the job, sorted by Provider ID. For each Provider ID, the Report displays totals for selected fields for all claim detail lines that will be modified if the job is run in Post mode. Grand totals for all providers are also provided.

Detail Audit Report

Details

S

View or print a report showing the information in the summary audit report as well as additional details of each claim, such as Procedure Code, Billed Amount, and Copay Deductible.

If you select this command when no claim lines were selected for a Cap Fund run, DIAMOND displays an error message.

The report displays a summary of the results of the job, sorted by Provider ID. For each Provider ID, the Report displays totals for selected fields for all claim detail lines that will be modified if the job is run in Post mode. Grand totals for all providers are also provided.
