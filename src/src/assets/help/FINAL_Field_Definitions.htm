FINAL Field Definitions

Field Name

Req

Description

Source

Comments

Company Code

O

The general ledger company associated with the transactions selected.

Manually entered

F5 lookup is available.

Leave blank to update qualifying transactions for all companies.

From Received Date

O

If you want to include claims received by the health plan on or after a particular date, then type the from received date.

The date a claim was received by a health plan displays in the Date Recv box on the claim header window in keyword DNCLM, INCLM, or PSCLM.

Manually entered

Type in MMDDYY format.

Group

D

Indicator of whether Group is a selection criterion for claims included in this finalize job.

Displayed

If field displays All, group is not a criterion for selection claims; if Some, group is a criterion for selection of claims.

Provider

D

Indicator of whether Provider is a selection criterion for claims included in this finalize job.

Displayed

If field displays All, provider is not a criterion for selection claims; if Some, provider is a criterion for selection of claims.

Thru Received Date

O

If you want to include claims received by the health plan through a particular date, then type the through received date.

The date a claim was received by a health plan displays in the Date Recv box on the claim header window in keyword DNCLM, INCLM, or PSCLM.

Manually entered

Must be greater than the date in the From Received Date box.

Type in MMDDYY format.

File Type

D

Indicator of whether File Type is a selection criterion for claims included in this finalize job.

Displayed

If field displays All, file type is not a criterion for selection claims; if Some, file type is a criterion for selection of claims.

Claims Status

D

Indicator of whether Claim Status is a selection criterion for claims included in this finalize job.

Displayed

If field displays All, claim status is not a criterion for selection claims; if Some, claim status is a criterion for selection of claims.

Job ID

R

Unique identifier of this finalize job.

System generated

This field may be manually overwritten.

Request User

D

User requesting this job.

System generated

Status

D

Status of the finalize job.

Displayed

Values are:

  • New: Job has not been run.

  • Error: Job ran, had errors, and claims were not finalized.

  • Posted: Job ran successfully.

Request Type

R

Indicator of when job will run.

Selected from predefined list

Only one value is valid, Immediate. Deferred is reserved for future functionality.

Comments
