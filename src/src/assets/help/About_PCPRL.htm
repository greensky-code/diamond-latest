About PCPRL

PCPRL enables you to create rules for applying PCP Auto Assignment criteria. Each rule defines the specifics for applying a single criterion. For example, you can define a rule that uses as the assignment criterion the provider’s accepted gender as specified on the provider’s enrollment rule (PROVE).

You create each PCP Auto Assignment rule once in PCPRL. You can then use these rules in different combinations to build PCP Auto Assign rule packages for each line of business as needed. You build PCP Auto Assign rule packages in keyword PCPAA (PCP Auto Assign).
