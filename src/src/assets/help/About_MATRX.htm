About MATRX

Use keyword MATRX (A/R Matrix Determinants) to define criteria that the premium billing programs use to find the correct premium rates for groups using the S-Subscriber or W-Whole Family premium billing tiering methods (GRUPB > Tiering Mthd). For such groups, a matrix determinant is required.

Matrix Determinant is a way to set up rows (Y) and columns (X) that categorize members or subscribers (depending on whether the group uses S-Subscriber or W-Whole Family) according to age, gender, salary, and Medicare status, family size, and the presence or absence of a subscriber spouse or domestic partner. DIAMOND processing uses this data to build the Matrix Premium Rate Entry (PREMM) matrix, displaying the column and row descriptions entered here in MATRX. This structure allows premium rates, which are based on combinations of attributes of subscribers and members.
