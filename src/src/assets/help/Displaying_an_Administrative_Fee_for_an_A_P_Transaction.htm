Display an Administrative Fee for an A/P Transaction

Use the following procedure to display an administrative fee for an A/P transaction.

To display an administrative fee for an A/P transaction

  1. Press F3, type ACPAY, and then press ENTER.

  2. In the AP Transaction ID box, type the A/P transaction for which you want to display the administrative fee. F5 lookup is available.

  3. On the Special menu, click Admin Fee Info.
