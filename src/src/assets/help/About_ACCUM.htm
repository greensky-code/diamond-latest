About ACCUM

Use keyword ACCUM (Benefit Accumulator Base Values) to manually enter one-time credits toward a member’s benefit limit in cases where those credits are not carried forward automatically. This situation occurs either because of how the benefit packages are defined or because the credits were recorded on an outside system.

For a member, benefit amounts automatically carry over from one benefit package to another when the same benefit rule ID (BRULE) is used in both the old and the new benefit packages and the benefit rule has been flagged to apply to all groups.
