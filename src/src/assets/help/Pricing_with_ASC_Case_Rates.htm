Pricing with ASC Case Rates

Restrictions

Instructions

ASC Groups are sets of procedures that are associated with each other in treating certain outpatient surgeries. Some health plans define case rates which apply to these groupings, which are similar to DRGs but apply in an outpatient setting.

Restrictions

None.

Instructions

Step

Comment

1. Define a medical definition (MEDEF) for each ASC group, using Procedure Code as the determinant.

If the contract lists a rate for all other CPT codes, you’ll need a MEDEF for "all other" CPT codes.

2. Create a PDTER record for each ASC Group, using the applicable medical definition.

 

3. For each unique reimbursement rate, create a procedure code for the system-generated line (PROCD/PROCP).

 

4. Create a PRULW record for each ASC Group reimbursement arrangement.

 

5. DIAMOND will apply up to 3 case rates, the first of which will always be applied at 100% of the reimbursement rate.

Determine the order that DIAMOND should evaluate each case rate so that DIAMOND will know which case rate to apply first.

If the contract indicates that multiple case rates apply, then enter percentages for the second and third case rates in the Case 2 % of Allowed and Case 3 % of Allowed boxes on the DRG/APC Information dialog box (in keyword PROVC, on the Special menu, click DRG/APC Information).

If the contract states that only one case rate shall be applied, leave the Case2 % Allowed and Case 3 % of Allowed boxes blank.
