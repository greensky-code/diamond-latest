Add a User Assigned to a Template

Use the following procedure to add a user to the DIAMOND program and assign it a security template. By assigning a user to a template, the user's privileges are based exclusively on the privileges assigned the template. Any privileges granted or denied this user will be superceded by the privileges granted or denied the security template.

To add a user assigned to a template

  1. Press F3, type SUSER, and then press ENTER.

  2. In the User ID box, type an ID for the user, and then press TAB.

  3. In the Template ID box, type the security template ID. All user privileges will then be based on the template.

  4. Fill in the fields in the User Information section box.

  5. In the User Type list, click Operator.

  6. In the Language list, click the default language.

  7. On the File menu, click Save.

  8. On the Special menu, click Add Database User.

  9. In the Password box, type the user password.

  10. In the Confirm Password box, re-type the user password, and then click OK. The user is connected to the Oracle roles DIAMONDVIEW and DIAMONDUPDATE.
