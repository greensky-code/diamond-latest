Sort records in a picklist

Use the following procedure to sort records in a picklist.

To sort records in a picklist

  1. In the picklist, display the records you want to sort.

  2. Do one of the following:

  3. To sort in ascending order, click the column header you want to use for sorting records.

  4. To sort in descending order, press CTRL, and then click the column header you want to use for sorting records.
