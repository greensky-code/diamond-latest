Request an ID card for a member

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to manually request an ID card for a subscriber or dependent.

Restrictions

If "do not produce ID cards" is indicated in the IDCARDORDER parameter in keyword PARAM (Parameter), then your system administrator has prohibited you from requesting an ID card for a member.

Instructions

Step

Comment

1. Press F3, type MEMBR, and then press ENTER.

 

2. In the Subscriber ID box, type the subscriber ID, and then press TAB.

F5 lookup is available.

3. If the List of Family Members dialog box displays, then double-click the name of the subscriber or dependent you want to request an ID card.

 

4. On the Special menu, click ID Card Request.

When ordering ID cards manually for a member with an outstanding ID card print record (the ID card has not yet been printed/posted using keyword IDPRT), then DIAMOND prompts you to do one of the following:

  • Update the outstanding record (i.e., the ID card will be printed based on current data), or

- or -

  • Leave the outstanding record "as is" (i.e., the ID card will be printed based on the outstanding record’s data).
