Delete a Modifier Code

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to delete a modifier code.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type MODIF, and then press ENTER.

 

2. In the Modifier ID box, type the modifier ID.

F5 lookup is available.

3. On the File menu, click Delete.

 

Frequently Asked Questions

None.
