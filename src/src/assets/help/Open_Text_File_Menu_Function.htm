Open Text File Menu Command

The Heads Down Claims executable displays an Open File dialog box. Navigate to and double-click on the claims text file (previously saved on your workstation) to open the file. The executable displays the first claim header record entered in the claims text file opened. After opening the file you can:

Edit claim header fields.

Add new claims (append).

Edit claim detail line fields.

Delete claim detail lines.

Edit lens prescription data fields.

Delete entire claim.

Add claim detail lines.

Save edited claim text file.

Edit Claim Header Fields

Step

1. With the cursor positioned in the Header window, use the up-arrow or down-arrow keys to navigate to a specific header record within the file.

2. Press CTRL+HOME to display the first header record in the file.

3. Press CTRL+END to display the last header record in the file.

4. Edit header fields as required.

Edit Claim Detail Line Fields

Step

1. With the cursor positioned in the Header window, use the up-arrow or down-arrow keys to navigate to the desired header record within the file.

2. Use Topic Menu > Detail to display the Detail window and previously entered claim detail lines.

3. Edit claim detail line fields as required.

Edit Lens Prescription Data Fields

Step

1. With the cursor positioned in the Header window, use the up-arrow or down-arrow keys to navigate to the desired header record within the file.

2. Use Topic Menu > Lens Prescription to display the Lens Prescription window and previously entered lens prescription data.

3. Edit claim lens prescription data as required.

Add Claim Detail Lines

Step

1. With the cursor positioned in the Header window, use the up-arrow or down-arrow keys to navigate to the desired header record within the file.

2. Use Topic Menu > Detail to display the Detail window and previously entered claim detail lines.

3. Press CTRL+A to open an additional claim detail line record for entry. See Add Record Menu Command for additional information.

Add New Claims (append)

Step

1. Position the cursor in the header record of any claim in the file.

2. Press CTRL+A to open a new (additional) claim header/detail record for entry. See Add Record Menu Command for additional information.

Delete Claim Detail Lines

Step

1. With the cursor positioned in the Header window, use the up-arrow or down-arrow keys to navigate to the desired header record within the file.

2. Use Topic Menu > Detail to display the Detail window and previously entered claim detail lines.

3. Position the cursor in any field on the claim detail line to be deleted.

4. Press CTRL+D to delete the claim detail line. See Delete Record Menu Command for additional information.

Delete Entire Claim

Step

1. With the cursor positioned in the Header window, use the up-arrow or down-arrow keys to navigate to the desired header record within the file.

2. Position the cursor in any field in the header of the claim to be deleted.

3. Press CTRL+D to delete the entire claim from the claim text file. See Delete Record Menu Command for additional information.

Save Edited Claim Text File

Step

1. Press CTRL+S to save the edited claim text file. See Save Text File Menu Command for additional information.
