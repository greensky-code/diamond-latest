Entering a State

The DIAMOND program uses pre-loaded U.S. Standard Attributes and Edits that control how you enter a state.

To enter a state in keyword windows, use the official United States Postal Service abbreviation. For example, if you type CA, then DIAMOND displays CA. You can also enter Canada Post province and territory symbols.

You can override these internal pre-loaded U.S. Standard Attributes and Edits and create a customized field label for the state field. In addition, you can expand the field to accommodate the entry of six alphanumeric characters. You can then enter postal codes with formats that differ from the U.S. Standard. For example, you want to enter Canadian postal codes. For more information, see UDATT, User Defined Attributes.

Automating city and state entries

You can automate the entry of cities and states on many keyword windows by typing a ZIP Code. For example, if you type 94112 in the ZIP Code field, then San Francisco automatically displays in the City field and CA automatically displays in the State field.

To automate city and state entries

  1. In keyword ZIPRF, enter the ZIP Code, city, and state. For more information, see ZIPRF, ZIP Code Lookup

  2. In keyword PARAM, display the DFLTFZIP parameter and set it to "Y." For more information, see PARAM, Parameter.
