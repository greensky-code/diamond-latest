MEMBR D/C Information Menu Command

 

Field Name  Req Description Source Comments

Account No

R

The member’s account number.

Manually entered

 

Type

O

A category of account.

Selected from predefined list

Types beginning with CR indicate credit card.

Types are maintained in keyword CODES (System Codes Maintenance), with a type ACCTINFTYPE.

Use

R

An indicator of the use of the account.

Selected from predefined list

Hard coded values are:

Both

Credit reserved for future use.

Debit indicates the account is used to pay premiums.

Eff Date

R

Date the account is effective.

Manually entered

 

Exp Date

C

The date the account expires.

Manually entered

This field is required if the Type begins with CR, and not used if the Type does not begin with CR.

Status

O

Indicator of whether the account is active.

Selected from predefined list

Hard coded values are:

Active

Inactive

Prenote

Routing Number

O

Routing Number used for electronic funds transfer.

Manually entered
