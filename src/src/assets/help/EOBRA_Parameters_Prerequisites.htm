The following is only a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

Parameter

Description

Comments

DFLTPRINTSTAT

In cases where a claim is processed for an ineligible member, determines whether or not a remittance advice or explanation of benefits will be printed.

When the member is eligible LINBS determines whether the member receives an EOB and PROVC determines whether the vendor associated with the provider contract receives a remittance advice.
