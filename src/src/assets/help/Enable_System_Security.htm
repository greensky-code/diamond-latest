Enable System Security

Use the following procedure to enable system security, which includes row level, data subset level, and claim level security.

To enable system security

  1. Press F3, type PARAM, and then press ENTER.

  2. In the picklist, click the SECURITY parameter ID.

  3. In the Parameter 1 box, type Y

  4. On the File menu, click Save.
