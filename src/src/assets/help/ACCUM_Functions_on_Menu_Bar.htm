ACCUM Commands on the Menu Bar Command  Menu  Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Claim Report

S

Prints a report of claims for a selected date range showing members who have accumulator file entries:

  • For a different benefit package than the one credited

  • For claims adjudicated after credits were entered on the new benefit package using ACCUM

  • For claim dates of service prior to detail dates of ACCUM credits

  • For claims with an entry date equal to or greater than the Thru Date entered on the ACCUM Claims Report request

Notes

details

N

This command is only available if the ACCUM window has been associated with at least one note type in NOTEW.

Takes you to the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.
