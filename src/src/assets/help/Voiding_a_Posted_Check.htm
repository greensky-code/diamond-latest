Void a Posted Check

Use the following procedure to void a posted check. This procedure assumes that the associated claim(s) was originally processed in error and a new claim(s) will be entered and paid.

Important Note  In keyword CHECK (Check Register), the check status must be "Posted."

To void a posted check

  1. Press F3, type CHECK, and then press ENTER.

  2. Select the check or EFT transaction.

  3. On the Special menu, click Stop/Void.

  4. In the Action list, click Void.

  5. In the Reason list, click a reason.

  6. Click OK. The check's status changes to "In Process."

  7. On the File menu, click Save.

  8. In keyword CVSUP, run the clear/void/stop job to verify its accuracy, and then post it. In keyword CHECK, the Check Status field is set to Voided.
