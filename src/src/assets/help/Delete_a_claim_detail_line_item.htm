Delete a claim header or detail line item

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to delete a claim header or detail line item with a processing status of "U" (unposted) or "H" (held):

Restrictions

If you are deleting a detail line item, the Proc Status list must be set to H – Held or U – Unposted (open).

If you are deleting a claim header, no detail line items may be associated with the claim.

Instructions

Step

Comment

1. Display the detail line item you want to delete.

 

2. On the File menu, click Delete.

If the detail line item has already been processed by DIAMOND, you cannot delete it.

If deleting a claim header, the entire claim is deleted from DIAMOND.

Frequently Asked Questions

  • What is the keyboard shortcut to delete a detail line item?

Press CTRL+D.
