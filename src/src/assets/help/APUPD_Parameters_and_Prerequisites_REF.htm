Comments

COMPCODEREQ

Use to make the Company Code box a required field.

PARTIALPAY

Use this parameter to determine whether or not partial payments – payments of some detail lines on a claim, but not others – is allowed.

If you are using partial pay functionality, you cannot use claims discount or interest calculation functionality.
