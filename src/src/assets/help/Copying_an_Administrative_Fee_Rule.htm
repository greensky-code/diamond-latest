Copying an Administrative Fee Rule

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to copy an administrative fee rule.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type ADRUL, and then press ENTER.

 

2. In the ID box, type the rule ID you want to copy, and then press ENTER.

F5 lookup is available.

3. On the Special menu, click Rule Copy.

 

4. In the Eff Date box, type the effective date of the rule.

If you type a past date, then previously entered claims are not affected.

5. If you want the rule to eventually termination, then in the Term Date box, type the date the rule terminates, and in the Term Rsn list, click the reason code to use when the rule terminates.

 

6. Click OK.

 

7. Make any other changes you want.

 

8. On the File menu, click Save.

 

Frequently Asked Questions

None.
