About DNCLM

Use keyword DNCLM (Dental Services Claims) to enter, change, price, and adjudicate dental services claims.

Keyword DNCLM has two primary windows to enter dental services claims: the header window and the detail window. The header window is the first window that displays when you enter keyword DNCLM. It is where you can enter member information, provider information, and claim information. You can also display the header window by clicking Header File from the Topic menu. The detail window displays when you press TAB from the last field on the header window. It is where you can enter individual claim detail lines. You can also display the detail window by clicking Detail File from the Topic window.

Member, vendor, and provider information can be changed on a claim that has not been finalized.
