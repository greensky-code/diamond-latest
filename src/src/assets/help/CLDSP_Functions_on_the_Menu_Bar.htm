CLDSP Commands on the Menu Bar

 

Command             Menu   Description

Administrative Fee Info

S

Displays the administrative fee, if charged to the health plan, for a claim detail line.

Audit Display

Details

W

Displays audit trail records for the selected record.

Claim Audit Details

Details

S

For the claim and claim line selected in the keyword, displays summary information for:

  • Claim detail lines that were impacted by the CodeReview evaluation

  • Claim detail lines that caused CodeReview to change the claim.

Hold Reason Display

Details

W

Displays the Hold Reasons dialog box showing a list of all hold reasons applying to a claim.

NDC Codes

S

Use to view the National Drug Code (NDC) from this detail line item.

Notes

details

N

This command is only available if the CLDSP window has been associated with at least one note type in NOTEW.

Takes you to the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.

Other Claim Info

S

Displays the Other Claim Information window for the professional services claim selected in the upper focus box. This window displays information included in PSCLM.

See the PSCLM Other Claim Info menu command for field descriptions and additional information.

Patient Liabilities

S

Displays the Patient Liabilities window for the claim detail line selected in the upper focus box.

When asterisks appear in:

  • Not Cov Rsn

  • Copay Rsn

  • Co-ins Rsn

  • Deduct Rsn

Use this window to view:

  • Copay

  • Coinsurance

  • Limits (Not Covered)

  • Deductible

  • Exclusions (Not Covered) benefit rules/amounts for claim detail lines having more than one benefit rule of the same benefit rule type

See PSCLM or INCLM Patient Liabilities menu command for additional information.

UB92 Other Carriers

S

Displays the UB92 Other Carrier window for the institutional claim selected in the upper focus box. See INCLM for field descriptions and additional information.

UB92

S

Displays the UB92 window for the institutional claim selected in the upper focus box. See INCLM for field descriptions and additional information.

View Administrative Fee Info

S

Use to view the administrative fee details, if any, for the displayed claim.

View Cap Fund Info

S

Displays Capitation Fund Information for the current claim detail line. Data for all fields listed (except Cap Fund Run Month and Cap Fund Withhold Amt) is available if the claim detail line was priced using a provider contract including:

  • Contract Type of F (Cap Fund)

  • Cap Fund Model

Cap Fund Run Month and Cap Fund Withhold Amt are available if the Cap Fund Status is 3 (Posted). Posted indicates that the claim line was selected and posted by a Capitation Fund Distribution (CFDST) batch job. For more information on how these fields are updated, Capitation/Special Fund Distribution How DIAMOND Thinks.

View Claim Totals

S

Displays a list of all claim totals from the Claim Detail sections from INCLM or PSCLM. See INCLM or PSCLM for information about the Claim Detail totals displayed here.

See Billed Minus Allowed Amount, How DIAMOND Thinks for information about the Billed Minus Allowed total in this window.

Note: if DIAMOND displays the error message "70894: Totals for a claim that is whole claim priced, cap fund or capitated may not be accurate," this indicates that, depending on the claim status of each claim line, the net amount displayed in the Claims Display Totals may not reflect the amount paid in the check run.

View Claims Interest/
Penalty/Discount Information

S

Displays the claim line’s interest, penalty, or discount calculation information.

For information about interest and discount calculations, see Claims Discount and Interest Calculations, How DIAMOND Thinks.

View Claims with Auth Number

S

Displays a list of other institutional, professional services, and dental claims with the same authorization number on the claim header as the current claim. You can select a claim on the list and view it in an active Claims Display (CLDSP) window. To sort the list, click on a column header.

Use this command any time after you enter an authorization number on the claim header. The current claim is not included in the list.

View Dental Information

S

Displays dental information for the current claim detail line, including:

  • Tooth Number

  • Arch

  • Surface (five entries)

  • Quadrant

  • Oral Cavity

  • Tooth Status

View IPA Info

S

Displays the member’s associated IPA contract information, including Delegated Responsibilities, Special Programs, Service Area, and the Postal Codes included in this Service Area. Available from both the claim header and the claim detail screens. 

IPA Information is based on the LOB and IPA entered on the member’s MELIG record, and the date of service on the claim. Therefore, the member number and date of service must be entered into the claim prior to viewing the IPA information.
