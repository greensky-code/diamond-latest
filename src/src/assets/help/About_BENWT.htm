About BENWT

Use keyword BENWT (Benefit Weighted Accumulator) to maintain the values associated with accumulating benefits that use weighted values. You may create one or more Benefit Weight Accumulators. A Benefit Weighted Accumulator can be linked to one or many Benefit Rules within a Benefit Package. The Benefit Rule is linked by Benefit Weight Accum ID on BENEF.  

Changes you make to Benefit Weighted Accumulator records are written to AUDIT (Audit Trail).
