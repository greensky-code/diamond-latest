Set up a Capitation Model to use Incentive Rules

Use the following procedure to set up a capitation model to use incentive rules.

To set up a capitation model to use incentive rules

  1. Press F3, type CPMOD, and then press ENTER.

  2. In the picklist, click the capitation model.

  3. Click to select the Use Incentive Rules check box.

  4. On the File menu, click Save.
