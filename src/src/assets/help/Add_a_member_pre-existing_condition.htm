Add a member pre-existing condition

Use this procedure to add a member pre-existing condition.

To add a member pre-existing condition

  1. Press F3, type MCOND, and then press ENTER.

  2. Fill in the fields on the window in tab order up to the field Condition Type. For more information, see MCOND Field Definitions.

  3. In the field Condition Type, select one of the pre-existing condition types Pre-Existing, Rated-up, or Ridered-out as appropriate.

  4. Fill in the remaining fields.

  5. Press CTRL+S to save.
