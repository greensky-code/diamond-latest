Using Site Validation

You can use Site Validation functionality to do the following:

1. Specify a PCP and Site when adding an initial member eligibility record

Specify a PCP and Site when you add an initial member eligibility record using keyword MEMBR (Member Master). PCP and Site can be required by the line of business. In addition, the line of business may require assignment of a member to a Site which is certified to provide particular medical services.

For more information, see Specify a PCP and Site when adding an initial member eligibility record.

2. Add a Site to an existing member eligibility record

Add a Site to an existing member eligibility record using keyword MELIG (Member Eligibility History).

For more information, see Add a site to an existing member eligibility record.

3. Change the PCP and Site on an existing member eligibility period

Change the PCP and Site on an existing member eligibility period using keyword MEMBR Special menu command Provider Change. Do not use keyword MELIG for this purpose.

For more information, see Change the PCP and Site on an existing member eligibility period.
