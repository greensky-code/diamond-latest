Description

Company Code

R

General ledger company assigned to the claim detail line when the entries for parameter values match those on the claim.

Selected from a predefined list.

Company codes are maintained in COMPF.

G/L Ref Code

R

General ledger reference code assigned to the claim detail line when the entries for parameter values match those on the claim.

Selected from a predefined list.

General ledger reference codes are maintained in GLREF, where the general ledger account numbers that correspond to each reference code are stored.

Description

O

Description of the record.

Manually entered.

 

GLASSIGN1-6 Parameter Values

O

The determinant that defines company code and general ledger reference code assignment for claim detail lines.

Selected from a predefined list.

Field labels will correspond to the entries in PARAM records.

Each of the up to six parameters carries a determinant.
