Calculate a Check Print Job for Administrative Fees

Use the following procedure to calculate a check print job for administrative fees, only.

To calculate a check print job for administrative fees

  1. Press F3, type CKPRT, and then press ENTER.

  2. On the File menu, click New.

  3. In the Payable Type list, click M – Medical.

  4. In the Calc Admin Fee list, click O – Only Admin Fee.

  5. In the Item Selection Criteria section box, fill in the fields on the screen. For more information, see CKPRT Field Definitions.

  6. In the Next Action section box, click Calc.

  7. On the File menu, click Run Job.
