Add a Quality Program to a Provider Contract

Use the following procedure to add a quality program to a provider contract.

To add a quality program to a provider contract

  1. Press F3, type PROVC, and then press ENTER.

  2. In the Provider ID box, type the provider ID, and then press TAB. F5 lookup is available.

  3. In the picklist, click the provider contract.

  4. On the Special menu, click Quality Program Eligibility.

  5. In the Quality Program Selection dialog box, click Add.

  6. In the Quality Program list, click the quality program.

  7. In the Effective Date box, type the effective date of the program. You can type a date greater than the provider contract effective date or equal to or less than the provider contract termination date.

  8. In the Term Date box, type the termination date of the program (optional).

  9. In the Term Reason list, click the termination reason code. This field is required if you typed a date in the Term Date box.

  10. Click OK.
