About LBPEC

Use keyword LBPEC (Line of Business Pre-Existing Conditions) to specify the groups in a line of business for which one or more default pre-existing condition rules apply. The default pre-existing condition rules are considered "default" rules in that they are only applied to a claim when a member-specific rule does not apply.

You define default pre-existing condition rules using keyword PXRUL (Pre-Existing Condition Rule). You then use keyword LBPEC to associate PXRUL default pre-existing condition rules with a specific line of business and certain groups within that line of business.

Member-specific rules are defined using keyword MCOND (Member Conditions).
