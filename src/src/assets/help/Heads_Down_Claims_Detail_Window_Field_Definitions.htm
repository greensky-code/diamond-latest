Heads Down Claims Detail Window Field Definitions

Field Name  Req Description Source Comments

DOS From (Date)

O

Claim detail line’s from date of service.

Manually entered

 

DOS To (Date)

O

Claim detail line’s to date of service.

Manually entered

 

POS

O

Claim detail line’s place of service code.

Manually entered

If you add/enter an additional claim detail line, the Heads Down Claims executable defaults the place of service code from the current claim detail line to the next claim detail line.

TOS

O

Claim detail line’s type of service code.

Manually entered

 

Proc

O

Claim detail line’s procedure code.

Manually entered

 

Mod 1-4

O

Claim detail line’s modifier codes.

Manually entered

 

DX Ptr (1-4)

O

Claim diagnosis pointers.

Manually entered

Accept the default value of 1 (in the first DX Ptr entry box) if the claim detail line is related to the corresponding diagnosis code entered in DX 1 on the claim header.

You can override the default by entering a pointer value of 2 or 3 or 4 in the first DX PTR entry box provided a corresponding DX 2-4 diagnosis code value has been entered in the claim header.

If you add/enter an additional claim detail line, the Heads Down Claims executable defaults the diagnosis pointer value from the current claim detail line to the next claim detail line.

Billed Amt

O

Claim detail line’s billed amount.

Manually entered

Corresponds to HCFA 15 "Charges."

Qty

O

Claim detail line’s quantity.

Manually entered

Corresponds to DIAMOND’s "Quantity."

Anes Start

O

Anesthesia start time.

Manually entered

Enter in 24 hour clock format with no punctuation (0700 = 7:00 a.m., 1300 = 1:00 p.m.)

Anes End

O

Anesthesia end time.

Manually entered

Enter in 24 hour clock format with no punctuation (0700 = 7:00 a.m., 1300 = 1:00 p.m.)

EPSDT

O

Indicates if the claim detail line service is related to family planning.

Selected with checkbox

 

EMG

O

Indicates if the claim detail line service is related to an emergency.

Selected with checkbox

 

COB

O

Code to indicate the liability of the primary carrier.

Manually entered

Valid values:

A-Z Reserved for national use.

0-9 Reserved for local use.
