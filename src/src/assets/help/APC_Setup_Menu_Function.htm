DIAMOND Action (Grouper Error)

C

Click the action you want to occur to the institutional claim line item if there is an APC Grouper error.

Required if the Use APC Grouper check box is selected.

Selected from predefined list

Select from the following list:

  • Hold - In keyword INCLM, sets the Proc Status list to H – Held

  • Deny - In keyword INCLM, sets the Claim Status list to D – Denied

DIAMOND Action (Pricer Error)

C

Click the action you want to occur to the institutional claim line item if there is an APC Pricer error.

Required if the Use APC Pricer check box is selected.

Selected from predefined list

Select from the following list:

  • Hold - In keyword INCLM, sets the Proc Status list to H – Held

  • Deny - In keyword INCLM, sets the Claim Status list to D – Denied

Edit Type

C

Click the edit type the APC Editor will carry out.

Required if the Use APC Editor check box is selected.

Selected from predefined list

Select from the following list:

  • CCI Only—Correct Coding Initiative, which requests edits on every possible pairing of HCPCS codes inputted to the APC Editor.

  • OCE Only—Outpatient Code Edits.

  • OCE plus CCI—Outpatient Code edits + Correct Coding Initiative.
