Entering Dates

The DIAMOND program uses MM/DD/YYYY as the standard date entry/display format for all server processing. Client computer processing date entry is controlled by the Regional Settings (or Regional Options) in the Windows Control Panel.

You must enter the year as four digits (for the century and year) on all Date of Birth (DOB) fields or when a date is used as a determinant (Determinant > From/Through fields).

You can enter the year using two digits (for the year) or four digits (for the century and year) on all other date fields.

You can change the server processing date entry/display format with the DATEFORMAT parameter in keyword PARAM (System Parameters). However, you must also change the client processing date entry/display format to match the server processing format. For more information, see PARAM, Parameter.
