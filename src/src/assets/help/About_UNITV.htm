About UNITV

Use UNITV to enter the unit values for a procedure code for a given unit value scale and geographic region. This file is used exclusively for claims priced using the unit value calculation method. Each UNITV record may have several unit values—for professional and technical components, total units, and anesthesia units. An entry on PRULD tells DIAMOND which unit value to retrieve when pricing a claim.
