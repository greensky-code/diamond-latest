Site Validation - Overview

Site Validation logic enables you to:

  • Assign a facility site on a provider enrollment record

  • Assign a PCP and facility site on a member eligibility record

  • If required by the line of business, make these assignments only if the site is certified for a particular medical services program. For example, the Child Health and Disability Prevention (CHDP) program.

You must designate a PCP and facility site on a member eligibility segment when required by the member’s line of business. The PCP must have a valid provider contract, provider address, and provider enrollment record matching the eligibility record and site. This match includes effective period, LOB, Panel, IPA, and, if required, certification.

For more information, see the sections below and Site Validation, How DIAMOND Thinks.

Setting Up Site Validation

1. Keyword PARAM parameter MEMSITEVAL - Enable Site Validation

2. Keyword SITEM - Add a Site Code

3. Keyword SITEA - Add a site address

4. Keyword LINBS - Specify Site Validation options for a line of business

5. Keyword PROVA - Add a provider address linked to a site address

6. Keyword PROVE - Specify a Site and linked provider address on a provider enrollment rule

For more information, see Setting Up Site Validation.

Using Site Validation

You can use Site Validation functionality to do the following:

1. Keyword MEMBR - Specify a PCP and Site when adding an initial member eligibility record

2. Keyword MELIG - Add a site to an existing member eligibility record

3. Keyword MEMBR Special menu command Provider Change - Change the PCP and Site on an existing member eligibility period

For more information, see Using Site Validation.
