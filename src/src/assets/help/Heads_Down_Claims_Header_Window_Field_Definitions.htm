O

Claim total amount paid by the patient.

Manually entered

If Total Chg and Amount Paid amounts are equal, the Heads Down Claims executable:

  • Assumes the patient paid the provider directly and, therefore, the patient should be reimbursed (not the vendor).

  • Turns on the "Pay Subscriber" indicator field for this claim in the claims text file.

See PSCLM > Commands on the Menu Bar > Pay Subscriber.

Insured’s ID

O

The insured’s (patient’s) ID.

Manually entered (no dashes/slashes)

May also represent:

  • the insured’s DIAMOND subscriber ID.

  • the insured’s SSN as stored in DIAMOND.

  • an ID not related to either of the above.

When claims text file is processed, the IDE module attempts to find and/or verify an existing DIAMOND subscriber ID and person number based on the entered ID.

If insured’s ID is not entered, the IDE module attempts to derive an existing DIAMOND subscriber ID and person number based on other data entered in this window.

See Intelligent Data Enhancer (IDE) > Member Search documentation.

Tax ID

O

DIAMOND vendor tax ID.

Manually entered

If you enter a tax ID, you must enter it twice for verification purposes.

Enter the tax ID, press TAB to the Tax ID Confirm field, re-enter the tax ID, press TAB to the next field.

If the entered tax IDs match, DIAMOND displays (****) in both fields.

If the entered tax IDs do not match, DIAMOND displays a "Re-enter Tax ID" message and the cursor is re-positioned in the Tax ID field.

Leave both fields blank if the tax ID is unknown.

Tax ID Confirm
