PROVC Commands on the Menu Bar

 

Command             Menu   Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Auth Costing

S

Stores a price schedule and price region to be used in automatic costing of authorizations when this provider is the "referred to" provider for a member associated with this contract.

Claim Hold Codes

S

Use to place a hold (pend) on every claim processed under this contract; includes reason code and effective date.

Contract Copy

S

A shortcut for entering new contracts, allowing data from an existing contract to be copied over to a new contract record.

Contract Pricing Details

Details

S

Stores details about how claims for this provider, processed for members associated with this contract, are priced.

Contract Renew

S

A shortcut for terminating one contract and creating a new contract with an effective date immediately following the terminated one.

Covering Provider Relationships

S

Displays information for this provider from Covering Provider Groups (CVPVG).

DRG/APC Information
(contract level)

Details

S

At the contract level, a mechanism for calling an external DRG grouper or pricer, or an external APC editor, grouper or pricer during claims processing.

DRG Information
(pricing detail level)

Details

S

At the pricing detail level (within a contract), a mechanism overriding certain entries made to DRG pricer information at the contract level. Does not override DRG grouper or APC editor, grouper or pricer information entered at the contract level.

Notes

Details

N

This command is only available if the PROVC window has been associated with at least one note type in NOTEW.

Takes you to the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.

Pharmacy

S

For pharmacy providers, the average wholesale price percentage and dispensing fee used to price pharmacy claims.

Quality Program Eligibility

S

Use to add or terminate a quality program for a provider. Displays the Quality Program Selection dialog box. It contains the following fields:

  • Quality Program – Displays the quality programs you can select for the provider.

  • Effective Date – Displays the effective date of the quality program. You can type a date greater than the provider contract effective date or equal to or less than the provider contract termination date.

  • Term Date – Displays the termination date of the quality program. You can type a date that is equal to or greater than the date in the Effective Date box, or equal to or less than the provider contract termination date.

  • Term Reason – Displays the termination reason code of the quality program. Only reason codes with the Reason Code Type list set to TM – Termination in keyword REASN (Reason).

Vendor Add

S

Use to quickly add a vendor to keyword VENDR (Vendor Master).

View IPA Info

S

Displays the member’s associated IPA contract information, including Delegated Responsibilities, Special Programs, Service Area, and the Postal Codes included in this Service Area.
