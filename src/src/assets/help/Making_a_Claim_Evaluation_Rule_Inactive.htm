Make a Claim Evaluation Rule Inactive

Use the following procedure to make a claim evaluation rule inactive.

To make a claim evaluation rule inactive

  1. Press F3, type CERUL, and then press ENTER.

  2. In the Claim Type list, click D – Dental.

  3. In the Rule Order ID box, type the claim evaluation rule. F5 lookup is available.

  4. Click to clear the Active Rule check box.

  5. On the File menu, clicks Save.
