NDCCD Field Definitions

Field Name  Req Description Source Comments

Avg Wholesale Price

O

Type the average wholesale price of the drug.

Manually entered.

You can type up to 16 digits before the decimal and 2 after the decimal.

Code Qualifier

D

Displays the NDC code format, N4.

System generated.

N4 is the only supported format.

Description

O

Type the description of the drug.

Manually entered.

You can type up to 200 alphanumeric characters.

Drug Class

O

Type the class of the drug.

Manually entered.

You can type up to 6 digits.

Effective Date

O

Type the date this drug code is effective.

Manually entered.

The valid format is MM/DD/YYYY

For example, 11/25/2001

NDC Code

R

Type the 11-digit ID for this code.

Manually entered.

Do not type any dashes. DIAMOND automatically inserts a dash after the 5th and 9th characters (for example, 55555-4444-22).

If you need to search through your NDC Codes database, then press F5.

Term Date

O

Type the date this drug code become ineffective.

Manually entered.

The valid format is MM/DD/YYYY

For example, 11/26/2001
