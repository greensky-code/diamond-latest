Grant a Template Access to a Special Menu Command

Use the following procedure to grant a security template access to a command on the Special menu. By granting a security template access to a command, you also are granting all users assigned this template the same access.

To grant a template access to a Special menu command

  1. Press F3, type SFUNC, and then press ENTER.

  2. In the User ID box, type the user ID of the template, and then press TAB. F5 lookup is available.

  3. On the File menu, click New.

  4. In the Function list, click the command for which you want to grant access. For a list of commands and their descriptions, see About SFNDS.

  5. Click to select the Access check box.

  6. On the File menu, click Save.
