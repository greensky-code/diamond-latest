Run an A/P Update Job

Use the following procedure to run an Accounts Payable (A/P) update job for medical claims or capitation processing.

To run an A/P update job

  1. Press F3, type APUPD, and then press ENTER.

  2. On the File menu, click New.

  3. In the Payable type list, do one of the following:

  4. If you want to process agent commissions, then click A – Agent Commission.

  5. If you want to process capitation transactions, then click C – Capitation.

  6. If you want to process medical claims, then click M – Medical.

  1. In the Post Month box, type the G/L posting month. Type the date in MM/YYYY format. For example, 11/2002

  2. If desired, fill in the remaining fields in the window. For more information about the fields in the window, see APUPD Field Definitions.

  3. On the File menu, click Run Job. The A/P update job runs. When the job ID in the picklist displays Successful Run in the Status column, the job has completed.

Tips

  • To refresh the display of the Status column in the picklist, on the Window menu, click Refresh Job Status.

  • If the Status column displays "Error," then you can view the error messages by clicking Processing Messages from the Window menu.
