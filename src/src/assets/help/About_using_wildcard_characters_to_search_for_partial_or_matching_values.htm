About using wildcard characters to search for partial or matching values

When looking up data, you can use wildcard characters as placeholders for other characters. This may be convenient for the following reasons:

  • You know only part of the value you want to search

  • You want to search for a value that contains an apostrophe ( ‘ ). For example, the last name "O’Higgins."

  • You want to search for a value with a combination of upper and lower case letters. For example, the last name "McGregor."

You can use the _ (underscore) and % characters in the Lookup dialog box.

Using the % character

The % character matches any string of zero or more characters. It can be used in the middle or end of a search string. DIAMOND processing automatically adds % to the end of a search string.

The following table displays examples using the % wildcard character.

Using the _ (underscore) character

The _ (underscore) character matches any single character. It can be used in the middle of end of a search string. DIAMOND processing automatically adds % to the end of the search string.
