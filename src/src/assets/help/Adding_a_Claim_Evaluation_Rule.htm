Add a Claim Evaluation Rule

Use the following procedure to add a claim evaluation rule. When you add a claim evaluation rule, you must complete the following procedures:

  • Add the rule

  • Enter determinants

  • Enter field value determinants

  • Enter matching history

To add the rule

  1. Press F3, type CERUL, and then press ENTER.

  2. In the Rule Order ID box, type the rule order ID, and then press TAB. When a claim detail line is entered, it searches through the claim evaluation rules from lowest (0001) to highest (9999). When a match is found, it stops searching though the rules. Therefore, for specific rules, type a lower rule order ID. For general rules, type a higher rule order ID.

  3. In the Description box, type the description.

  4. In the Action list, do one of the following:

  5. If you want to display a message to the operator if a claim line matches this rule, then click Warn.

  6. If you want to hold the claim line if it matches this rule, then click Hold.

  7. If you want to deny the claim line if it matches this rule, then click Deny.

  1. In the Reason Code box, type the Reason code for which you want to use if the claim line is held or denied. F5 lookup is available. This box is unavailable if in the Action list, you selected Warn.

  2. In the Message to Operator box, type the message you want to display if a claim line matches this rule. You can type up to 60 alphanumeric characters.

To enter determinants

  1. Click to select the focus box that includes the Sequence, Determinant Table, and Determinant Column boxes.

  2. On the File menu, click New.

  3. In the Sequence box, type a sequence number for the rule. Claim lines evaluate sequences in numerical order from lowest (0001) to highest (9999).

  4. In the Determinant Table box, press F5. The Tables and Columns dialog box displays.

  5. In the Table Name box, click the table.

  6. In the Column Name box, click the column.

  7. Click OK.

  8. On the File menu, click Save.

To enter field value determinants

  1. Click to select the focus box that includes the Operator, From Value, and Thru Value boxes.

  2. On the File menu, click New.

  3. In the Operator list, do one of the following:

  4. If you want to include values, then click Include.

  5. If you want to exclude values, then click Exclude.

  1. In the From Value box, type a beginning value.

  2. In the Thru Value box, type an ending value.

  3. On the File menu, click Save.

  4. Repeat steps 1 through 6 for every field value determinant you want to add.

To enter matching history

  1. In the Timeframe list, click the period of time. This specifies how far back in tooth history to look when applying this rule.

  2. If in the Timeframe list, you selected Y – Calendar Year, C – Contract Period, E – Elapsed Year, or D – Elapsed Days, then in the #Years/Days box, type the number of years or days.

  3. In the Primary list, click the primary match type for which you want to use when this rule searches the member’s tooth history.

  4. In addition to the primary match, if you also want to match a provider when searching through tooth history, then click to select the Prov check box. If you want to include or exclude specific providers, then on the Special menu, click Additional Providers.

  5. In addition to the primary match, if you also want to match a procedure code when searching through tooth history, then click to select the Proc Code check box. If you want to include or exclude specific procedure codes, then on the Special menu, click Additional Procedure Codes.

  6. In addition to the primary match, if you also want to match a benefit package when searching through tooth history, then click to select the Benef Pkg check box. If you want to include or exclude specific benefit packages, then on the Special menu, click Additional Benefit Packages.

  7. In addition to the primary match, if you also want to match a medical definition when searching through tooth history, then on the Special menu, click Additional Medical Definitions. The Medical Definitions dialog box displays. To add medical definitions, click Add. To return to the Claim Evaluation Rules window, click OK.

  8. On the File menu, click Save.

Tip

Clearing this Active Rule check box make the rule inactive. That is, claim lines will not evaluate this rule when searching through the claim evaluation rule records.
