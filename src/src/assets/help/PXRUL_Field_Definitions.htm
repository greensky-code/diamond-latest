PXRUL Field Definitions

Field Name

Req

Description

Source

Comments

PEC Rule ID

R

Unique identifier of this pre-existing condition rule

Manually entered

F5 lookup available.

Effective Date

R

First date this pre-existing condition rule is effective

Manually entered

Valid format is defined using system parameter DATEFORMAT

Term Date

O

Last date this pre-existing condition rule is effective

Manually entered

Valid format is defined using system parameter DATEFORMAT.

Term reason

O

Reason for termination of this pre-existing condition rule

Selected from predefined list

Valid values are maintained using keyword REASN, with reason type of TM - Termination.

Description

O

Text description of this pre-existing condition rule

Manually entered

 

Diagnosis Type

R

The type of diagnosis or class value range for this pre-existing condition rule

Selected from predefined list

Valid values:

  • CODE - Diagnosis code

  • Class 1 - Diagnosis class 1

  • Class 2 -   Diagnosis class 2

  • Class 3 - Diagnosis class 3

From Value

O

Diagnosis code or class value, or the beginning of a range of diagnosis codes or class values for which this pre-existing condition rule applies

Selected from predefined list

F5 lookup is available.

Diagnosis codes and class values are maintained in keyword DIAGN.

Surgical procedure diagnosis codes are not valid diagnosis code values for this field.

Thru Value

R

End of a range of diagnosis codes or class values for which this pre-existing condition rule applies.

Selected from predefined list

F5 lookup is available.

Diagnosis codes and class values are maintained in keyword DIAGN.

Surgical procedure diagnosis codes are not valid diagnosis code values for this field.

Eff Date

R

First date the diagnosis code range or diagnosis class range applies for the pre-existing condition rule.

Manually entered

 

Term Date

O

Last date the diagnosis code range or diagnosis class range applies for the pre-existing condition rule.

Manually entered
