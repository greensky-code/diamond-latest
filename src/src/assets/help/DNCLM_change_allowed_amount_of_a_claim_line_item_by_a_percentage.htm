Changing the Allowed Amount of a Claim Line by a Percentage

Restrictions

Instructions

Use the following procedure to reduce or increase the allowed amount of a claim line by a percentage.

Restrictions

None.

Instructions

Step

Comment

1. From DNCLM, display the claim detail line you want to change.

 

2. On the Special menu, click Allowed Factor.

 

3. In the Allowed Factor box, type the percentage by which you want to reduce or increase the allowed amount for the first modifier code.

 

4. If necessary, in the Sec. Allowed Factor field, type the percentage by which you want to reduce or increase the allowed amount for the second modifier code.

 

5. Click OK.
