COBOR Field Definitions

Action Code

R

Indicates the specific COB Order to apply to a COBHS record if all COBOR rule criteria are met

Selected from predefined list

Valid values:

O Other

P Primary

S Secondary

T Tertiary

Description

R

Meaningful description of the COB Order Rule

Manually entered

 

Determinant Column

R

Column to use as a COB Order rule determinant

Selected from predefined list

Valid values:

Any column in the selected Determinant table.

Determinant Table

R

DIAMOND table from which to select individual COB Order rule determinants

Selected from predefined list

Valid values:

MEMBER_ELIGIBILITY_HISTORY

MEMBER_MASTER

MEMBER_OTHER_COVERAGE (not available if COBOR Subs Flag is selected)

PREMIUM_MASTER

From Value

O

Determinant value or beginning of a determinant value range

Manually entered

If you select a Sec Determinant Column, From Value is not available since you are comparing the Determinant Column and Sec Determinant Column values.

Important: If you are entering a date(s) in the From/Through determinant fields, you must enter the date in MM/DD/YYYY format (including slashes). For example, 12/25/1999, 02/01/2000.

Operator

R

Comparison operator for the rule determinants

Selected from predefined list

Valid values:

Equal

Greater Than

Greater Than (or) Equal

Less Than

Less Than (or) Equal

NOT

Rule ID

R

COB Order Rule identifier

Manually entered

 

Rule Order

R

COB Order rule order number. Establishes the order in which the COB Order rules are evaluated.

Manually entered

 

Search Seq

R

Evaluation sequence number of a specific determinant within a COB Order rule

Manually entered

 

Sec Determinant Column

R

Specific secondary determinant to compare to the selected Determinant column

Selected from predefined list

Valid values:

Any column in the selected Determinant table.

Sec Determinant Table

R

DIAMOND table from which to select secondary COB Order determinant columns.

Selected from predefined list

Valid values:

MEMBER_ELIGIBILITY_HISTORY

MEMBER_MASTER

MEMBER_OTHER_COVERAGE

PREMIUM_MASTER

Subs Flag

R

Flag that indicates whether the determinant refers to the member or the member’s subscriber

Selected with check box

Valid values:

If the check box is selected, the determinant refers to the subscriber of the member selected on the COBHS record.

If the check box is clear, the determinant refers to the member selected on the COBHS record.

Thru Value

O

Ending value of a determinant value range

Manually entered

If you select a Sec Determinant Column, Thru Value is not available since you are comparing the Determinant Column and Sec Determinant Column values.

Important: If you are entering a date(s) in the From/Through determinant fields, you must enter the date in MM/DD/YYYY format (including slashes). For example, 12/25/1999, 02/01/2000.
