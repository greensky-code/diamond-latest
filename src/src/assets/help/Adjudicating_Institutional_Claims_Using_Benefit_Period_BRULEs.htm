Adjudicating Institutional Claims Using Benefit Period BRULEs

Enter benefit rules (BRULE) that have benefit renewal periods, enter appropriate required supporting records, and adjudicate institutional claims in the order as specified under Instructions.

1.   Add benefit rules

In keyword BRULE, add new institutional claim benefit rules for the required deductibles, coinsurance, copayments, limits, and/or out-of-pocket maximums.

Use BRULE – Renewal, Yrs./Months/Days, and From fields. See BRULE and Benefit Rules and Packages User Documentation for additional information.

2.   Add a benefit package

In keyword BENEF, add a benefit package for the benefit rules created in step 1. See BENEF – Benefit Package User Documentation for additional information.

3.   Add rules to packages

Add the benefit rules created in step 1 to the benefit package created in step 2.

4.   Add group detail

In keyword GRUPD, add a group detail record (to an existing group) with the benefit package created in step 2.

See GRUPD – Group Detail User Documentation for additional information.

5. Add members to the package

In keyword MEMBR, add members and associate them with the created group and benefit package.

See MEMBR – Member Master User Documentation for additional information.

6.   Enter claims for members

Enter and adjudicate Institutional claims for added members.
