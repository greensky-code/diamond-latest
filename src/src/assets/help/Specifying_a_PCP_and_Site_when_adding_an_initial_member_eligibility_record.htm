Specifying a PCP and Site when adding an initial member eligibility record

You can specify a PCP ID and Site when you add an initial member eligibility period using keyword MEMBR (Member Master).

The Site field is only available if Site Validation is enabled in system parameter MEMSITEVAL and the line of business Site check box is selected.

To specify a PCP and Site when adding an initial member eligibility record

  1. Press F3, type MEMBR, and then press ENTER.

  2. Fill in the member demographic fields.

  3. In the current eligibility period section box, fill in the fields preceding the PCP ID field as appropriate.

  4. In the PCP ID field, press F5 for lookup. DIAMOND logic filters the records displayed in the Lookup window to valid selections for this member. For information on this filtering, see Site Validation, How DIAMOND Thinks section Finding PCPs Using Site Validation.

  5. Select the desired record in the Lookup window. DIAMOND logic displays the values from the selected record in the PCP ID, Site, IPA, and Panel fields.
    Note
    You can also type a PCP ID in the PCP ID field. However, if you do so, DIAMOND logic does not include Match All Contracts logic nor are the IPA and Panel fields automatically populated.
    You can also press F5 to lookup in the Site box before entering a PCP ID. If you do, the DIAMOND application performs the same filtering logic and upon selection displays the same values as when you use F5 lookup in the PCP ID box. For more information, see Site Validation, How DIAMOND Thinks section Finding Sites Using Site Validation.

  6. Complete the remaining fields in the current eligibility period section box as appropriate.

  7. Press CTRL+S to save.
