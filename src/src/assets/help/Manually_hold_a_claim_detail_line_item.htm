Manually hold a claim detail line item

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to manually hold an institutional professional services, or dental services claim detail line item.

Restrictions

None.

Instructions

Step

Comment

1. At the Main Menu, press F3.

 

2. In the Keyword box, do one of the following:

  • If you want to hold on institutional claim, then type INCLM, and then press ENTER.

  • If you want to hold a professional services claim, then type PSCLM, and then press ENTER.

  • If you want to hold a dental services claim, then type DNCLM, and then press ENTER.

 

3. On the Claim header window, display the claim you want to manually hold.

 

4. On the Topic menu, click Detail File.

 

5. Display the claim detail line item you want to manually hold.

 

6. On the Topic menu, click Hold Reasons.

 

7. On the File menu, click New. Select the desired hold reason.

See INCLM, PSCLM, or DNCLM User Documentation for field descriptions of all fields included in the Topic Hold Reasons.
