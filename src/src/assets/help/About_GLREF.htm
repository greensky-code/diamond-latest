About GLREF

Use keyword GLREF (G/L Reference) to designate the codes that cross reference to general ledger (G/L) account numbers. DIAMOND uses G/L reference codes to create general ledger transactions related to financial processes such as cash receipt, premium billing, and claims payments. Each G/L reference code corresponds to a set of primary debit and credit accounts. For claims, the primary set is used to post the net amount; a secondary set of accounts can be used to post provider withhold amounts; a third set can be used to post interest calculation amounts; and a fourth set can be used to post discount calculation amounts.
