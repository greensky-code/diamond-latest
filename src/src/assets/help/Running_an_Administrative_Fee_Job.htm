Use the following procedure to run an administrative fee job to calculate administrative fees.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type JOBAD, and then press ENTER.

 

2. On the File menu, click New.

 

3. If you want to include or exclude one or more lines of business, then on the Special menu, click Line of Bus Selection.

 

4. In the Company Code box, type the company code (optional).

F5 lookup is available.

5. In the Vendor box, type the vendor (optional).

F5 lookup is available.

6. In the Claims From Entered Date, type the beginning date range of claims to include.

 

7. In the Claims Thru Entered Date, type the ending date of claims to include.

 

7. Click Calculate.

 

9. Do one of the following:

  • If you want to run the job now, then click Immediate.

  • If you want to run the job later, then click Deferred.

Deferred is future functionality.

10. On the File menu, click Run Job.
