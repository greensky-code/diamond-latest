Viewing an Administrative Fee for a Claim

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to view an administrative fee for a claim.

Restrictions

To view an administrative fee for a claim detail line, the Admin Fee check box must appear selected in the Line Item Detail section box in keyword CLDSP (Claims Display) or in the claims detail window in keywords DNCLM (Dental Services Claims), INCLM (Institutional Claims), or PSCLM (Professional Services Claims).

Instructions

Step

Comment

1. Press F3, type CLDSP, DNCLM, INCLM, or PSCLM, and then press ENTER.

 

2. Display the claim for which you want to view the administrative fee.

 

3. On the Special menu, click Administrative Fee Info.

 

Frequently Asked Questions

None.
