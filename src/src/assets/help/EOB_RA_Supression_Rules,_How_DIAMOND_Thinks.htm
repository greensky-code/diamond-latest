EOB/RA Suppression Rules, How DIAMOND Thinks

Use keyword SEBRA (Supress EOB/RA Rules) to add a EOB/RA suppression rule. These rules automatically suppress printing of EOBs and RAs under certain conditions based on fields in the claim header and detail.

Rules can be different for keywords PSCLM, INCLM, and DNCLM, and for EOB and RA.

Rules are ordered and each rule may have multiple determinants, and each determinant must be true for the rule to be applied.

Any changes to rules in keyword SEBRA will take effect only after quitting and restarting the DIAMOND program.
