Setting up Medicare + Choice functionality

Use the following to set up Medicare + Choice functionality. Effective date validations are only available for member eligibility periods with Medicare + Choice a line of business.

1. Keyword MEMMC (M+C History) - Add member Medicare + Choice information, Change member Medicare + Choice information, Delete member Medicare + Choice information

2. Keyword PARAM (Parameter) parameter MEMMEDAEFF - Enable validation of eligibility effective date against Medicare Part A effective date

3. Keyword PARAM parameter MEMMEDBEFF - Enable validation of eligibility effective date against Medicare Part B enrollment date

4. Keyword PARAM parameter MCMEMEFFDT - Enable validation that eligibility effective date is the first of the month

5. Keyword PARAM parameter MEDABTERMDT - Enable validation that the eligibility termination date is the first of the month following the Medicare Part A/B termination date
