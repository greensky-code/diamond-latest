Adjust Interest/Discount Amount Menu Command

Field Name

Req

Description

Source

Comments

Discount Amount

O

Amount of discount to apply to the adjusted claim detail line.

Manually entered

A negative or positive number.

The discount amount will only be applied to the claim payment if you enter it here before running APUPD and check print.

If this field is filled in, the Penalty Amount field is not available.

Interest Amount

O

Amount of interest to apply to the adjusted claim detail line.

Manually entered

A negative or positive number.

The interest amount will only be applied to the claim payment if you enter it here before running APUPD and check print.

If this field is filled in, the Discount Amount field is not available.

Penalty Amount

O

Amount of penalty to be applied to the claim detail line.

Manually entered.

A negative or positive number.

Reason for Adjust

O

Reason code for the manually applied discount or interest amount.

Selected from predefined list

Must be an AD (adjust) reason code type.

F5 lookup is available.
