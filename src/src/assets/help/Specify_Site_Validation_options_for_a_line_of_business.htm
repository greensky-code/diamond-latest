Specify Site Validation options for a line of business

For each line of business, you can specify what Site Validation logic to apply.

To specify Site Validation options for a line of business

  1. Press F3, type LINBS, and then press ENTER.

  2. In the Line of Business box, type the line of business for which you want to specify Site Validation options. F5 lookup is available.

  3. If you want to require Site Validation for this line of business, select the Site check box.

  4. If you want to require that a site be certified for specific medical services, select the Cert check box. An age range may also apply as described in the next step.

  5. If you want to require an age range be satisfied before requiring certification of a site, type an age range in the Age From and Age Through boxes.
    If you specify an age range, both the member age and the provider enrollment rule's age range must satisfy this age range for site certification to be required.

  6. Press CTRL+S to save your changes.
