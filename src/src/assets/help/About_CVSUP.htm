About CVSUP

Use keyword CVSUP (Clear, Void, Stop Update) to do the following:

  • Make accumulated clear, void, stop check/EFT transaction status changes

  • Print the Clear, Void, Stop Update Audit Report

  • Remove claim detail line payment data and accounts payable transactions associated with "Reissued" status changes

  • Create offsetting claim detail line payment data and accounts payable transactions associated with "Reversed" status changes

When keyword CVSUP updates check reissues, the original issue status, reason, and date are written to keyword PSCLM (Professional Services Claims), INCLM (Institutional Claims), or DNCLM (Dental Services Claims) Notes.

Run a job in keyword CVSUP after you enter check/EFT transaction status changes in keyword CHECK (Check Register).
