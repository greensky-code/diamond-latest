VPDSP Field Definitions

Field Name

Req

Description

Source

Comments

Account Number

 

The identifying number of the vendor's advance payment account.

Manually entered

 

Address Line 1

 

The first address line for the vendor.

Manually entered

 

AP Status

 

The accounts payable status of the transaction(s).

Selected from predefined list.

Valid values are:

  • F

Final

  • H

Hold

  • P

Paid

  • N

Paid No Check

  • U

Unposted

Check/EFT Date

 

The date of the check or EFT transaction.

Manually entered

 

Check/EFT Number

 

The number of the check or EFT transaction.

Manually entered

 

Check/EFT Type

 

The type of transaction; either check or EFT.

Selected from predefined list

Valid values are:

  • Check

  • EFT

Vendor ID

 

The ID of the vendor.

Manually entered or selected from predefined list
