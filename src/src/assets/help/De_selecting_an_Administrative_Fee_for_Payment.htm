De-select an Administrative Fee for Payment

Use the following procedure to create an Accounts Payable (A/P) selection job for A/P processing. By creating an A/P selection job, you can de-select an administrative fee for payment.

To de-select an administrative fee for payment

  1. Press F3, type APSEL, and then press ENTER.

  2. On the File menu, click New.

  3. Fill in the remaining fields in the window. For more information about the fields in the window, see APSEL Field Definitions.

  4. In the Action Type list, click Deselect for Pay.

  5. On the File menu, click Run Job. The A/P Selection job runs. When the job ID in the picklist list displays Successful Run in the Status column, the job has completed.

Tips

  • To refresh the display of the Status column in the picklist, on the Window menu, click Refresh Job Status.

  • If the Status column displays "Error," then you can view the error messages by clicking Processing Messages from the Window menu.
