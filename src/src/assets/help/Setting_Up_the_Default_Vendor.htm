Setting Up the Default Vendor

Restrictions

Instructions

Restrictions

None.

Instructions

Step

Comment

1. In keyword VENDR, create the default vendor (entry values represent examples).

  1. Vendor ID = DEFAULTV (clearly identifies the default vendor)

  2. Full Name = Default Vendor

  3. Vendor Type = Medical

  4. Vendor 1099 Flag = No 1099

You are creating the default vendor master record that will be linked to the default provider record.

You will override the default vendor name/address when you pay a claim from an out-of-area, non-participating, or one-time provider.

2. Create the default vendor address using the VENDA window

  1. Vendor ID = DEFAULTV

  2. Address Line 1/Zip Code = Default Vendor/94102

You are creating the default vendor address record that will be linked to the default provider record.

You will override the default vendor address when you pay a claim from an out-of-area, non-participating, or one-time provider.
