About CLTRN

Use keyword CLTRN (Claims Transmittals) to request and post or cancel batch runs of Explanations of Payments and Batch Transmittals for pre-priced claims. Batch transmittals document the cost savings resulting from preferred provider contracts so that percentages of savings can be calculated. In DIAMOND percentage of savings arrangements are maintained in Group Percentage of Savings [GRPCT].

This keyword should only be used for claims for which you do not administer benefits or print checks.
