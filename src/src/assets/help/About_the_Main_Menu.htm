About the Main Menu

When you start the DIAMOND program, the Main Menu displays icons for each major module or functional group. Keywords are functions within these groups. When a functional group icon is selected, the keywords associated with it display in the Keyword List box in the lower right-hand corner. For some modules, there may be more keywords than can be shown in the Keyword List box. In these cases, a scroll bar appears next to the list. To display a keyword window, in the Keyword List box, select the keyword and press ENTER, or double click the keyword. See Figure 2.

You can also use the Shortcut Menu to display a keyword window.
