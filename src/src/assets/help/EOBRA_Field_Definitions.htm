EOBRA Field Definitions

Field Name

Req

Description

Source

Comments

Bank Acct Code

C

The displayed Bank Account Code of the batch job or a selected Bank Account Code for a single transaction.

System displayed for Mode All; selected from a predefined list for RA, Mode Single.

Not used for EOB, Mode Single.

Check/EFT No.

C

The check or EFT transaction number.

Selected from a predefined list

Applies to RA, Mode Single only.

CKPRT Job ID

D

The Job ID from CKPRT post.

Displayed

Will be displayed for the following requests only:

  • EOB or RA, Mode All.

  • RA, Mode Single, after selecting Type, Bank Acct Code and Check No.

  • EOB, Mode Single, after selecting Member ID, Claim Type, and Claim No.

Claim No.

C

The claim number associated with the EOB to be printed or viewed.

Selected from predefined list

Applies to EOB, Mode Single only, after the Member ID is entered.

Claim Type

 

C

The claim type associated with the EOB to be printed or viewed.

Selected from predefined list

Applies to EOB, Mode Single only, after the Member ID is entered.

Valid values are:

  • Professional

  • Institutional

  • Dental

Deferred

 

Not available.

 

 

Immediate

R

Prints or displays the EOBs or RAs immediately.

Selected from predefined list

 

Member ID

C

Unique identifier of the member associated with the check or EFT transaction.

Selected from predefined list

Applies to EOB, Mode Single only.

Mode

R

Selection of either a single transaction or a set.

Selected from predefined list

Valid values are:

  • All: All transactions in selected batch job.

  • Single: For EOBs, all transactions for a specified member/claim type/claim number within a batch job. For RAs, a specific check/EFT transaction with the batch job.

Output Type

 

R

Indicator of whether print/view is of RAs or EOBs.

Selected from predefined list

For either output type, applies to posted CKPRT or MCHEK jobs.

Valid values are:

  • EOB

  • RA

Print

O

Sets up and prints either alignment records or all records with no alignment opportunity.

Selected from predefined list

Available as an option when menu command Run Job is selected.

Valid values are:

  • Print Alignment: Prints alignment records only for the selected transactions.

  • Print all EOBs/RAs: Prints for all selected transactions without alignment step.

  • Cancel: Cancel job request.

Type

R

For RA, Single Mode only, a selection of either check or EFT.

Selected from predefined list

Valid values are:

  • Check

  • EFT

Vendor Address

D

The address of the vendor associated with the check or EFT transaction.

Displayed

 

Vendor ID

D

Unique identifier of the vendor who was paid by the selected transaction.

Displayed

 

View

O

When selected, displays EOBs or RAs for the requested transactions.

Selected from predefined list

Available as an option when menu command Run Job is selected.
