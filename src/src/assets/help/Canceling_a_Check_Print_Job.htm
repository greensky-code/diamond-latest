Cancel a Check Print Job

Use the following procedure to cancel a check print job. Once canceled, you can run the job anew, as if the original job had never taken place.

Important Note  You can only cancel a check print job with the status set to Calculated.

To cancel a check print job

  1. Press F3, type CKPRT, and then press ENTER.

  2. In the picklist, click the check print job for which you want to cancel.

  3. Click Cancel.

  4. On the File menu, click Run Job.
