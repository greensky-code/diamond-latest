Add a Reason Code for Holds.

Use the following procedure to add a reason code for holds. Hold reason codes are applied during claims and authorization processing.

To add a reason code for holds

  1. Press F3, type REASN, and then press ENTER.

  2. On the File menu, click New.

  3. In the Reason Code box, type the reason code ID. You can type up to 5 alphanumeric characters.

  4. In the Reason Code Type list, click HD - Hold.

  5. In the Description box, type the description of the hold reason code (optional).

  6. If you want to make this reason code unclean, click the Unclean Indicator check box.

Caution â€“ once you select the Unclean Indicator, you cannot clear it. If you want to clear the unclean indicator, you must create a new reason code.

The Unclean Indicator only works on hold or deny reason codes. When a hold or deny reason code that has the Unclean Indicator selected is applied to a claim detail line, that line begins to accumulate unclean days that can affect interest payments applied to that claim. For information on the unclean indicator, see Claims Discount and Interest Calculations, How DIAMOND Thinks.

  1. On the File menu, click Save.
