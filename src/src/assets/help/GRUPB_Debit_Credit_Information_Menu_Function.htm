GRUPB Debit/Credit Information Menu Command

Field Name  Req Description Source Comments

Account No.

R

Account number for this group.

Manually entered.

 

Type

O

Account information type.

Selected from a predefined list.

Types are maintained in CODES with a type ACCTINFTYPE.

Credit card account types must begin with CR.

Use

R

Use of account.

Selected from a predefined list.

Valid values are

Credit

Debit

Both

Each customer (group or individual) may have one Debit and one Credit account with the same effective date, or one Both account.

If a customer has a Debit or Both account, the new balance is written to the pmb_eft_interface table.

Eff Date

R

Date account is effective.

Manually entered.

 

Exp Date

C

Date a credit account expires.

Manually entered.

Required if Type begins CR; not used otherwise.

Status

O

Account status.

Selected from a predefined list.

Valid values are

Active

Inactive

Prenote: used for testing EFT interface with bank.

Routing Number

C

EFT routing number.

Manually entered.

Not used if Type begins CR; otherwise optional.
