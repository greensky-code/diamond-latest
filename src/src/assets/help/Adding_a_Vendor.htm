Add a Vendor

Use the following procedure to add a vendor and vendor address.

To add a vendor

  1. Press F3, type VENDR, and then press ENTER.

  2. In the Vendor ID box, type the vendor ID, and then press TAB. You cannot type a vendor ID beginning with the letter "S".

  3. Fill in the remaining fields in the window. For more information about the fields in the window, see VENDR Field Definitions.

  4. On the File menu, click Save.

To add a vendor address

  1. On the Topic menu, click Addresses.

  2. On the File menu, click New. The primary mailing address must be entered first.

  4. On the File menu, click Save.

  5. Repeat steps 2 through 4 for each address you want to add.
