Add Record Menu Command

The Heads Down Claims executable adds an entire claim record or a claim detail line record to the claims text file depending on the active window (Header or Detail).

Active Window = Header

  1. Press CTRL+A. The executable displays a blank Header window.

  2. Enter the new claim header/detail data.

  3. Use Topic Menu > Header to toggle back to the Header window.

  4. Press CTRL+A again to append the entered claim to the current claims text file and enter another claim.

  5. When you have completed all claims entry, press CTRL+S to save the claims text file.

Active Window = Detail

  1. Press CTRL+A. The executable opens an additional claim detail line record for entry.

  2. Enter the additional claim detail line record data.

  3. Press CTRL+A again to open an additional claim detail line record for entry.

  4. When you have completed all claims entry, press CTRL+S to save the claims text file.
