Setting up Notes Notification

1. Select a default notes priority level for each note type in keyword NOTET

The default note priority level is 0 for each new note type you create. You can change the default note priority level of a note type at any time.

Caution   When you change the note priority level of a note type it only affects notes created after the change, not notes that existed prior to making the change.

2. Set the NOTEALERT parameter 1 to yes

This parameter is a toggle for the notes notification functionality. When the parameter is set to Y (yes), notes notification is in effect, when it is set to N (no), notes notification is not in effect.

This parameter sets notes notification on or off for all note priority levels that you have assigned to your organization’s Note Types. You cannot set notification on for some priority levels and off for other priority levels – they are either all on or all off.
