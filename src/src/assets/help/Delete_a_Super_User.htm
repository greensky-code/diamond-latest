Delete a Super User

Use the following procedure to delete a super user.

Important Note Â  To delete super users, you must be a super user or have been granted access to the Delete Super User command on the Special menu in keyword SUSER (Function F_DEL_SUPERUSER).

To delete a super user

  1. Press F3, type SUSER, and then press ENTER.

  2. In the User ID box, type the user ID, and then press TAB. F5 lookup is available.

  3. On the Special menu, click Delete Super User. The user is deleted from the DIAMOND program, but not dropped from the database.

  4. Forward the ID of the deleted user to the database administrator/security administrator to be dropped from the database.

Note

If you get a message "Unable to Delete User" one or more tables within DIAMOND has a foreign key constraint on USER_ID and there is data in the table associated with the user ID you are trying to delete. Currently the only foreign key restraint enforced by Oracle, which could generate this message, is related to the assignment of customer service cases. If you get this message, see your System Administrator.
