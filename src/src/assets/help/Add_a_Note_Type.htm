Add a Note Type

Use the following procedure to add a note type. Note types are categories of notes you define.

To add a note type

  1. On the NOTET (Note Types) window File menu, click New.

  2. In the Note Type box, type the note type.

  3. In the Priority box, do one of the following to select a method for notifying users that a note exists:

  4. If you want to display "Notes Exist" in the title bar, type O.

  5. If you want to display "High Priority Notes Exist" in the title bar, type 1.

  6. If you want to display a dialog box that reads "There are notes of a high priority that must be read," type 2. The end-user must read each of these notes before returning to the screen.

  1. In the Description box, type the note description.

  2. On the File menu, click Save.
