About PRULE

Use keyword PRULE (Price Rule) to add, change or delete Price rules, which manage certain components of pricing logic. Price rules describe the methods used to calculate allowed amounts. Keyword PRULE is the first step in defining these sets of records.

For professional service claims, the following keywords are used to define pricing logic:

  • Keyword PRULE

  • Keyword PRULD (Price Rule Detail)

  • Keyword PRULS (Price Rule Detail Selection)

For institutional whole claims, the following keywords are used to define pricing logic:

  • Keyword PRULE

  • Keyword PRULD

  • Keyword PRULS

  • Keyword PRULW (Whole Claim Price Rule)

  • Keyword PDTER (Price Determinant Maintenance)

Price rules together serve the following functions:

  • Specify pricing methods and the circumstances under which each is used

  • Calculate allowed amounts and assign Reason codes

  • Hold a claim line and assign a Hold Reason code

  • Send a message to the claims processor

  • Add units to base procedure units

  • Override the allowed factor

  • Prompt the claims processor to enter an allowed factor

  • Apply provider-specific "% of allowed"
