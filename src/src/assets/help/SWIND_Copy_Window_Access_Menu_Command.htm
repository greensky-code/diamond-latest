SWIND Copy Window Access Menu Command Field Definitions

Field Name

Req

Description

Source

Comments

Department

D

User’s department.

System generated

Value displayed from corresponding field value in SUSER.

Location

D

User’s location.

System generated

Value displayed from corresponding field value in SUSER.

Max Open

D

Number of windows the user can have open simultaneously.

Manually entered

Regardless of the entry here, there is an overall limit of 20 windows open at the same time across the DIAMOND program.

P-Del

R

Indicates if the user has security access to delete records in the Window ID.

Selected from check box

Valid values:

  • Selected: User can delete records in the Window ID.

  • Cleared: User cannot delete records in the Window ID.

P-Ins

R

Indicates if the user has security access to add records in the Window ID.

Selected from check box

Valid values:

  • Selected: User can add records in the Window ID.

  • Cleared: User cannot add records in the Window ID.

P-Sel

R

Indicates if the user has security access to retrieve records in the Window ID.

Selected from check box

Valid values:

  • Selected: User can retrieve records in the Window ID.

  • Cleared: User cannot retrieve records in the Window ID.

P-Upd

R

Indicates if the user has security access to change records in the Window ID.

Selected from check box

Valid values:

  • Checked: User can change records in the Window ID.

  • Unchecked: User cannot change records in the Window ID.

User ID

R

Unique identifier of a DIAMOND user.

Manually entered

User IDs are maintained in keyword SUSER.

F5 lookup is available.

User Type

D

User’s user type.

System generated

Value displayed from corresponding field value in keyword SUSER.

Win ID

R

DIAMOND keyword and associated sheet or sheets for which this user’s access privileges are described.

Selected from predefined list

For a list of keywords and the sheets with which they are associated, see Security and Users, How DIAMOND Thinks.

The entries describing access must be the same for the keywords and their associated sheets.
