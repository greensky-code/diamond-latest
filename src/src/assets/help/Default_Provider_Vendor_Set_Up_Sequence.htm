Default Provider/Vendor Set Up Sequence

To set up and use a default vendor and provider, do the following tasks in the following order:

1. Set up the Default Vendor

2. Set up the Default Provider

3. Set up the non-specific provider

4. Turn on Default Provider Logic

5. Pay a claim using Default Provider logic

6. Verify Default Provider logic claim pricing/payment
