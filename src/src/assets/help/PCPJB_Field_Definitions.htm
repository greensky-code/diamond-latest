PCPJB Field Definitions

Field Name

Req

Description

Source

Comments

Action

R

The mode in which DIAMOND runs the job.

Selected from predefined list

Valid value:

Run: DIAMOND performs PCP Auto Assignment logic and updates the associated records.

Comments

O

Free form comments about the PCPJB batch job

Manually entered

 

(Request) Date

D

Current date

Display

 

Insert Date From

R

Earliest member eligibility record insert date to include in the job

Manually entered

Can be changed to a date earlier than the current date.

Insert Date Thru

O

Latest member eligibility record insert date to include in the job

Manually entered

 

Insert Process

R

Member eligibility record Insert Processes to include in the job

Manually entered

Valid values are:

1 - Batch

2 - Batch and Manual/Online

Job ID

R

Unique PCPAA job identifier

System generated

Can be edited.

Line of Business

D

Indicates if all or some lines of business are included in the job

Display

Valid values are:

All

Some

Use the Special menu command PCP Job Setup Selection to define the lines of business to include or exclude. If you do not make an entry, DIAMOND includes all lines of business.

Request Type

R

Processing request type.

Selected from predefined list

Valid values are:

Immediate

Deferred

Accept Immediate as default. Deferred is not supported at this time.

Request User

D

User ID of the current user

Display

 

Status

D

Status of the PCPJB job

Displayed

Valid Values:

New: You created a new PCPJB job record by selecting option from the File Menu, entering data, and saving the job.

Successful: In the Next Action section box, you selected the Action option button Run, followed by Run Job from the File Menu. DIAMOND ran the job in accord with your specifications and the job completed successfully.

Error: In the Next Action section box, you selected the Action option button Run, followed by Run Job from the File Menu. DIAMOND ran the job in accord with your specifications and the job completed successfully but with critical errors.

In Process: In the Next Action section box, you selected the Action option button Run, followed by Run Job from the File Menu, and the job is still running.

Cancelled: Before the job could complete, in the Next Action section box you selected the Action option button Cancel, followed by Run Job from the File Menu.
