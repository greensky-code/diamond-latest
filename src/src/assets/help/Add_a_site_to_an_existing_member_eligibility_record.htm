Add a Site to an existing member eligibility record

To add a Site code to an existing member eligibility record where the Site field is blank, use keyword MELIG (Member Eligibility History).

This is typically only necessary for member eligibility records you enter into DIAMOND using processes that do not include Site in the input transaction. This can result in a blank value in the Site field for records with a line of business that requires Site. Such processes include Member Batch processing, the Member API, and the DDS (DIAMOND Data Services) Member API.

Do not use keyword MELIG to change the Site on an existing member eligibility record. Use keyword MEMBR (Member Master) instead. Â  For more information, see Change the PCP and Site on a member eligibility period.

To Add a Site to an existing member eligibility record

  1. Press F3, type MELIG, and then press ENTER.

  2. In the Subscriber ID box, type the Subscriber ID. F5 lookup is available.

  3. In Site box, type the Site code. F5 lookup is available.

  4. Press CTRL+S to save.

Note

In keyword MELIG, when you type a Site code in the Site box or use the F5 lookup, DIAMOND logic does not include any validation using provider enrollment rules.
