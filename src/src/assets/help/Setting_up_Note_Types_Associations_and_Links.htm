Setting up Note Types, Associations, and Links

1. Add note types in keyword NOTET.

Note types are categories of notes such as ELG for member eligibility notes or ATH for authorizations notes.

Learn how to add a note type.

2. Set up window and note type associations in keyword NOTEW.

This step is required for each note type that you create. Any note type that is not associated with at least one window is not usable.

This step is also required for each notes-capable window for which you want to activate notes functionality. In order for a notes-capable window to utilize notes functionality, it must be associated with at least one note type.

3. Set up window and note type links in keyword NOTEW.

This step is not required; however, you must set up links if you want to create a note in one window and access that same note in a different window.
