Verifying Default Provider Logic Claim Pricing/Payment

Restrictions

None.

Instructions

Step

Comment

1. Access the paid claim using a PSCLM window.

 

2. On the Window menu, click Processing Messages and verify the pricing method DIAMOND used (based on the default provider).

 

3. From the claim detail window, on the Special menu, click Payment Detail and write down the A/P Trans. No. (accounts payable transaction number).

 

4. Open a new ACPAY window.

 

5. Enter AP Transaction ID = the A/P Trans. No. from step 3.

 

6. Verify that the Vendor ID, Vendor Name, and Address Line 1 = the provider who submitted the claim.
