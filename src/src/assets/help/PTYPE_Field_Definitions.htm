AutoAudit

If you use AutoAudit claims review software, enter the AutoAudit 2-character specialty or type code.

Manually entered.

See Auto Audit Rules (AARUL) user documentation for links to information on use of this code.

O

Text description of the type or specialty.

Manually entered.

Text description of the type or specialty that is displayed on some DIAMOND screens.

Manually entered.

Up to 15 characters are available.

The code for this type or specialty.

Manually entered.

Up to 6 alphanumeric characters are available.

An indicator of whether the record pertains to a type or a specialty.

Manually entered.
