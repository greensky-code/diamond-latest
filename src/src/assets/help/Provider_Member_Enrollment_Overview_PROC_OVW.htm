Provider/Member Enrollment Overview

You may want DIAMOND to automatically verify all or some of the following when you assign a Primary Care Provider (PCP) to a member during online membership enrollment (MEMBR) or during Member Batch processing:

  • The provider accepts new patients.

  • The provider accepts male, female or both patient genders.

  • The member’s age falls within the patient age range accepted by the provider.

  • The provider’s enrollment limit will not be exceeded as a result of the member assignment.

DIAMOND automatically verifies these conditions of provider/member assignment when you activate provider/member enrollment functionality. In addition, DIAMOND also:

  • Updates and displays the provider’s current member enrollment count.

  • Provides a way for you to override the pre-defined member assignment conditions.

We strongly urge you to run the Provider/Member Enrollment Utility to turn on this functionality. When you run this utility, DIAMOND automatically creates a "generic" provider enrollment rule for each valid PCP provider based on user selected criteria.

See PROVE reference documentation for links to provider enrollment Task and How DIAMOND Thinks documentation for additional information regarding this functionality.
