CFDST Field Definitions

Field Name

Req

Description

Source

Comments

Action

R

The mode in which the job is to be run.

Selected from a predefined list.

Values:

Run

Post

Cancel

Run: An "edit" mode whereby DIAMOND performs all the CFDST calculation logic and lets you review the results of that logic WITHOUT actually updating the associated claim detail lines with new Net Amount and Cap Fund Withhold Amount values.

Post: Posts the results of the Run/edit mode job. Updates all included claim detail lines with the new Net Amount and Cap Fund Withhold Amount values. Puts the claim lines in a "posted" status for the Accounts Payable Update (APUPD) process.

Cancel: Completely undoes all Run mode actions and returns all work files to the status they had before the run started. You may cancel a CFDST Run job any time prior to starting the job in Post mode. Posted jobs cannot be cancelled.

Allocation

R

The Capitation/Special Fund amount available for allocation for the Capitation Fund and Month specified.

Manually entered

You can type up to 16 digits before the decimal and 2 digits after the decimal. Do not type commas. The allocation amount cannot be a negative amount.

If zero (default value), DIAMOND will use the Min Allocation for this model to calculate the net amounts

Cap Month

R

The month for which you are running Cap/Special Fund Distribution

Manually entered

Format: MM/YYYY.

From Date

O

The earliest date of service allowable on claim detail lines selected for this batch job

Manually entered

Must be before or equal to the first day of the Cap Month for the job.

If no date entered, DIAMOND uses the first day of the CFDST Cap Month when selecting claim detail lines for the batch job.

Status

R

Status of the CFDST job.

System generated

Valid Values:

New: You created a new CFDST job record by selecting New from the File menu, New (or New From Template), entering data, and saving the job.

Successful Run: You selected Action > Run, followed by File menu > Run Job. DIAMOND ran the job in edit mode in accord with your specifications and the job completed successfully.

Cancelled: You selected Action > Cancel for a job which has not been posted, followed by File menu > Run Job. The job completed successfully and the job was cancelled.

Posted: You selected Action > Post for the job, followed by File menu > Run Job. The job completed successfully.
