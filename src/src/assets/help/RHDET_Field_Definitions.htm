RHDET Field Definitions

Field Name

Req

Description

Source

Comments

Determinant

R

Determinant column selected from the Determinant Table indicated

Selected from predefined list

Valid values include all columns in the selected Determinant Table except date columns. Date columns only available for GROUP_MASTER table.

Determinant Table

R

DIAMOND table from which to select individual claim hold release determinants

Selected from predefined list

Valid values include:

GROUP_MASTER

DENTAL_CLAIM_DETAIL (Dental services rules only)

DENTAL_CLAIM_HEADER (Dental services rules only)

INST_CLAIM_DETAIL (Institutional rules only)

INST_CLAIM_HEADER (Institutional rules only)

PROVSVC_CLAIM_DETAIL (Professional services rules only)

PROVSVC_CLAIM_HEADER (Professional services rules only)

From

R

From value of the range of determinant values

Manually entered

Must be a valid value for the determinant selected.

Operator

R

Operator to be applied to the selected determinant

Selected from predefined list

Valid values include:

Equal

Not Equal

Release Rule

R

Release Rule identifier

Selected from predefined list

Release rules are entered in RHRUL, Claim Hold Release Rule

Through

R

Through value of the range of determinant values

Manually entered

Must be a valid value for the determinant selected.
