Using keyboard shortcuts

General shortcut keys for working in DIAMOND 950

Press

To

F1

Display online Help.

F2

Display the Main Menu.

F3

Display the Shortcut Menu.

F4

Display the Notes dialog box (where available; e.g. MEMBR).

F5

Display the Lookup dialog box.

F7

Display the previous window view (where available; e.g., AUTHS).

F8

Display the next window view (where available; e.g., AUTHS).

ALT+F4

Exit the DIAMOND program.

ALT+UP ARROW

or

ALT+DOWN ARROW

Open a drop-down data window or list box.

CTRL+F4

Close the window.

SHIFT+ALT+C

Cascade open windows.

SHIFT+ALT+I

Arrange minimized application icons.

SHIFT+ALT+L

Layer open windows.

SHIFT+ALT+T

Tile open windows.

SHIFT+TAB

Move to the previous window field.

TAB

Move to the next window field.

UP ARROW

DOWN ARROW

Navigate when within an opened drop-down data window or list box or toggles the selection choices available when the cursor is on an unopened drop-down data window or list box field.

Main Menu shortcut keys

Press

To

ALT+C

Close the Main Menu.

DOWN ARROW

Scroll down through the keywords in the Keyword list.

ENTER

Display the keyword selected in the Keyword list.

LEFT ARROW

Move left through the icons in the Functional Groups box.

RIGHT ARROW

Move right through the icons in the Functional Groups box.

SHIFT+TAB

Move right through the icons in the Functional Groups box.

TAB

Move left through the icons in the Functional Groups box.

UP ARROW

Scroll up through the keywords in the Keyword list.

Picklist shortcut keys

Press

To

CTRL when clicking a column header

Sort records in descending order by the column header.

DOWN ARROW

Move down through the records in the picklist.

F7

Move up through the records in the picklist.

F8

Move down through the records in the picklist.

PAGE DOWN

Move down five records in the picklist.

PAGE UP

Move up five records in the picklist.

UP ARROW

Move up through the records in the picklist.

When a menu or command name displays with an underlined letter or letters, you can use a keyboard shortcut. For example, File Open means that you can press ALT+F, then O to access the Open command on the File menu. In each case, the ALT key is used with the first letter.
