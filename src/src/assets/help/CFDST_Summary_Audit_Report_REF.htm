Field Name

Description

Source

Comments

Adjust Net Payable

Total of the adjusted Net Amounts payable to the CFDST selected providers after running Cap/Special Fund Distribution

Calculated by CFDST logic

 

Allocation Amt

Amount to be allocated from the fixed cap/special fund for this CFDST job

Entered in Allocation field in CFDST

Total amount available for distribution for claims for all providers with contracts associated with the CFD Model ID specified in CFDST

Allocation Method

The field on claim detail lines used as the basis to allocate the Cap/Special Fund amount

Specified in CFMOD

May be Allowed Amount or Net Amount

Allocation Rate

Rate applied calculated to apply to the Allowed Amount or Net Amount on CFDST selected claim detail lines

Calculated by CFDST

See Capitation/Special Fund Distribution, How DIAMOND Thinks for more information on how the Allocation Rate is calculated and used

Allowed Amount

Total of Allowed Amounts Amount allowed for all the claim detail lines selected by the CFDST job

Calculated by CFDST logic

 

Cap Fund Withhold Amt

Total Cap Fund Withhold Amounts for the CFDST selected claim detail lines.

Calculated by CFDST logic

The Cap Fund Withhold Amount is calculated as the difference between the original claim detail line Net Amount and the adjusted claim detail line Net Amount after running Cap/Special Fund Distribution.

Cap Month

Month for which the CFDST job was run

Entered in CFDST

 

Cap Run From Date

Earliest date of service allowable on claim detail lines selected for this job

Entered in CFDST

If this date was not entered in CFDST, no date is displayed. By default, DIAMOND used the first day of the CFDST Cap Fund Month as the From Date.

Claim Type

Type of claims

Specified in CFMOD

 

Comments

Free form comments

Entered in CFDST

 

Job ID

Unique ID for this job

Entered in CFDST

 

Job Run Date

Date this CFDST job was run

System generated

 

Max Allocation

Maximum allocation percentage applied to limit the amount allocated to a provider

Specified in CFMOD

 

Min Allocation

Minimum percentage of the original claim detail line Net Amount that the provider must be paid

Specified in CFMOD

 

Model ID

ID of the model used in this CFDST run

Specified in CFDST

 

Net Amount

Total of the original Net Amounts prior to running Cap/Special Fund Distribution for the CFDST selected claim detail lines

Calculated by CFDST logic

 

Orig Net Amt

Total of Net Amounts on CFDST-the selected claim detail lines prior to running the CFDST job and calculating a new Net Amount

Calculated by CFDST logic

 

Status

Status of the CFDST job.

System generated

Valid values are:

Successful Run

Posted

Total Units

Total of the total unit value units (resulting from use of the Unit Value pricing calculation method) for the claim detail lines selected by the CFDST job

Calculated by CFDST logic

Total units will only be displayed when the claim detail line was priced using the Unit Value pricing calculation method

Withhold Amount

Total of the withhold amounts for the CFDST selected claim detail lines

Calculated by CFDST logic
