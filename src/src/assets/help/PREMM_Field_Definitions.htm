Matrix Definition

R

Unique identifier of the rate matrix.

Manually entered, or selected from predefined list

 

Matrix Description

R

Description of the rate matrix.

Manually entered

If a previously defined matrix is selected, the description is displayed.

This description will be displayed when F5 is pressed in GRUPD to select a matrix definition to associate with the group.

Matrix Calc Method

R

Indicator of whether percentages or dollar amounts will be used.

Selected from predefined list

Valid values are:

A Amount: Dollar amounts are included in the matrix and will be used to calculate the premium due.

P Percent of Salary: Percentages are included in the matrix and will be applied to the subscriber’s salary to calculate the premium due.

Changes to Matrix Calc Method potentially trigger group rate retroactivity, at your option, for all groups sharing this matrix definition. You can save the change with or without retroactivity. For more information, see Retroactive Rate Adjustments.

Matrix Determinant

R

Unique identifier of this matrix definition.

Manually entered, or selected from predefined list

 

Premium Rate Entry

R

Either the dollar amount, if Matrix Calc Method is D, or a percentage, if Matrix Calc Method is P.

Manually entered

If you are typing a dollar amount, then you can type up to 16 digits before the decimal and 2 digits after the decimal.

Changes to Premium Rate Entry potentially trigger group rate retroactivity, at your option, for all groups sharing this matrix definition. You can save the change with or without retroactivity. For more information, see Retroactive Rate Adjustments.
