LINBS Commands on the Menu Bar

 

Command             Menu   Description

APC Setup

Details

T

Use to set up Ambulatory Payment Classifications (APC), including using an editor, grouper, and pricer, for this line of business.

Audit Display

Details

W

Displays audit trail records for the selected record.

Auto Letter Setup

T

Use to display keyword ALSET (Auto Letter Setup) and create an Auto Letter event for this line of business.

Claims Discount Calculation Rules

T

Use to open keyword LBCDC (Line of Business Claims Discount Calculations). For more information, see LBCDC or Claims Discount and Interest Calculations, How DIAMOND Thinks.

Claims Interest & Penalty Calculation Rules

T

Use to open keyword LBCIC (Line of Business Claims Interest Calculations). For more information, see LBCIC or Claims Discount and Interest Calculations, How DIAMOND Thinks.

Delete Contract Search

S

Use to remove the data in the contract search fields.

Pre-Existing Condition Rules

T

Use to access keyword LBPEC (Line of Business Pre-Existing Conditions).

PCP Auto Assign

T

Use to define a set of PCP Auto Assign rules for the specified line of business.

PCP Support Info Details

T

Use to specify Specialty age range and gender restrictions for use in PCP Auto Assignment for the line of business.
