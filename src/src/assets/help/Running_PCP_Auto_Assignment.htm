Running PCP Auto Assignment

Restrictions

Instructions

Frequently Asked Questions

PCP Auto Assignment is a batch process by which you can:

  • Validate/Replace a submitted PCP assignment

  • Automatically assign a suitable PCP to a current member eligibility record

You use PCPJB as described below to run a PCP Auto Assignment batch job that applies the PCPAA rules and other criteria and logic to assign a PCP or validate a submitted PCP for selected member eligibility records

For more information on PCP Auto Assignment batch processing, see PCP Auto Assignment Processing Logic.

Restrictions

It is recommended that you ensure that all Member batch and Member API transactions have completed (that is, are no longer running) before running a PCP Auto Assignment batch job.

Instructions

Step

Comment

1. Ensure that all PCP Auto Assignment setup tasks are complete.

For more information, see Setting Up PCP Auto Assignment.

2. Use PCPJB to enter the PCP Auto Assignment batch job selection criteria:

  • Lines of business included

  • Date range of member eligibility record insert dates included

  • Member eligibility record insert processes (batch, online) included

To select the included lines of business, from the Special menu select Line Of Business. Then complete the fields as needed to include or exclude specific lines or business or ranges of lines of business.

3.  Run the PCPJB batch job. From the File menu, select Run.

You cannot cancel a PCPJB job.
