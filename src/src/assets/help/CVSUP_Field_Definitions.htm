CVSUP Field Definitions

Field Name

Req

Description

Source

Comments

Action

R

Action being requested for this job run.

Selected from predefined list

Valid values are:

  • Run: Runs accumulated clear, void, stop check/EFT transaction changes and prints the Clear, Void, Stop Update Audit Report.

  • Post: Posts accumulated transaction changes; removes payment/accounts payable data for status change "Reissued"; writes status, reason and date (that existed prior to posting Reissued status changes) to PSCLM/INCLM/DNCLM (Claims) Notes; creates offsetting payment/accounts payable data for "Reversed" changes.

  • Cancel: Cancels a job; must be prior to posting.

Comments

O

Text comments about the job.

Manually entered

 

Job ID

R

Unique identifier of the job.

System generated, can be manually overridden.

 

Req User

D

User submitting the job.

Displayed

 

Request Type

R

Indicator of when job is to be run.

Selected from predefined list

Only "Immediate" is available as a valid selection.

Run Date

D

System date the job is run.

Displayed

 

Status

D

Status of the selected job.

Displayed

Valid values are:

  • New: Job has been saved but not yet run.

  • Successful run: Job ran successfully.

  • Error: Job was unsuccessful.

  • Canceled: Job action was Cancel.
