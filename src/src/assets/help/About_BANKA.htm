About BANKA

Use keyword BANKA (Bank Account) to define each bank account and the following for processing checks:

  • General ledger (G/L) cash account

  • ABA routing number for Electronic Fund Transfers (EFT)

  • Bank account type

  • Intelligent Queryä (IQ) template (print format) for checks,

Use keyword BANKA to define each bank account and the following for processing claims and encounters:

  • Remittance Advices (RA)

  • Explanations of Benefits (EOB)

  • Explanations of Pricing (EOP)
