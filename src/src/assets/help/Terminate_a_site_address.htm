Terminate a site address

Use this procedure to terminate a site address.

When you terminate a site address, DIAMOND logic displays the same Term Date and Term Rsn values on all provider address records linked to that site address. These linked provider address records can then no longer be selected on provider enrollment rules, though they are not actually terminated.

To terminate a site address

  1. Press F3, type SITEA, and then press ENTER.

  2. In the Site Code box, type the Site code for which you want to terminate a site code address. F5 lookup is available.

  3. In the site address picklist, click the site address you want to terminate.

  4. In the Term Date box, type the date on which you want to terminate this site address.

  5. In the Term Rsn box, type a termination reason code representing the reason for terminating this site address. F5 lookup is available.

  6. Press CTRL+S to save.
