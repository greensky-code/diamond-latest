Claim Detail Indicators Menu Command

Field Name

Req

Description

Source

Comments

COB Indicator

O

Indicates if Coordination of Benefits applies to the submitted service.

Selected by check box.

 

Emergency Services Indicator

O

Indicates if the submitting service has Emergency Services involvement.

Selected by check box.

 

EPSDT

O

Indicates that the submitted service has Early and Periodic Screening, Diagnosis, and Treatment.

Selected by check box.

 

Family Planning Indicator

O

Indicates if the submitted service has Family Planning Services involvement.

Selected by check box.

 

Init. Treat. Date (Spinal Manip)

O

Initial treatment date for spinal manipulation services.

Manually entered.

Date must be in the mm/dd/yyyy date format.

Latest Creatine Test

O

Most recent serum creatine date associated with the claim detail line.

Manually entered.

Date must be in the mm/dd/yyyy date format.

Latest Hemo Test

O

Latest hemoglobin or hematocrit test date associated with the claim detail line.

Manually entered.

Date must be in the mm/dd/yyyy date format.

Latest Visit (PT)

O

Latest physical therapy visit date associated to the claim detail line.

Manually entered.

Date must be in the mm/dd/yyyy date format.

Mammography Cert. Number

O

Member’s Mammography Certification number.

Manually entered.

 

Proc Code

O

Procedure code for the DME item that is associated with the claim detail line.

Manually entered.

Must be a valid procedure code.

Prov. Line Item Control Number

O

Unique identifier number for the claim detail line.

Manually entered.

 

Purchase Price

O

Purchase Price associated with the DME item that is specified

Manually entered.

Enter a number up to 16 characters before the decimal and 2 characters after the decimal.

Purchase Service Amount

O

Indicates the amount of the item that was purchased.

Manually entered.

Enter a number up to 16 characters before the decimal and 2 characters after the decimal.

Purchase Service Prov

O

Indicates the name or ID number associated with the provider that the services were purchased from.

Manually entered.

Enter up to 30 characters. No validation is performed on this field.

Quantity

O

Number of services for the DME item that is associated with the claim detail line.

Manually entered.

 

Rental Frequency

O

Rental frequency by which the service will be billed that is associated with the DME item specified.

Selected from a pre-defined list.

Valid values are:

1 – weekly

4 – Monthly

6 – Daily

Rental Price

O

Rental Price associated with the DME item that is specified.

Manually entered.

Enter a number up to 16 characters before the decimal and 2 characters after the decimal.

Reserved for Local Use

O

Text field for notes.

Manually entered.
