Display A/P Transaction Information for a Vendor

Use the following procedure to display A/P transaction information for a vendor.

To display A/P transaction information for a vendor

  1. Press F3, type APVDS, and then press ENTER. The APVDS Claim Filter Data dialog box displays.

  2. In the Vendor ID box, type a vendor, and then press TAB. F5 lookup is available.

  3. In the Claim File Type list, click the claim type.

  4. If desired, fill in the remaining fields.

  5. Click AP Trans Detail. The Lookup dialog box displays a list of claims.

  6. Double-click a claim to display A/P transaction information.
