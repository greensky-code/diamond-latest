PROVA Field Definitions

Field Name

Req

Description

Source

Comments

AL Dist.Meth.

 

Displays the preferred method of Auto Letter distribution.

Selected from a predefined list.

Select the contact’s preferred method of receiving Auto Letter correspondence: printed, faxed, or sent in e-mail.

For more information about Auto Letters, see Auto Letters, How DIAMOND Thinks.

Addr 1-2

O

Two lines of the provider’s street address.

Manually entered.

 

Cert

D

Cert check box   value from the selected site address record.

Displayed

Unavailable if parameter MEMSITEVAL Parameter 1 = N.

City

O

Provider’s city.

System generated from ZIP Code or manually entered.

Depending on the values in keywords ZIPRF, UDATT, and the DLFLTZIP parameter in keyword PARAM, this field is automatically filled based on the ZIP Code you enter.

Contact Name

O

Name of office contact person.

Manually entered.

 

Country

O

Provider’s country.

Selected from a predefined list.

May or may not be used depending on implementation setup.

County

O

The provider’s county code.

System generated from ZIP Code or manually entered.

Depending on the values in keywords ZIPRF, UDATT, and the DLFLTZIP parameter in keyword PARAM, this field automatically is filled based on the ZIP Code you entered.

Email ID

 

The contact’s e-mail address.

Manually entered.

If in the AL Dist.Meth. field you selected E-mail, then this e-mail address is used to send Auto Letter correspondence. If E-mail is not selected in the AL Dist.Meth. field, then this field is for informational purposes.

For more information about Auto Letters, see Auto Letters, How DIAMOND Thinks.

Ext#

O

Extension for the contact person.

Manually entered.

 

Fax Number

O

The contact’s fax telephone number.

Manually entered.

If in the AL Dist.Meth. field you selected Fax, then this telephone number is used to fax Auto Letter correspondence. If Fax is not selected in the AL Dist.Meth. field, then this field is for informational purposes.

For more information about Auto Letters, see Auto Letters, How DIAMOND Thinks.

Med Grp

O

The name of the provider’s medical group associated with this address.

Manually entered.

Primary Address

Selected from a predefined list.

At least one address must be designated as primary. Use on ID Cards is controlled by IDPRT.
