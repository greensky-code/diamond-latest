Delete a Quality Program from a Provider Contract

Use the following procedure to delete a quality program from a provider contract.

Important Note   You cannot delete a quality program from a provider contract if the quality program has been used in calculating an amount to be included in the capitation payment.

To delete a quality program from a provider contract

  1. Press F3, type PROVC, and then press ENTER.

  2. In the Provider ID box, type the provider ID, and then press TAB. F5 lookup is available.

  3. In the picklist, click the provider contract.

  4. On the Special menu, click Quality Program Eligibility.

  5. For the quality program you want to delete, click in the Effective Date box.

  6. Click Delete.
