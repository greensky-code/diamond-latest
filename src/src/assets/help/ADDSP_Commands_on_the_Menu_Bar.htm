ADDSP Commands on the Menu Bar

 

Command             Menu   Description

Administrative Fee Display Selection

S

Use to select the administrative fee record you want to display in the Administrative Fee Display window.

Reverse Administrative Fee

S

Use to reverse or adjust an administrative fee record.

View Administrative Fee Rule

T

Use to display the administrative fee rule from keyword ADRUL (Administrative Fee Rule) that determined the selected administrative fee.

View Claim Display

T

Use to display the selected claim record in keyword CLDSP (Claims Display).

View Payment Detail

T

Use to display the associated A/P payment detail record from keyword ACPAY (Accounts Payable Transactions) for the selected administrative fee.
