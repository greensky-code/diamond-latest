Add a member privacy contact

Use this procedure to add a member privacy contact. You can use member privacy contacts in conjunction with member conditions functionality. For more information, see Member Conditions - Overview.

To add a member privacy contact

  1. Press F3, type ALIAS, and then press ENTER.

  2. In the Contact Information section box, fill in the fields as follows. For more information, see ALIAS Field Definitions.

  3. In the Contact Source box, select P - Privacy.

  4. In the Distr. Method box, select the preferred method of receiving auto letter correspondence for the privacy contact.

  5. In the Email Id box, type the email address of the privacy contact.

  6. In the Fax Number box, type the fax number of the privacy contact.

  7. Press CTRL+S to save.
