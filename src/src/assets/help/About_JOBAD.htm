About JOBAD

Use keyword JOBAD (Administrative Fee Batch Job) to calculate, post, or cancel an administrative fee job.

Calculating Administrative Fees

When you run an administrative fee job to calculate fees, the provider administrative fee rules in keyword ADRUL (Provider Administrator Fee Rule) evaluate the claim detail lines. This process identifies the services eligible for an administrative fee, as well as the administrative fee.

Claim detail lines with the following characteristics in the claim detail window are evaluated:

  • The Claim Status list is set to C – Capitation, N – No Check, or P – Payable

  • The Proc Status list is set to C – Check, P – Paid, N – No Check, or F – Final (posted)

Claim detail lines with the following characteristics are ignored:

  • In the claim detail window, the Audit Status box is set to E or Y.

  • Adjusted claim detail lines

  • Reversed claim detail lines

Once a claim detail line is evaluated for an administrative fee, then it is not available to be evaluated again.

To learn how to calculate an administrative fee job, see Running an Administrative Fee Job.

Posting Administrative Fees

After running an administrative fee job to calculate fees, you can post the job to finalize the calculations. After posting job, you can run an A/P update job in keyword APUPD (Accounts Payable Update) to select the administrative fee job for payment.

To learn how to post an administrative fee job, see Posting an Administrative Fee Job.

Canceling Administrative Fees

If you calculate an administrative fee job and find a problem, then you can cancel the job and make your changes. Later, you can calculate a new administrative fee job.

To learn how to cancel an administrative fee job, see Canceling an Administrative Fee Job.
