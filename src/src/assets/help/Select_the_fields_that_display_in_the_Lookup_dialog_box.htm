Select the fields that display in the Lookup dialog box

Use the following procedure to select the fields that display in the Lookup dialog box.

To select the fields that display in the Lookup dialog box

  1. Display the Lookup dialog box.

  2. Click Columns.

  3. In the Columns list, click the fields you want to display as columns in the Lookup dialog box.

  4. Click Custom.

Tips

  • These settings stay in effect until you click Default on the Lookup dialog box.

  • To clear a field in the Columns list, click the field again.

  • Primary key fields automatically display in the Lookup dialog box and cannot be cleared.
