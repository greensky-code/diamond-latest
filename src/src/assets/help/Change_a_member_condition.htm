Change a member condition

Use this procedure to change a member condition record.

To change a member condition

  1. Press F3, type MCOND, and then press ENTER.

  2. In the DIAMOND ID box, type the DIAMOND ID of the member for whom you want to change a member condition record.

  3. In the Subscriber ID box, type the Subscriber ID of the member for whom you want to change a member condition record.

  4. In the person number picklist, click the person number of the member for whom you want to change a member condition record.

  5. In the member condition picklist, click the member condition record you want to change.

  6. Change the fields in the window as needed. For more information, see MCOND Field Definitions.

  7. Press CTRL+S to save.
