Priority

R

Default priority level for all notes of this type.

Selected from predefined list

Valid values are:

0 - low

1 - medium

2 - urgent

Each of these values has an associated notes notification action. For a description of the note notification actions, see the Notes Notification section of Notes, How DIAMOND Thinks user documentation.

O

Field labels for user-defined date fields.

Manually entered

If you do not supply a field label, the user defined date fields are not available on the Notes Detail screen.

User Labels 1-5

O

Field labels for user-defined text fields.

Manually entered
