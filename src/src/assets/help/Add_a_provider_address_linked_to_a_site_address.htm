Add a provider address linked to a site address

  1. Press F3, type PROVA, and then press ENTER.

  2. In the Provider ID box, type a Provider ID. F5 lookup is available.

  3. In the Site box, type a Site code. F5 lookup is available. Â  DIAMOND logic validates that this Site Code has one or more site address records that match a provider contract for the entered Provider ID and Site code. For more information, see Site Validation, How DIAMOND Thinks section Finding a matching provider contract and site address when adding a provider address.

  4. Select the site address from the resulting displayed list. DIAMOND logic displays the Site code in the Site box and copies the following information from the site address record: Certification, Site name, site address, office hours, contact information, and user-defined fields. If a site address list is not displayed, DIAMOND logic did not find any matches. You must then research why a matching contract was not found and remedy the problem before this provider address can be added.

  5. Fill in the remaining fields in the window. For more information, see PROVA Field Definitions.

  6. Press CTRL+S to save. Once you save a provider address record linked to a site address, you cannot change the provider address record using keyword PROVA. You must use keyword SITEA. For more information, see Change a provider address linked to a site address.
