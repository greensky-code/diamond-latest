Post Checks

Use the following procedure to post checks printed by a check print job.

The checks you want to post must have been printed by a check print job in keyword CKPRT (Check Print Setup).

For manual checks, use keyword MCHEK (Manual Checks).

To post checks

  1. Press F3, type CKPST, and then press ENTER.

  2. In the picklist, click the check print job ID for which you want to post checks. The checks printed by the job display. EFT transactions do not display.

  3. For each check you want to post, in the Status list, click Posted.

  4. On the File menu, click Run Job.
    The following occurs:

  5. The Processing Status list is set to P – Paid for the corresponding A/P transaction record in keyword ACPAY (Accounts Payable).

  6. The Proc Status list is set to P – Paid for the corresponding claim detail line in keyword DNCLM, INCLM, or PSCLM.

Tip

View paid checks in keyword CHECK (Check Register).
