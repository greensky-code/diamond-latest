About SITEA

Use keyword SITEA to add site address records. On site address records you can specify such information as site name, primary and mailing addresses, office hours, and contact information. You can also specify whether the facility at this site address is certified to provide services for a special program, for example, the Child Health and Disability Prevention Program (CHDP).

You define site address records for a specific effective period. You cannot delete a site address record.

Keyword SITEA is only available if parameter MEMSITEVAL Parameter 1 = Y.
