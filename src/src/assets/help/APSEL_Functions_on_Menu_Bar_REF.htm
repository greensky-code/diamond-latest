APSEL Commands on the Menu Bar

 

Command             Menu   Description

Close

F

Close the Accounts Payable Selection window.

If there is an open or unsaved A/P selection job, you will be prompted to run it.

New

F

Create a new A/P selection job.

New from Template

F

Create a new A/P selection job using an existing template.

Run Job

F

Run the active A/P selection batch job.

Save as Template

F

Save the current A/P selection job as a template.

Once saved as a template, the job will display in the dialog box that displays when New from Template is selected from the File menu.
