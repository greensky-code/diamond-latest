BENEF Commands on the Menu Bar

 

Command             Menu   Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Benefit Package User Fields

S

A screen that contains 18 user-defined text fields and 18 user-defined date fields that are associated with the benefit package. These fields are informational only and have no functionality.

You can use keyword UDTXT, User Defined Text, to change the on-screen field names, to make any of the user-defined fields required, or to select any of the user-defined fields to be validated against predefined values.

Copy Benefit Package

S

Copies some or all of the benefit rules from one package to another.

Define Benefit Value Filters

Details

S

View, add, update, or delete Benefit Value Filters that are associated with the Benefit Rule. Benefit Value Filters consist of Par, Auth, and Rider codes.

Delete Benefit Package

F

Deletes a benefit rule from a benefit package. If a rule has been used to process a claim, DIAMOND will not permit it to be deleted from a package.
