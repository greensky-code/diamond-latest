National Drug Codes, How DIAMOND Thinks

NDC codes can be applied to unfinalized claim line items in keyword INCLM (Institutional Claims), DNCLM (Dental Services Claims), and PSCLM (Professional Services Claims). No pricing or adjudication is associated with NDC codes; adding an NDC code to a claim line item is informational, only. DIAMOND adds the NDC code if the claim line item’s date of service is between the code’s effective date and term date. If an NDC code was not set up with an effective date, then the code is considered effective on the date of service. The following applies to claim line items and NDC codes:

  • If a claim line item with an NDC code is deleted, then the NDC code is also deleted from the line item.

  • If a claim line item with an NDC code is reversed, then the NDC code is also reversed from the line item.

  • If a claim line item with an NDC code is adjusted, then the NDC code is not adjusted. You must manually add a new NDC code to the adjusted line item, or delete the NDC code form the original line item.
