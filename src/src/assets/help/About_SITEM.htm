About SITEM

Use keyword SITEM to add site code master records. Each site code master record is unique for a line of business/IPA/Panel combination.

Once you save a site code master record, you cannot make any changes except to the Description field and the user defined fields.

Keyword SITEM is only available if parameter MEMSITEVAL Parameter 1 = Y.
