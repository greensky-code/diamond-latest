PREMM Commands on the Menu Bar

 

Command             Menu   Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Copy

S

Copies rates from an existing PREMM record into a new one. In order to use Copy, the rates being copied must have the same Matrix Calc Method and Matrix Determinant as the new PREMM Matrix Definition. After rates have been copied to a new PREMM record, they may be updated. Once the new PREMM record has been saved, rates may not be copied to it from another record.
