Change the PCP and Site on a member eligibility period

To change the PCP and Site on the current member eligibility period, use keyword MEMBR (Member Master) Special Menu command Provider Change. You should not change PCP or Site using keyword MELIG (Member Eligibility History).

However, do use keyword MELIG to add a Site to an existing eligibility period where Site is required by the member's line of business and Site is blank. This situation can occur when you enter such member eligibility records using processes that do not include Site in the input transaction. These include Member Batch processing, the Member API, and the DDS (DIAMOND Data Services) Member API. For more information, see Add a Site to an existing member eligibility record.

To change the PCP and Site on a member eligibility period.

  1. Press F3, type MEMBR, and then press ENTER.

  2. In the Subscriber ID box, type the Subscriber ID for the desired member. F5 lookup is available.

  3. In the List of all Family Members window, click the desired member.

  4. On the Special menu, click Provider Change.

  5. In the Effective Date box in the New column, type the effective date of the new Site code.

  6. In the PCP ID box in the New column, press F5 for lookup. DIAMOND logic filters the records displayed in the Lookup window to valid selections for this member. For information on this filtering, see Site Validation, How DIAMOND Thinks section Finding PCPs Using Site Validation.

  7. Select the desired record. DIAMOND logic displays the values from the selected record in the PCP ID, Site, IPA, and Panel boxes.
    Note
    You can also type a PCP ID in the PCP ID field. However, if you do so, DIAMOND logic does not include Match All Contracts logic nor are the IPA and Panel fields automatically populated.
    You can also press F5 to lookup in the Site box before entering a PCP ID. If you do, the DIAMOND application performs the same filtering logic and upon selection displays the same values as when you use the F5 lookup in the PCP ID box. For more information, see Site Validation, How DIAMOND Thinks section Finding Sites Using Site Validation.

  8. In the Term Date box in the Current column, type the termination date of the prior information.

  9. In the Term Reason box in the Current column, type the termination reason code reflecting the reason for this change. F5 lookup is available.

  10. Press CTRL+S to save.
