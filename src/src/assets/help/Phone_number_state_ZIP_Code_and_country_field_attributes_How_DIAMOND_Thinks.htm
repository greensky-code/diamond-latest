Phone number fields

DIAMOND uses U.S. Standard Attributes to edit and display the data you enter into any phone number field, verifying that you have entered one of the following:

  • Nothing, if the field is not required

  • A seven-digit telephone number, such as 555-1212

  • A ten-digit telephone number, such as (415) 555-1212

When you type a telephone number in a phone number field, a "display mask" automatically formats the phone number to display according to the field attributes in keyword UDATT. Any characters added by the display mask are not stored in the database. For example:

Changing the attributes of a phone number field

Use keyword UDATT to specify the display of telephone numbers with formats that differ from the U.S. Standard, and, if your telephone number format is consistent, then you can include mask display characters.

For example, you want to display all entered phone numbers as:

  • A 4 digit area code, followed by

  • A space, a dash, a space, followed by

  • A 2 character alphanumeric prefix, followed by

  • A colon, followed by

  • A 3 digit number

If your phone number format is not consistent, one set of mask display characters cannot cover all entry/display possibilities, for example, area codes which vary in length from 3 to 7 alphanumeric characters, phone numbers which vary in length from 4 to 7 alphanumeric characters, and a dash separating the area code from the phone number.

In this case, Diamond stores the dash as part of the data entered in the field because the dash was entered on the keyword function window.

Similarly, use keyword UDATT to specify the display of Postal Codes that differ from the U.S. Standard. In addition, you can include mask display characters if your ZIP/Postal Codes format is consistent.

Valid telephone number lengths

The internal Pre-loaded U.S. Standard Attribute for phone number field length is 0;7;10

  • 0, Entry of phone number is not required in the keyword window (that is, the phone number field can be left blank).

  • 7, Phone number may be entered without the area code (7 digits).

  • 10, Phone number may be entered with the area code (10 digits).

ZIP Code fields

DIAMOND uses U.S. Standard Attributes to edit and display the data you enter into any ZIP Code field, verifying that you have entered one of the following:

  • Nothing, if the field is not required

  • A five-digit ZIP Code, such as 94101

  • A nine-digit ZIP Code (ZIP+4), such as 94101-4444
