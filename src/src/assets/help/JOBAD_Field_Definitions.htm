JOBAD Field Definitions

Field Name

Req

Description

Source

Comments

Action

R

Click the action you want the job to take.

 

The following buttons are available:

  • Calculate – Runs the job.

  • Post – Posts the job.

  • Cancel – Cancels the job.

Claim From Entered Date

O

If you want the job to evaluate only claims entered from a particular date, then type the date.

Manually entered

 

Claim Thru Entered Date

O

If you want the job to evaluate only claims entered through a particular date, then type the date.

Manually entered

 

Comments

O

Type the description of the job.

Manually entered

 

Company Code

O

If you want the job to evaluate only claims in a particular G/L company, then type the company code.

Manually entered

F5 lookup is available.

Job ID

R

Displays the administrative fee job ID.

System generated

 

LOB

R

Displays Some or All, depending on the selection made when the Line of Bus Selection command was selected from the Special menu.

System generated

The following values can display:

  • Some – Indicates one or more lines of business were included or excluded in the job.

  • All – Indicates all lines of business are included in the job.

Request Date

D

Displays the date the user requested the job.

System generated

 

Request Type

Y

Click when to run the job.

 

The following buttons are available:

  • Immediate – Run the job now.

  • Deferred – Run the job later. Future functionality.

Request User

D

Displays the user ID of the user that requested the job.

System generated

 

Status

D

Displays the status of the job.

Displayed

The following status can display:

  • New – Displays during entry of job information, until Run Job is selected from the File menu.

  • In Process – Displays while the job is running.

  • Successful Run – Displays after a job has run successfully and before posting.

  • Posted – Displays after a job has posted successfully.

  • Cancelled – Displays if the job was cancelled.

  • Error – Displays if the job did not complete successfully.

Vendor

O

If you want the job to evaluate only claims for a particular vendor, then type the vendor ID.

Manually entered

F5 lookup is available.
