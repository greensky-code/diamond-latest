Address 1

R

If customer is a group, corresponds to Address 1 on the Group record; if an individual, Address 1 on the Member record.

System generated, or manually entered

DIAMOND fills this field if the record was automatically generated through premium billing.

If the record is created through manual entry, this field must be manually entered and must correspond exactly to the Group or Subscriber fields, as appropriate.

Customer Name 1

R

If customer is a group, corresponds to Name 1 on the Group record; if an individual, corresponds to subscriber’s First Name, Middle Initial, and Last name, no spaces.

Short Name

R

If customer is a group, corresponds to the group short name; if an individual, the subscriber’s last name.

System generated, or manually entered

DIAMOND will create the short name if the record was created automatically as a result of premium billing.

If the record is entered manually, this field must be entered manually and must correspond exactly to the Group short name or Subscriber last name.
