Turning on Default Provider Logic

Restrictions

Instructions

Restrictions

None

Instructions

Step

Comment

1. Open a new PARAM window.

  1. Set the DFTCNTRCTPRV parameter 1 = DEFAULTP and parameter 2 = the default provider’s Seq Prov ID

You are telling DIAMOND where to find the default provider and to use this provider when DIAMOND invokes Default Provider Logic.

2. Quit and restart DIAMOND to re-set the DFTCNTRCTPRV parameter.
