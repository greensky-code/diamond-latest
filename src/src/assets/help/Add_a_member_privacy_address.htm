Add a member privacy address

Use this procedure to add a member privacy address. You can use member privacy addresses for third party ad hoc reporting, ad hoc correspondence, and EOBs.

Also, under certain conditions, DIAMOND logic uses member privacy addresses as the mailing address for claims payments for Pay to Subscriber and Pay to Dependent claims. For more information, see Member Conditions, How DIAMOND Thinks section Using the Privacy Address for Pay to Subscriber and Pay to Dependent Claims.

To add a member privacy address

  1. Press F3, type ALIAS, and then press ENTER.

  2. In the Privacy Information section box, fill in the fields as follows. For more information, see ALIAS Field Definitions.

  3. In the Last Name box, type the last name of the privacy entity.

  4. In the First box, type the first name of the privacy entity.

  5. In the Addr 1 box, type the first line of the privacy entity's address.

  6. In the Addr 2 box, type the second line of the privacy entity's address.

  7. In the City box, type the city of the privacy entity's address.

  8. In the State box, type the state of the privacy entity's address.

  9. In the Zip Code box, type the zip code of the privacy entity's address.

  10. In the Phone Number box, type the phone number of the privacy entity's address.

  11. In the Country box, type the country of the privacy entity's address.

  12. Press CTRL+S to save.
