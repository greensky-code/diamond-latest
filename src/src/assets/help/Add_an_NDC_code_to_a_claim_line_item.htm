Add an NDC code to a claim line item

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to add an NDC code to an unfinalized claim line item. No pricing or adjudication is associated with NDC codes; adding an NDC code to a claim line item is informational, only.

DIAMOND adds the NDC code to the claim line item if the date of service is between the code’s effective date and term date. If an NDC code was not set up with an effective date, then the code is added to the claim and it is considered effective on the date of service.

Restrictions

This feature is only available in keywords INCLM (Institutional Claims), PSCLM (Professional Services Claims), and DNCLM (Dental Services Claims).

Instructions

Step

Comment

1. On the Claims Detail screen, save your detail line item (on the File menu, click Save).

 

2. On the Special menu, click NDC Codes.

The Attach NDC Codes dialog box displays.

3. Click Add.

 

4. In the NDC Code box, type the 11-digit NDC code (without dashes).

F5 Lookup available.

5. In the Qty box, type the quantity.

 

6. Click OK.

The code is added to the claim line item.

None.
