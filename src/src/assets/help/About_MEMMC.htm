About MEMMC

Use keyword MEMMC (M+C History) to add or change member Medicare + Choice History information. Keyword MEMMC is available only if for the member's line of business, keyword LINBS (Line of Business) M+C Plan Ind check box is selected.

Medicare + Choice history data included on MEMMC includes entitlement information, demographic information on the plan benefit package (PBP), and institutional data.
