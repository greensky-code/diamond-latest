Other Claim Info Menu Command

Field Name

Req

Description

Source

Comments

Accident

O

Indicates if the claim is due to an accident.

Selected from predefined list

Hard coded entries are:

Auto accident

Other, non-auto

No accident

Account Number

O

Patient’s account, medical record, or control number.

Manually entered

You can type up to 38 alphanumeric characters.

Assign Benefits

O

Indicates patient has authorized payment to physician (or other entity represented by Vendor on claim).

Selected from check box

Click to select to indicate benefits were assigned.

Condition Applies

O

Indicates if EPSDT condition codes apply to the claim.

Selected from a pre-defined list.

Valid values are:

Y – Yes

N – No

Blank

Condition Code

O

HIPAA Condition Code.

Selected from a pre-defined list.

Valid values are:

A – Acute

C – Chronic

D – Non-acute

E – Non-Life Threatening

F – Routine

G – Symptomatic

M – Acute Manifestation of a Chronic Condition

Blank (no description needed)

Condition Code 1-3

C

HIPAA Condition Code.

Selected from a pre-defined list.

Condition Code 1 is available only if Condition Applies is set to Y – Yes.

Condition Code 2 is available only if Condition Code 1 has a valid value.

Condition Code 3 is available only if Condition Code 1 or 2 has a valid value.

Valid values are:

NU – Not used

S2 – Under Treatment

ST – New Services Requested

Blank (no description needed)

Condition Desc 1-2

O

Description of the spinal manipulation procedure.

Manually entered.

Enter up to 80 characters. No validation is performed on this field.

Date of Death

O

Date of death for the patient

Manually entered.

Date must be in the mm/dd/yyyy date format.

Dx 5-8

O

Member’s diagnosis codes associated with this claim.

Manually entered.

You can type up to 30 alphanumeric characters. Unless the insertion point is in this box, then only 7 characters display.

The primary diagnosis code must go in Dx 1. Entered codes are validated against the member’s gender/age and a warning is issued if the code is flagged for case management. DIAMOND displays an error message if codes do not pass validation.

Diagnosis codes are maintained in keyword DIAGN (Diagnosis Code).

Employment

O

Indicates the claim is due to an employment-related accident.

Selected from predefined list

Hard coded entries are:

Yes

No

Unknown

Hospital Discharge Date

O

If current services are related to a hospital stay, the discharge date of that stay.

Manually entered

 

Initial Treatment Date

O

Patient’s initial spinal treatment date.

Manually entered.

Date must be in the mm/dd/yyyy date format.

Lab ID

C

Unique identifier of the outside Lab. Matches an institutional authorization to a professional services claim.

Manually entered

This entry is required if the outside lab services were rendered at a hospital or other valid place of service specified in the FACILITYIDRQ parameter.

Last Foot Care Visit

O

Date of the most recent foot exam.

Manually entered.

Date must be in the mm/dd/yyyy date format.

Mammography Cert Number

O

Member’s Mammography Certification number.

Manually entered.

The Medicaid specific resubmit code

Manually entered

You can type up to 5 characters.

First date of the patient’s current illness, accident, or injury.

Manually entered

 

Onset Time

Manually entered

 

Original Reference Number

O

If applicable, the original reference number for the claim

Hard coded values are:

Outside Lab

O

Indicates the claim includes charges for lab work done by a lab not affiliated with the billing provider.

Selected from check box

Click to select to indicate an outside lab was used.

Outside Lab Chrgs

O

Manually entered

O

Patient signature source code

Selected from   predefined list
