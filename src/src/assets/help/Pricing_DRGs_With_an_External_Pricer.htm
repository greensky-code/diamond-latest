Pricing DRGs With an External Pricer

Restrictions

Instructions

Frequently Asked Questions

You can use external pricing software to price institutional claims at the whole claim level using Diagnosis-Related Groups (DRGs). For external pricing, DIAMOND uses the HSS EASYGroupÔ 2000 Optimizer.

Restrictions

To use an HSS external pricer, you must first have a DRG code. You can accept the submitted DRG on the claim, enter a valid DRG, or call an external DRG grouper to determine the DRG. For more information on using an external grouper, see Grouping DRGs with an External Grouper.

Instructions

Step

Comment

1.  Before setting up DRG contracts, analyze precisely how the contracts work and what they mean.

 

2. In keyword PDTER (Price Determinant Maintenance), add price determinants that define the characteristics of a claim to be priced using DRGs.

Keyword PRULW (Whole Claim Price Rule) records are selected using price determinants defined in keyword PDTER.

3. In keyword PRULE (Price Rule), add a price rule. Be sure to select the Use External DRG Pricer check box.

In most cases, there will be a separate price rule record for each institutional provider; in other words, it is rare that more than one hospital will share the same contractual requirements for claims pricing.

4. If there are institutional services to be priced at the procedure or revenue code level and not included in the DRG rate, add records in keyword PRULD (Price Rule Detail) to define those ‘carved out’ procedures or revenue codes.

 

5. To define the specific procedure codes and/or revenue codes to which records in keyword PRULD apply, add PRULS (Price Rule Detail Selection) records as needed.

DIAMOND reads keyword PRULD records in level+sequence order. If there is more than one keyword PRULD record, there may be no need to add a keyword PRULS record for the last keyword PRULD record read. By the point that last record is evaluated, all specific exceptions will have been defined as part of the ‘earlier’ keyword PRULD records. In other words, the last rule read is the fall through, or all else, rule.

6. Add a PRULW record with a Calc Method of 04 – DRG. This calculates the base allowed amount.

No Base Rule Stop Logic applies to DRG rules.

7.  In keyword PROVC, add contract records as necessary for institutional providers. Include using the DRG/APC Information command on the Special menu to enter the required DRG data, including:

  • Facility Number

  • Paysource

 

Frequently Asked Questions

  • Where can I get background information about DRG grouper and pricer functionality and how the HSS Optimizer works?

See DRG Grouping and Pricing, How DIAMOND Thinks.

  • How do I price DRGs without an external pricer?

If it suits your business requirements, you can have DIAMOND price DRGs without using an external pricer. You can accept the submitted DRG on the claim, enter a valid DRG, or call an external DRG grouper to determine the DRG, and then have DIAMOND whole claim price the claim. See Pricing DRGs Without an External Pricer for more information.
