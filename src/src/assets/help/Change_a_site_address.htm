Change a site address

Use this procedure to change a site address. Changes to a site address automatically ripple to all linked provider address records, except terminations.

When you terminate a site address, the corresponding provider address records are not terminated, since provider address records cannot be terminated. Instead, the DIAMOND application displays the termination date on all corresponding provider address records. For more information, see Terminate a site address.

To change a site address

  1. Press F3, type SITEA, and then press ENTER.

  2. In the Site Code box, type the site code for which you want to change the site address. F5 lookup is available.

  3. In the site address picklist, click the site address you want to change.

  4. Change the fields in the window as needed. For more information, see SITEA Field Definitions.

  5. Press CTRL+S to save.
