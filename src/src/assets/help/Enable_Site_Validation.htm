Enable Site Validation

When you enable Site Validation, you make available keywords SITEM (Site Code Master), SITEA (Site Address), and all fields used specifically for Site Validation. You also enable Site Validation processing logic.

To enable Site Validation

  1. Press F3, type PARAM, and then press ENTER.

  2. In the Parameter ID column, select MEMSITEVAL.

  3. In the Parameter 1 box, type Y.

  4. On the File menu, click Save.
