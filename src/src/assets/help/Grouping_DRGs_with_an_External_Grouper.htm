Grouping DRGs with an External Grouper

Restrictions

Instructions

Frequently Asked Questions

You can determine a DRG code for an institutional claim using an external grouper. For external grouping (and pricing), DIAMOND uses the HSS EASYGroupÔ 2000 Optimizer. The HSS Optimizer determines which version of the HCFA Grouper to use based the claim primary date of service. The Optimizer also interfaces with the single HSS-supported DRG Pricer.

For more information about DRG grouper and pricer functionality and how the HSS Optimizer works, see DRG Grouping and Pricing, How DIAMOND Thinks.

Restrictions

None.

Instructions

Step

Comment

1. On each provider contract for which you want to use an external DRG grouper, enter Facility Number and Paysource using the PROVC Special menu command DRG/APC Information, available in contract pricing details.

 

2. Enter the institutional claims for which you want to use an external DRG grouper.

DIAMOND automatically calls the external DRG grouper when the claim is entered and the corresponding provider contract record’s Facility Number and Paysource have been entered.

To perform DRG grouping, the external DRG grouper needs only the information on the claim and the corresponding provider contract record’s Facility Number and Paysource.
