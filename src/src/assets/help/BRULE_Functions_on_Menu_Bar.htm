BRULE Commands on the Menu Bar

 

Command             Menu   Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Med Def

S

Use to include or exclude one or more medical definitions on the benefit rule. The medical definitions you select have the following functions:

  • Identify the claim lines that trigger the use of this rule.

  • Identify medical definitions, along with those identified in Other Med Defs, which are accumulated together toward any limits or maximums defined in this rule so long as they also meet the conditions defined in Count Toward Max.

You must select at least one medical definition to include or exclude for each benefit rule.

Count Toward Max

S

Use to define those claim detail reason codes that are included in any accumulations to maximums defined by this rule. Definition may be by inclusion or exclusion, i.e., lists either those reason codes which are included in accumulations, or the converse, those which are excluded from those accumulations thereby including amounts from detail lines with reason codes not so excluded.

Other Med Defs

S

Use to define the medical definitions which are accumulated along with those identified in Med Def toward any limits or maximums defined in this rule, so long as they also meet the reason code conditions identified in Count Toward Max. Definition may be by inclusion or exclusion, i.e., lists either those medical definitions which are included in accumulations, or the converse, those which are excluded from those accumulations, thereby including amounts from detail lines with medical definitions not so excluded.
