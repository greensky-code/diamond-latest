GEOZIPMATCH

Controls the length of the values that can be entered in the From and Through fields of the Geo Zip Code Ranges section box on the REGIN (Region) screen

Geo Zip is a calculation method that can be selected on the PRULD (Price Rule Detail) screen. The Geo Zip Code Ranges defined on the REGIN screen work in conjunction with the Geo Zip calculation method.
