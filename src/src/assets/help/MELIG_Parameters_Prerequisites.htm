MELIG Parameters & Prerequisites

The following is a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

 

Parameter

Description

Comments

MEMSITEVAL

Used to enable member Site Validation functionality.

If MEMSITEVAL Parameter 1 = N, keyword MELIG field Site is unavailable.

PRIVACYIND

Sets the default value for the Privacy On check box in MELIG.

Parameter 1:

Y Sets the Privacy On check box to Yes.

N Sets the Privacy On check box to No.

MEDABTERMDT

Specifies whether DIAMOND logic validates that the termination date for a member's eligibility period for a Medicare + Choice plan line of business is the first of the month following the MEMMC Medicare Part A/B Term Dt.

 

MEMMEDAEFF

Specifies whether DIAMOND logic validates that the effective date for a member's eligibility period for a Medicare + Choice plan line of business is equal to or later than the member's MEMMC Medicare Part A Eff Dt.

 

MEMMEDBEFF

Specifies whether DIAMOND logic validates that the effective date for a member's eligibility period for a Medicare + Choice plan line of business is equal to or later than the member's MEMMC Medicare Part B Enrl Dt.
