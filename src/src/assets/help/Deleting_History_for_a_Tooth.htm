Deleting History for a Tooth

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to delete history for a tooth.

Restrictions

You can only delete a tooth history record if it was originally added to keyword THIST (Tooth History) manually. That is, if DIAMOND added the tooth history record because a claim was entered in keyword DNCLM (Dental Services Claims), then you cannot delete the tooth history record.

Instructions

Step

Comment

1. Display keyword THIST.

 

2. In the Diamond ID or Member ID boxes, type the DIAMOND or Member ID, and then press TAB.

F5 lookup is available.

2. In the T# picklist, click the tooth history record.

 

3. On the File menu, click Delete.

 

Frequently Asked Questions

None.
