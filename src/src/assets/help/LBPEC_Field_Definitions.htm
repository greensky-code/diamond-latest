Line of Business

Unique identifier of the line of business for which this default pre-existing condition rule applies

Selected from predefined list

Unique identifier of this default pre-existing condition rule

Pre-existing condition rules are entered using keyword PXRUL (Pre-Existing Condition Rule)

Description of the pre-existing condition rule from keyword PXRUL

Effective Date

R

Manually entered

 

Term Date

O

Last date this default pre-existing condition rule is effective for this line of business

Manually entered

 

Apply to all Groups in LOB

R

Indicator of whether all groups in this line of business include this default pre-existing condition rule, or if the groups are defined in the Group Specifications section box.

Selected with check box

Valid values:

  • Selected

All groups in this line of business include this pre-existing condition rule. Group Specifications section box   disabled.

  • Clear

If no records entered in the Group Specifications section box, all groups in this line of business include this pre-existing condition rule.

If one or more records entered in the Group Specifications section box, groups are included or excluded from this pre-existing condition rule as specified.

Term Reason

C

Code explaining the reason for termination of this default pre-existing condition rule for this line of business.

Selected from predefined list

Required if there is a date entered in the Term Date field.

F5 lookup is available.

Termination reason codes are maintained in keyword REASN with reason code type TM.

Operator

C

Indicates whether the specified Group ID or range of Group IDs are included or excluded from this default pre-existing condition rule

Selected from predefined list

Required if entering a line in the Group Specifications section box.

Valid values:

  • Include

  • Exclude

Group ID From

C

Group ID or the beginning of a range of Group IDs which are included or excluded (per the specified Operator) for this default pre-existing condition rule.

Selected from predefined list

Required if entering a line in the Group Specifications section box.

F5 lookup is available.

Groups are maintained in keyword GROUP.

Group ID Thru

O

End of a range of Group IDs which are included or excluded (per the Operator specified) for this default pre-existing condition rule.

Selected from predefined list

F5 lookup is available.

Groups are maintained in keyword GROUP.
