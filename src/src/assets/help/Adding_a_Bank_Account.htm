Add a Bank Account

Use the following procedure to add a bank account.

To add a bank account

  1. Press F3, type BANKA, and then press ENTER.

  2. On the File menu, click New.

  3. In the Bank Account Code box, type the bank account ID.

  4. Fill in the remaining fields in the window. For more information about the fields in the window, see BANKA Field Definitions.

  5. On the File menu, click Save.
