Setting Up PCP Auto Assignment

Restrictions

Instructions

Frequently Asked Questions

In order to use the PCP Auto Assignment batch process to automatically assign a PCP or validate a submitted PCP, you first identify the lines of business and providers for which you want to use PCP Auto Assignment functionality. Then for each line of business and provider included, you determine and enter appropriate values, based on your business requirements, for the key data used exclusively by PCP Auto Assignment.

For more information about how the key values you specify are used in PCP Auto Assignment, see PCP Auto Assignment, How DIAMOND Thinks.

Restrictions

None.

Instructions

Step

Comment

1. Determine and enter appropriate values for the system parameters:

  • ENROLLLIMITS

  • PCPAAFAMUSRD

System parameters apply to all included lines of business and providers.

For more information, see PARAM Record Definitions.

2. Identify all PCPs who should be available for PCP auto assignment, and for what date ranges.

 

3. On each included PCP’s PROVE (Provider Enrollment) record, complete fields User Defined 1 and User Defined 2 as appropriate if you intend to use them in PCP Auto Assignment. Save each record.

See PCP Auto Assignment, How DIAMOND Thinks for more information.

4. On each excluded provider’s PROVE (Provider Enrollment) record, in the PCP Auto Assign Info section box, clear the Avail for Auto Assign check box.

The default for the Avail for Auto Assign check box default is selected. So you do not need to select this check box for included PCP’s. You only need to clear this check box for excluded PCPs.

5.  For each included PCP, use PROVF to enter the MAX Enroll Lmt and save the record.

MAX Enroll Lmt defaults to 99,999.

6. Identify the lines of business for which you want to perform PCP Auto Assignment and determine key PCP Auto Assignment particulars for each.

 

7.  For each included line of business, complete the fields in the LINBS PCP Auto Assign Info section box and save the record.

LINBS PCP Auto Assign Info fields include:

  • Ind PCP/LINBS MaxEnrollLmt

  • Ind PCP/LINBS Threshold

  • Default PCP

  • No Auto PCP Found

  • ID Card Reinst Days

  • Family Affiliation

The Default PCP ID is designed to enable you to add a member eligibility record to DIAMOND when no PCP assignment exists and the member’s line of business requires a PCP. You can then run PCP Auto Assignment to replace the Default PCP. For each line of business for which you want to do this, you must set up a valid provider contract for the Default PCP ID.

For more information, see LINBS Field Definitions and PCP Auto Assignment, How DIAMOND Thinks.

8.  For each included line of business, use keyword PCPSI to enter the PCPAA Support Info Details.

PCPSI includes four section boxes:

  • Pediatric Indicator - Use to specify an age range that qualifies a member as a child for possible assignment of a PCP whose primary specialty is pediatrics

  • OB/GYN Indicator - Use to specify a gender and age range that qualifies a member for possible assignment of a PCP whose primary specialty is OB/GYN

  • Specialty Restrictions by Gender - Use to specify a gender(s) that excludes a member from assignment of a PCP with a selected primary specialty

  • Specialty Restrictions by Age - Use to specify an age range that excludes a member from assignment of a PCP with a selected primary specialty

9.  Use keyword PCPRL to enter PCP Auto Assignment rules available for all included lines of business by Event Type.

These rules can be used as needed across all included lines of business.

PCP Auto Assignment includes two Event Types:

  • Event 1 - Validate/Replace Submitted PCP

  • Event 2 - Replace a Default PCP or No PCP Selection

For more information on PCP Auto Assignment rules and how they are applied, see PCP Auto Assignment Rules.

10.  For each included line of business, use keyword PCPAA to:

  • Build a rules package for PCP Auto Assignment

  • Select special conditions when you can override enrollment threshold capacity levels (section box Override Enrollment Threshold Capacity Levels)

  • Select priority assignment evaluations to perform in order when a submitted PCP does not pass validation (section box Evaluate Possible Selections)

 

11.  Use keyword ZIPRF to complete the field Zip Code Qualifier for each ZIP Code for you want to include in PCP Auto Assignment.

PCP Auto Assignment processing assumes a Zip Code Qualifier of blank means Urban, so only Rural ZIP Codes need be specifically entered.
