Add a Tooth Rule

Use the following procedure to add a tooth rule.

To add a tooth rule

  1. Press F3, type TRULE, and then press ENTER.

  2. In the Tooth Rule ID box, type the rule ID, and then press TAB. You can type up to 15 alphanumeric characters. When a claim detail line is entered in keyword DNCLM, it searches through the tooth rules from lowest rule ID (1) to highest rule ID (Z). When a match is found, it stops searching through the rules. Therefore, for specific rules, type a lower rule ID. For general rules, type a higher rule ID. Alphabetical characters (A-Z) have a higher value than numeric characters (1-9).

  3. Do one of the following:

  4. If you want to add a rule for a tooth number, then in the Tooth Number box, type or select the tooth number. Continue to step 4.

  5. If you want to add a rule for a procedure code, then in the Procedure Code box, type the procedure code. F5 lookup is available. Continue to step 4.

  6. If you want to add a rule for a tooth number and procedure code, then in the Tooth Number box, type or select a tooth number. Then, in the Procedure Code box, type the procedure code. F5 lookup is available.

  1. In the Description box, type the description of the rule. You can type up to 30 alphanumeric characters.

  2. In the Eff Date box, type the date this rule is effective. Type the date in the MM/DD/YYYY format. For example, 12/31/2002.

To enter rule criteria

  1. If you want to make this rule effective for members at or over a particular age, then in the From Age box, type the age. You can type a whole number from 1 to 999. For example, if you type 10, then this rule is effective for members aged 10 and over.

  2. If you want to make this rule effective for members under a particular age, then in the To Age box, type the age. You can type a whole number from 1 to 999. For example, if you type 12, then this rule is effective for members aged 11 and under.

  3. On the File menu, click Save.

  4. If you want to add this rule for a surface, then on the Special menu, click Surfaces. The Tooth Rules dialog box displays. To add surface criteria, click Add. To return to the Tooth Rule window, click OK.

  5. If you want to add this rule for an Arch, then on the Special menu, click Arch. The Tooth Rules dialog box displays. To add surface criteria, click Add. To return to the Tooth Rule window, click OK.

  6. If you want to add this rule for a quadrant, then on the Special menu, click Quadrant. The Tooth Rules dialog box displays. To add surface criteria, click Add. To return to the Tooth Rule window, click OK.

  7. If you want to add this rule for an Oral Cavity, then on the Special menu, click Oral Cavity. The Tooth Rules dialog box displays. To add surface criteria, click Add. To return to the Tooth Rule window, click OK.

  8. On the File menu, click Save.
