Change member Medicare + Choice information

Use this procedure to change member Medicare + Choice information.

To change member Medicare + Choice information

  1. Press F3, type MEMMC, and then press ENTER.

  2. In the DIAMOND ID box, type the DIAMOND ID of the member for whom you want to change member Medicare + Choice information.

  3. In the Subscriber ID box, type the Subscriber ID of the member for whom you want to change member Medicare + Choice information.

  4. In the person number picklist, click the person number of the member for whom you want to change member Medicare + Choice information.

  5. In the effective period picklist, click the effective period you want to change.

  6. Change the fields in the window as needed. For more information, see MEMMC Field Definitions.

  7. Press CTRL+S to save.
