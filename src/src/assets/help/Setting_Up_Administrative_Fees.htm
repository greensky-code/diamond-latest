Setting Up Administrative Fees

To set up administrative fees, do the following:

1. Add a reason code

2. Add a G/L reference code

3. Exclude provider contracts

4. Add administrative fee rules

1. Add a reason code

Add a reason code in keyword REASN (Reason Code) to store the reason why a provider charged an administrative fee to a health plan. Later, you select the reason code when adding an administrative fee rule in keyword ADRUL (Provider Administrative Fee Rule).

Learn how to add a reason code for administrative fees.

2. Add a G/L reference code

Add a G/L reference code in keyword GLREF (G/L Reference Code) to store the credit and debit accounts for administrative fees. That way, you can keep administrative fee financial data separate from other accounts.

Learn how to add a G/L reference code for administrative fees.

3. Exclude provider contracts

If necessary, you can exclude a provider contract from administrative fees. This may be convenient for provider contracts that do not charge administrative fees.

Learn how to excluding a provider contract from administrative fees.

4. Add administrative fee rules

Add an administrative fee rule to select the services eligible for administrative fees and to enter the fee itself.

Learn how to adding an administrative fee rule.
