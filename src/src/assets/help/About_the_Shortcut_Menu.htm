About the Shortcut Menu

You can use the Shortcut Menu to display a keyword window. See Figure 3.

To use the Shortcut Menu

  1. On the File menu, click Shortcut Menu

  2. In the Keyword list, click the keyword.

  3. Click OK.

- or -

  1. Press F3.

  2. In the Keyword box, type the five-letter keyword acronym.

  3. Press ENTER.

You can also use the Main Menu to display a keyword window.
