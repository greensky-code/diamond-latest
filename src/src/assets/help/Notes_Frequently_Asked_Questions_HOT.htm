Frequently Asked Questions

  • Can I view all authorization notes from the AUTHS screen if the member has notes for multiple authorizations?

  • No. The default Note List view shows only those note associated with the specific authorization record that you were viewing, and none of the consolidation options will display all authorization notes for this member. You can, however, go to MEMBR, access notes, and consolidate the note list to view all notes for that member’s authorizations if the proper Window and Note Type Links have been set up in NOTEW.

 

  • I get the Notes Go To Window popup each time I look at a particular claim record. I’ve already read the note, how can I keep the popup window from appearing?

  • The only way you can keep the popup window from appearing when you access the claim is to expire the note. To do this, access the note and enter an expiration date that is earlier than the current date.
