UNITV Field Definitions

Field Name

Req

Description

Source

Comments

Anesthesia

R

Unit value ascribed to the provision of anesthesia for purposes of performing a surgical procedure. The combination of procedure and modifier indicate the service is the anesthesia, not the surgery.

Manually entered.

Must be a positive number between 0 and 99999.999.

Note that of the four Modifier codes on the professional services claim detail line, DIAMOND only uses Modifier 1 in pricing and adjudication.

Conversion Factor Types

R

Category of conversion (dollar) factor which is applicable to each of the professional, technical, and anesthesia components.

Selected from a predefined list.

Conversion factor types are maintained in FACTY.

Effective Date

R

First date the entries in this record apply.

Manually entered.

 

Geographic Region

R

Geographic region to which this unit value record applies.

Selected from a predefined list.

Regions are maintained in REGIN.

Procedure Code

R

Unique code representing a medical service.

Manually entered.

Typically a CPT, HCPCS, ASA, or UB92 code.

Professional

R

Component of Total Units ascribed to the interpretation of the results of a diagnostic procedure.

Manually entered.

Must be a positive number between 0 and 99999.999.

The combination of procedure and modifier indicate the service is the professional component only.

Scale

R

Origin of the unit values used, such as Medicare, McGraw-Hill, RBRVS.

Selected from a predefined list.

Scale values are maintained in SCALE.

Technical

R

Component of Total Units ascribed to the performance of a diagnostic procedure.

Manually entered.

Must be a positive number between 0 and 99999.999.

The combination of procedure and modifier indicate the service is the technical component only.

Total Units

R

Measure of the difficulty, complexity, or time required for a procedure relative to other procedures.

Manually entered.

Must be a positive number between 0 and 99999.999.

If Professional and Technical units are entered they should sum to Total Units.
