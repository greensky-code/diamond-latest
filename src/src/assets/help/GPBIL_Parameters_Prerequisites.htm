GPBIL Parameters & Prerequisites

Keyword GPBIL should not be run while member maintenance is underway.

The following is only a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

 

Parameter

Description

Comments

PRORATE

Sets amount of monthly premium to charge for effective members terminated in the first half of the month. Use parameter PRORTMTHDEF to define the middle date of the month.

 

PRORTMTHDEF

Used in conjunction with the PRORATE parameter. PRORTMTHDEF enables you to set the middle date of the month as the 14th or the 15th.
