TRULE Commands on the Menu Bar

 

Command             Menu   Description

Arch

S

Specify the arch values that will be used when evaluating this tooth rule.

Audit Display

Details

W

Displays audit trail records for the selected record.

Notes

Details

N

This command is only available if the TRULE window has been associated with at least one note type in NOTEW.

Takes you to the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.

Oral Cavity

S

Specify the oral cavity values that will be used when evaluating this tooth rule.

Quadrant

S

Specify the quadrant values that will be used when evaluating this tooth rule.

Surfaces

S

Specify the surface values that will be used when evaluating this tooth rule.
