Print a Check Print Job

Use the following procedure to print a check print job. You must have created a check template in Intelligent QueryÔ (IQ) to print checks. Checks are printed for vendors with an A/P balance equal to zero or greater dollars ($0.00+).

Important Note  You can only print a check print job with the status set to Calculated.

To print a check print job

  1. Press F3, type CKPRT, and then press ENTER.

  2. In the picklist, click the check print job for which you want to print checks.

  3. Click Print.

  4. On the File menu, click Run Job.

  5. If you want to print alignment checks, then click Print Alignment Check. You can print as many alignment checks as necessary. The IQ software icon displays while alignment checks are being printed. The next available check number is displayed in Start Check No (specific to the Bank Account selected). The first valid check printed will be numbered with the sum of the Start Check No. plus the number of alignment checks printed in this step.

  6. Click Print All Checks. The IQ software icon displays while checks are being printed.
