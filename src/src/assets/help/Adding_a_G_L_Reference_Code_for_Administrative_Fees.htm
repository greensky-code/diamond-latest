Add a G/L Reference Code for Administrative Fees

Use the following procedure to add a G/L reference code for administrative fees.

To add a G/L reference code for administrative fees

  1. Press F3, type GLREF, and then press ENTER.

  2. On the File menu, click New.

  3. In the Company Code list, click the company code.

  4. In the G/L Reference Code box, type the code for which you want to use for the administrative fee debit and credit accounts.

  5. In the Description box, type the description of the G/L accounts (optional).

  6. Under Administrative Fee G/L Codes, do the following:

  7. In the Expense Account box, type the G/L debit account number. You can type up to 24 alphanumeric characters.

  8. In the Payable Account box, type the G/L credit account number. You can type up to 24 alphanumeric characters.

  1. On the File menu, click Save.
