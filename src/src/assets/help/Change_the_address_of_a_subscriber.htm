Change the address of a subscriber

Restrictions

Instructions

Frequently Asked Questions

Use the following procedure to change the address for a subscriber and any dependents with the same address as the subscriber.

Restrictions

DIAMOND will only change the addresses of dependents that have identical address information as the subscriber. The information in the following address fields must be identical:

  • Addr 1

  • Addr 2

  • City

  • State

  • ZIPPOST

  • Home Ph

If the information in any of these fields is different or a field is blank for a dependent in a subscriber unit, then that dependent's address will not change with this procedure.

Instructions

Step

Comment

1. Press F3, type MEMBR, and then press ENTER.

 

2. In the Subscriber ID box, type the subscriber ID, and then press TAB.

F5 lookup is available.

3. If the List of Family Members dialog box displays, then double-click the name of the subscriber.

Subscribers are indicated with a Person No. of 0 or 1.

4. On the Special menu, click Address Ripple.

 

5. In the Address Ripple dialog box, type the new address information, and then click OK.

 

6. On the File menu, click Save.
