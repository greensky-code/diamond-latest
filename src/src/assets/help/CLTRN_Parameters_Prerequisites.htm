The following is only a brief listing and description of the parameters that affect this keyword. See PARAM Record Map for links to more information about each parameter. How DIAMOND Thinks documents may also give further details on how to configure these parameters for your particular needs.

 

Parameter

Description

Comments

CLTRNRULE

Controls whether posted or unposted claim records are available for selection in the claims transmittal report.

U for unposted, P for posted.

If DIAMOND is used for both pricing and adjudicating, must be set to P.

If you choose U, no entries will be made to the general ledger interface file.
