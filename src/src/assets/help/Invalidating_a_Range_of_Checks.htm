Invalidate a Range of Checks

Use the following procedure to run a check posting job to invalidate a range of checks. When you invalidate a check, you cannot reuse the check number.

When you run a check posting job, it cancels, invalidates, or posts all check numbers in the job.

The checks you want to invalidate must have been printed by a check print job in keyword CKPRT (Check Print Setup).

To invalidate a range of checks

  1. Press F3, type CKPST, and then press ENTER.

  2. In the picklist, click the check print job ID for which you want to invalidate checks.

  3. Click Check Status.

  4. In the From Check Num box, type the beginning check number to invalidate.

  5. In the Thru Check Num box, type the ending check number to invalidate.

  6. Under Status, click Cancel, and then click OK.

  7. If there are any other check numbers in the check print job that you want to post or cancel, then in the Status list, click a check status for each check number.

  8. On the File menu, click Run Job. Check print jobs with one or more invalidated checks continue to display in the picklist with a status of Printed.
