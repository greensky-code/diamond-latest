Cancel a Range of Checks

Use the following procedure to run a check posting job to cancel a range of checks. When you cancel a check, you can reuse the check number.

The checks you want to cancel must have been printed by a check print job in keyword CKPRT (Check Print Setup).

Warning!  When you run a check posting job, it cancels, invalidates, or posts all check numbers in the job.

To cancel a range of checks

  1. Press F3, type CKPST, and then press ENTER.

  2. In the picklist, click the check print job ID for which you want to cancel checks.

  3. Click Check Status.

  4. In the From Check Num box, type the beginning check number to cancel.

  5. In the Thru Check Num box, type the ending check number to cancel.

  6. Under Status, click Cancel, and then click OK.

  7. If there are any other check numbers in the check print job that you want to post or invalidate, then in the Status list, click a check status for each check number.

  8. On the File menu, click Run Job. Check print jobs with one or more canceled checks continue to display in the picklist with a status of Printed.
