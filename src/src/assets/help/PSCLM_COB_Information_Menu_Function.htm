COB Alwd Amt Rsn

O

Code explaining the COB Allowed Amt rule value.

System generated; may be manually overridden

DIAMOND displays the reason code from the provider contract. Only used with a COB calculation method of Integration of Benefits or Non-Duplication of Benefits.

COB Allowed Amount Reason codes are maintained in REASN with a type of CB.

COB Alwd Amt Rule

R

Indicates allowed amount used in automatic adjudication when a member has other coverage and the claim is automatically adjudicated using a Professional or Institutional COB Calculation method of Integration of Benefits or Non-Duplication of Benefits.

System generated; may be manually overridden

DIAMOND displays the COB Allowed Amount Rule from the provider contract. Only used with COB calculation methods Integration of Benefits and Non-Duplication of Benefits.

Valid values are:

A Use allowed amount

L Use lesser of allowed or other carrier allowed

OC Carrier Code

R

Code value assigned to an Other Carrier with whom the member has other coverage for the Primary Date of Service entered on the claim header.

Selected from a predefined list

OC Carrier Codes are associated with valid COBHS records for this member and dates of service.

When viewing this field, please note that the Carrier Name listed may not be reflective of the Carrier Code selected from the EOB Indicator - Other Carrier Information Pop-Up.

If the Other Carriers listed do not match the carrier from whom you received the EOB/EOMB, you may select the value Other. Selecting Other allows the claim to be processed and adjudicated using the COB Calculation method specified on the GRUPD record. You can use an OC Carrier Code of Other as a determinant value in a COB Hold/Deny rule if you want to hold or deny such claims.

If there are no existing COBHS records for a member but an EOB/EOMB is submitted with the claim, DIAMOND displays the OC Carrier Codes Other and Undefined. If you select Undefined you may adjudicate the claim detail lines but DIAMOND holds them all with a hold reason code of COBOC.
