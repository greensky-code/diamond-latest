About MEDCO

Use keyword MEDCO (Alternate Search Columns) to define the criteria that DIAMOND processing uses to find the correct medical definition when alternate search is used.

Alternate search allows you to tie certain medical definitions to specific values found on professional and institutional claim headers. In the absence of alternate search criteria, DIAMOND processing selects medical definitions based on determinant values that apply to all claims, i.e. across the board.
