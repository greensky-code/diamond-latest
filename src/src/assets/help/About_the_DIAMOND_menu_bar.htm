About the DIAMOND menu bar

This is a summary of the menu bar commands that are available from most keyword windows. If several keyword windows are open at the same time, the menu commands are available only for the active keyword window (the window on which the cursor is positioned).

Each menu displays commands; however, some commands may appear dimmed, which indicates that they are not available for this keyword window.

File menu (ALT+F)

Command

Shortcut Keys

Use

New

CTRL+N

Creates a new record.

Open

CTRL+O

Prompts you to save the current record if it has been changed, then opens a new data record for entry.

Delete

CTRL+D

Deletes the current record.

Save

CTRL+S

Saves the current record.

Close

CTRL+C

Prompts you to save the current record if it has been changed, then closes the current record.

Main Menu

F2

Displays the Main Menu without closing any opened windows.

Shortcut Menu

F3

Displays the Shortcut Menu without closing any opened windows.

Print

ALT+F then P

Prints the current record.

Printer Setup

ALT+F then R

Displays the Printer Setup dialog box.

Exit

ALT+F4

Quits the DIAMOND program.

Some keyword windows have additional File menu commands; such as, New from Template, Save As Template, or Run Job. When available, these commands are documented in the "Commands on the Menu Bar" Help topics.

Edit menu (ALT+E)

Command

Shortcut Keys

Use

Undo

CTRL+Z

Undoes the last action taken.

Cut

CTRL+X

Removes the selection and places it on the Windows Clipboard.

Copy

CTRL+C

Copies the selection and places it on the Windows Clipboard.

Paste

CTRL+V

Pastes from the Windows Clipboard into the current cursor location.

Lookup

F5

Displays the Lookup dialog box to find table data relevant to the field in which the cursor is positioned.

Topic menu (ALT+T)

This menu, where available, is unique to each keyword. You can access other keywords or commands that are associated with the active keyword using the Topic menu. Topic menu commands are documented in "Commands on the Menu Bar" Help topics for each keyword.

Special menu (ALT+S)

This menu, where available, is unique to each keyword. You can run commands that are associated with the active keyword using the Special menu. Special menu commands are documented in "Commands on the Menu Bar" Help topics for each keyword.

Notes menu (ALT+N)

This menu, where available, accesses the Notes Function. See NOTES, NOTET (Note Type), and NOTEW (Note Window) user documentation for additional information.

Window menu (ALT+W)

Command

Shortcut Keys

Use

Tile

SHIFT+ALT+T

Arranges open windows in Windows Tile Format.

Layer

SHIFT+ALT+L

Arranges open windows in Windows Layer Format.

Cascade

SHIFT+ALT+C

Arranges open windows in Windows Cascade Format.

Arrange Icons

SHIFT+ALT+I

Rearranges icons (for minimized windows) in a row at the bottom of the screen.

Show Timestamp

ALT+W then S

Shows the record creation (insert) and last update data (date, time, user and process) for the current record.

Help menu (ALT+H)

Command

Shortcut Keys

Use

Contents

ALT+H then C

Displays the Online Help contents menu.

Search for Help on

ALT+H then S

Displays the Online Help search dialog box.

This Window

F1

Displays Online Help for the current window (keyword).

Glossary

ALT+H then G

Displays the Online Help glossary.

Getting Started

ALT+H then D

Displays an overview of system-wide conventions.

How to use Help

ALT+H then H

Instructs how to use a Windows help system.

About Diamond Client/Server

ALT + H then A

Displays the version of DIAMOND software, database, system resource information, copyright statement, and current signed-on user.
