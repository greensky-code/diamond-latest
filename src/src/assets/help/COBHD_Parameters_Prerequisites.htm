COBHD Parameters & Prerequisites

If a COBHD rule is changed or newly added, it will only apply to new claims you process after logging out of DIAMOND and then logging back in. For an existing claim to be re-evaluated under the new COBHD rule, you must choose the claim header Special menu command Change Header, change the Subscriber ID or Person Number and save. Note that when you do so DIAMOND deletes the entries in the following fields: Other Carrier Code, Other Carrier Allowed and OC Paid Amt. DIAMOND also resets the EOB Indicator to None. You must make new entries as needed.
