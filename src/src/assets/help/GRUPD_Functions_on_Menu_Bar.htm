GRUPD Commands on the Menu Bar Command  Menu  Description

Audit Display

Details

W

Displays audit trail records for the selected record.

Copy Group Detail

S

Makes an exact duplicate of the selected GRUPD record and corresponding GRUPB record. Modify as needed and save as a new record.

D/C Information

Details

S

This command is available only if the GRUPB > Use EFT (Electronic Fund Transfer) check box is selected for the group. This Debit/Credit account information is stored in DIAMOND’s pmb_eft_interface table but not used during the premium billing process.

Notes

Details

N

This command is only available if the GRUPD window has been associated with at least one note type in NOTEW.

Takes you to the Notes Entry window where you can view a list of notes or add, view, or edit a note detail.

For information on setting up notes functionality, see NOTES, How DIAMOND Thinks; NOTET, Note Type; and NOTEW, Note Window user documentation.
