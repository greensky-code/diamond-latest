About SWIND

Use keyword SWIND (Window Access) to grant a user or user template access to a keyword window.

Because a security template association in keyword SUSER overrides individual user entries in keyword SWIND, you may only limit window access for that user’s template. That is, individual user entries in SWIND will be overridden by that user’s template entries in SWIND.

In order to grant access to a keyword, you must also grant access to its associated "sheet." For example, access to keyword INCLM requires access to both INCLM and INCLMSHT.

A master list of all keywords and associated sheets is available in Security and Users, How DIAMOND Thinks. In some cases, multiple keywords share one sheet.
