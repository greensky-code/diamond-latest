RHJOB Field Definitions

Field Name

Req

Description

Source

Comments

Action

R

The mode in which the job is to be run.

Selected from predefined list

Valid value:

Run: DIAMOND performs all the batch claim hold release and readjudication logic and updates the associated claims.

Claim Date Used

R

Date used to determine which claims of the designated claim type will be evaluated in this RHJOB job.

Selected from predefined list

Valid values:

Entered Date

Received Date

Claim From Date

R

Earliest Claim Date Used allowed on a claim to be evaluated in this RHJOB job

Manually entered

 

Claim Through Date

O

Latest Claim Date Used allowed on a claim to be evaluated in this RHJOB job

Manually entered

 

Claim Type

D

Type of claims to be evaluated

Displayed

Valid values are selected in the RHRUL rule. Valid values are:

Professional

Institutional

Comments

O

Free form comments about the RHJOB batch job

Manually entered

 

Job ID

R

Batch job identifier

Manually entered

Must be unique.

Once the job is saved or run, the Job ID may not be changed.

Reapply Header Holds

O

Controls whether CLHLD header hold rules are reapplied as part of the Batch Claim Hold processing

Selected with checkbox

If selected, reapplies CLHLD user defined header hold rules after specific holds are released.

Release Rule

R

Unique identifier of the Release Rule to be used during processing

Selected from predefined list

Release rules are entered in RHRUL and rule determinants in RHDET.

Request Date

D

Current date

System generated

 

Request Type

R

Processing request type.

Selected from predefined list

Valid values are:

Immediate

Deferred

Accept Immediate as default. Deferred is not supported at this time. Error will occur.

Request User

D

User ID of the current user

System generated

 

Status

R

Status of the RHJOB job.

System generated

Valid Values:

New: You created a new RHJOB job record by selecting File Menu, option New or option New From Template, entering data, and saving the job.

Successful: You selected Action > Run, followed by File Menu > Run Job. DIAMOND ran the job in accord with your specifications and the job completed successfully.

Error: You selected Action > Run followed by File Menu > Run Job. DIAMOND ran the job in accord with your specifications and the job completed successfully but with critical errors.

In Process: You selected Action > Run followed by File Menu > Run Job and the job is still running.
