Heads Down Claims Commands on the Menu Bar Command  Menu  Description

New File

F

The Heads Down Claims executable displays a blank Header window. Use this window to enter claims and create a new claims text file on your workstation. See Heads Down Claims Commands on the Menu Bar > Add Record for additional information.

Open Text File

Details

F

Opens a claims text file that has been previously saved on the workstation. Click Details for more information.

Save Text File

Details

F

Saves a claims text file to the workstation. Click Details for more information. Click Details for more information.

Add Record

Details

F

Adds an entire claim record or a claim detail line record to an existing claims text file. Click Details for more information.

Delete Record

Details

F

Deletes an entire claim record or a claim detail line record from an existing claims text file. Click Details for more information.

Exit

F

The Heads Down Claims executable exits (closes) the Heads Down Claims Entry windows.
