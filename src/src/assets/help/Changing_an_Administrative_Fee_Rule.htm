Change an Administrative Fee Rule

  1. Press F3, type ADRUL, and then press ENTER.

  2. In the ID box, type the rule ID you want to change, and then press ENTER. F5 lookup is available.

  3. Make the changes you want.

  4. On the File menu, click Save.
