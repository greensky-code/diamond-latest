Add a Message on an EOB

Use the following procedure to add a message to print on an EOB.

To add a message on an EOB

  1. Press F5, type RMRKS, and then press ENTER.

  2. On the File menu, click New.

  3. In the Remark Code box, type the ID of the message.

  4. In the Remark Type list, click E - EOB.

  5. In the EOB Remark box, type the message.

  6. On the File menu, click Save.
