Account No (Offset To)

 

Account number of the Vendor/Vendor Address combination

Manually entered

 

Adv Pay Acc No

 

The number that identifies the advance payment account.

Selected from predefined list

This field is available only if the Credit Type is Advance Payment.

F5 lookup is available.

Apply To

 

Apply to this account or move to another account.

Selected from predefined list

Valid values are:

  • Write Off

  • Apply to Another Account

If Write Off is selected, all other  fields in the Offset To section box disabled.

If Apply to another Account is selected, all other fields in the Offset To section box are enabled and required.
