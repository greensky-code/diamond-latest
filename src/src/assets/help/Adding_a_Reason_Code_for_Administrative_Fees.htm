Add a Reason Code for Administrative Fees

Use the following procedure to add a reason code for administrative fees.

To add a reason code for administrative fees.

  1. Press F3, type REASN, and then press ENTER.

  2. On the File menu, click New.

  3. In the Reason Code box, type the reason code ID. You can type up to 5 alphanumeric characters.

  4. In the Reason Code Type list, click AF – Administrative Fee.

  5. In the Description box, type the description of the reason code (optional).

  6. On the File menu, click Save.
