Save Text File Menu Command

The Heads Down Claims executable displays a Save As dialog box. You can:

Save a New Claim Text File

Step

1. Navigate to and double-click on the workstation directory under which the claims text file will be saved.

2. Enter the name of your claims text file using your organization’s file naming conventions.

3. Click Save or press ALT+S.

4. The executable saves the file, closes the file, and displays a blank Heads Down Claim Entry window.

Save an Edited Claim Text File

Step

1. Click Save or press ALT+S.

2. Double-click on the name of the existing (edited) claim text file displayed in the dialog box.

3. The executable saves the file, closes the file, and displays a blank Heads Down Claim Entry window.
