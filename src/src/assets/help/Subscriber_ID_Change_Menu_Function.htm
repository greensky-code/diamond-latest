Subscriber ID Change Menu Command

 

Field Name  Req Description Source Comments

Subscriber ID

R

Subscriber ID of the member currently displayed in the Member Master screen.

Displayed

 

New Subscriber ID

R

Suggested new Subscriber ID for the member displayed in the Member Master screen.

Manually entered

When a new Subscriber ID is entered and OK is clicked, DIAMOND searches to see if this ID has already been assigned. If it has not been assigned, then the ID is held in a worktable until the SUBID batch job is run which will change the ID.

For more information, see Subscriber ID Changes, How DIAMOND Thinks.
