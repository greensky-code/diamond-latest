About APSEL

Use keyword APSEL (Accounts Payable Selection) to run an A/P selection job to de-select or re-select a batch of A/P transactions for payment.

If an A/P transaction is de-selected for payment, then it is skipped by keyword CKPRT (Check Print Setup).

All de-selected transactions remain de-selected until you run a new A/P selection job to re-select a batch of A/P transactions.

You can run an A/P selection job based on company code, vendor ID, inserted date, or due date.
