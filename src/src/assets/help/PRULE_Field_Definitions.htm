Curr Conv Tier One – Tier Four

O

For each tier, the minutes per unit currently used.

Manually entered.

One current and one prior minutes per unit can be defined for each tier, making the field and its use semi-historical.

Curr Conv Effec Date

C

If Prior Conv values are entered, the first date the current values apply according to the date of service of the anesthesia.

Manually entered.

This field is the boundary between current unit and prior unit definitions.

Custom Price Rules

 

This field is not used. Do not check box.

Do not use.

 

Description

Type the description of this Price rule.

Manually entered.

 

From Min Tier One – Tier Four

O

Lower end of a time increment used to automatically calculate anesthesia units.

Manually entered.

Up to four different increments can be defined so that the number of minutes per unit can change over the course of the anesthesia. For more information, see Pricing Claim Line Items, How DIAMOND Thinks.

Minimum Min for a Unit

O

Minimum number of minutes of anesthesia defining a unit.

Manually entered.

If this field is blank, anesthesia units are calculated to the nearest tenth of a unit.

Price Rule

Type the Price rule ID.

Manually entered.

F5 lookup is available.

Prior Conv Tier One – Tier Four

O

For each tier, the minutes per unit used prior to the Curr Conv Tier as of the effective date.

Manually entered.

One current and one prior minutes per unit can be defined for each tier, making the field and its use semi-historical.

Procedure Price Search Option

R

Indicates whether or how the procedure modifier is used to find the correct price information (% billed, unit value, or fee will be used depending on PRULD).

Selected from a predefined list.

Values are:

N Modifier is not considered in looking for the PROCP record.

Y Modifier is considered in looking for the PROCP record.

W Modifier is considered, but if no record is found DIAMOND uses the PROCP record which includes the wildcard modifier.

In all cases, the procedure code, primary price schedule or alternate price schedule, price region and effective date are used.

Thru Min Tier One – Tier Four

O

Upper end of a time increment used to automatically calculate anesthesia units.

Manually entered.

Up to four different increments can be defined so that the number of minutes per unit can change over the course of the anesthesia. For more information, see Pricing Claim Line Items, How DIAMOND Thinks.

Use External DRG Pricer

O

Select to use an external DRG Pricer when this rule is applied.

Selected by check box.

Additional requirements apply for using an external DRG pricer. For more information, see Pricing DRGs With an External Pricer.

Wildcard Modifier

C

Modifier value that will be used to search for the correct PROCP record when no match can be found with the modifier entered on the claim line.

Selected from a predefined list.

Modifiers are maintained in keyword MODIF, including wildcards. For more information, see Pricing Claim Line Items, How DIAMOND Thinks.
