Frequently Asked Questions

Use the following procedure to change a price rule.

Restrictions

None.

Instructions

Step

Comment

1. Press F3, type PRULE, and then press ENTER.

 

2. In the Price Rule box, type the Price Rule ID.

F5 lookup is available.

3. Make the changes you want.

 

4. On the File menu, click Save.

None.
