ADRUL Field Definitions

Field Name

Req

Description

Source

Comments

Amount

C

Type the amount to be the administrative fee.

Manually entered

You can type a positive or negative number. A positive number increases the vendor payment. A negative number reduces the vendor payment.

If the Percent box is blank, then this field is required.

Apply Pct To

C

Click or type the amount to use in the percent calculation of the administrative fee.

Selected from predefined list

This field is required if there is a number in the Percent box.

Claim Type

R

Click or type the claim type the administrative fee can be applied.

Selected from predefined list

 

Description

O

Type the description of the rule.

Manually entered

 

Determinant Column

O

Displays the field you want to use as a determinant.

Selected from predefined list

This value is selected when the insertion point is in the Determinant Table box, and you press F5.

Determinant Table

O

Press F5 to select the table and column you want to use. The field you select displays in the Determinant Column box.

Selected from predefined list

Select from the following list:

  • ClaimType_CLAIM_HEADER

  • ClaimType_CLAIM_DETAIL

  • PROV_CONTRACT

(Where ClaimType is the claim type selected in the Claim Type list.)

Eff Date

R

Type the effective date of the rule.

Manually entered

If you type a past date, then previously entered claims are not affected.

From Value

O

Type a from value.

Manually entered

This box is available if in the Operator list, you selected Equal (=) or Not [equal to] (¹).

ID

R

Type the ID of the provider administrative fee rule.

Manually entered

F5 lookup is available.

Operator

O

Click the operand you want to use.

Selected from predefined list

The following operators are available:

  • Equal (=)

  • Not [equal to] (¹)

Percent

C

Type the percent to calculate an administrative fee.

 

To calculate an administrative fee, the percent you type in this box is applied to the claim amount.

If the Amount box displays $0.00, then this field is required.

Reason Code

R

Click or type an administrative fee reason code.

Selected from predefined list

Only administrative fee reason codes with the Reason Code Type list set to AF – Administrative Fee in keyword REASN (Reason Codes) display.

Require Cap Line

O

If you want the claim to have at least on claim line that has the Claim Stat list set to C – Capitation in the claim detail window, then click to select this check box.

Selected from check box

 

Require Clean Claim

O

If you want the claim line to be clean in order to be eligible for an administrative fee, then click to select this check box.

Selected from check box

A claim line item is clean if the Unclean Indicator check box is cleared on the claim detail window and it was not on hold at any point in time from the time the claim was entered until it is evaluated for administrative fees.

Term Date

O

Type the termination date of the rule.

Manually entered

You must type a date equal to or greater than the date in the Eff Date box.

Term Rsn

R

Select a termination reason code.

Selected from predefined list

Only termination reason codes with the Reason Code Type list set to TM – Termination Reason in keyword REASN (Reason Codes) display.

Thru Value

O

Type an ending value.

Manually entered

 

User Date 1

O

Installation-specific date field.

Manually entered

The label for this field is set in keyword UDTXT (User Defined Fields).

User Date 2

O

Installation-specific date field.

Manually entered

The label for this field is set in keyword UDTXT.

User Defined 1

O

Installation-specific text field.

Manually entered

The label for this field is set in keyword UDTXT.

User Defined 2

O

Installation-specific text field.

Manually entered

The label for this field is set in keyword UDTXT.
